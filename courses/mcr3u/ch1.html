Function notation, domain & range, mappings, even/odd, inverses, transformations

📚 Strand A: Characteristics of Functions ⏱️ ~14h 📹 6 topic blocks
1.1 — Relations vs. Functions; the Vertical Line Test
Define a relation; recognise a function from a set of ordered pairs, mapping diagram, table, or graph; apply the vertical line test.
A relation pairs inputs with outputs. A function is a relation in which each input has exactly one output. Graphically, a curve is a function iff every vertical line intersects it at most once. Common counter-example: \( x = y^2 \) (a sideways parabola) fails the test.
📖 Lesson Notes — Ontario Curriculum

A relation is any rule, table, set of ordered pairs, or graph that pairs values from one set (the inputs, or independent variable) with values from another set (the outputs, or dependent variable). A function is a special kind of relation — one with a strict promise: every input is paired with exactly one output. No input can produce two different outputs.

Why does this matter? Because a huge amount of mathematics relies on being able to ask "what is the output when the input is \(x\)?" and get a single, unambiguous answer. If a relation could spit out two values for the same input, you couldn't write \(y = f(x)\) — the symbol \(f(x)\) would be ambiguous. Functions give us a clean input-output machine, which is exactly what we need for modelling real-world processes (height vs. time, temperature vs. depth, cost vs. quantity).

The vertical line test turns this idea into a visual check. Draw any vertical line. If it ever crosses the graph in two or more places, then the \(x\)-value where it crosses is being mapped to multiple \(y\)-values — so the relation isn't a function. If every vertical line crosses the graph at most once, the relation is a function.

  • Relation — any pairing of inputs to outputs (a set of ordered pairs).
  • Function — a relation in which each input has exactly one output.
  • Domain — the set of all permissible inputs.
  • Range — the set of all outputs the function actually produces.
  • Vertical line test — a graph represents a function if and only if every vertical line meets the graph in at most one point.
  • Mapping diagram — a picture with arrows from inputs (left) to outputs (right); a function has at most one arrow leaving each input.

Example 1 — Decide whether each relation is a function.

(a) \(\{(1,2),(2,4),(3,6),(4,8)\}\). Every \(x\)-value is different, so each input has exactly one output. This is a function.

(b) \(\{(0,1),(1,2),(0,-1),(2,3)\}\). The input \(0\) is paired with both \(1\) and \(-1\). Not a function.

(c) The graph of \(x = y^2\) (a sideways parabola). At \(x = 4\), both \(y=2\) and \(y=-2\) lie on the curve. A vertical line at \(x = 4\) hits the curve twice. Not a function.

(d) The graph of \(y = x^2\). For every \(x\), there is exactly one \(y = x^2\). A vertical line meets the curve once. Is a function.

Example 2 — Does \(y^2 + x^2 = 25\) define \(y\) as a function of \(x\)?

Solving for \(y\): \(y = \pm\sqrt{25 - x^2}\). At \(x = 3\), \(y = \pm 4\) — two outputs for one input. The graph is a circle of radius 5, which fails the vertical line test. Not a function.

Watch out for:

  • Confusing the vertical line test with the horizontal line test — vertical decides "is this a function," horizontal decides "is the inverse a function."
  • Assuming any equation involving \(x\) and \(y\) is a function. Circles, sideways parabolas, and many implicit curves are not.
  • Forgetting that a function can repeat outputs — different inputs may share one output (e.g. \(f(-2) = f(2) = 4\) for \(y = x^2\)). What's forbidden is one input sharing two outputs.

Curriculum expectation: Strand A, Expectation A1.1 — "explain the meaning of the term function, and distinguish a function from a relation that is not a function, through investigation of linear and quadratic relations using a variety of representations (i.e., tables of values, mapping diagrams, graphs, function machines, equations) and strategies (e.g., identifying a one-to-one or many-to-one mapping; using the vertical-line test)."

1.2 — Function Notation \( f(x) \); Evaluation
Read and write \( f(x) \), \( g(t) \); evaluate at numerical and algebraic inputs; interpret the meaning of \( f(a) \) in context.
For \( f(x)=2x^2-3x+1 \): \( f(-2)=2(4)-3(-2)+1=8+6+1=15 \). For \( f(a+h) \), substitute \( a+h \) wherever \( x \) appears: \( f(a+h)=2(a+h)^2-3(a+h)+1 \). Function notation makes the input-output relationship explicit and is foundational for all later work in MHF4U and MCV4U.
📖 Lesson Notes — Ontario Curriculum

Up until now you've usually written equations as \(y = \text{(stuff with }x\text{)}\). Function notation just renames the output. Instead of \(y\), we write \(f(x)\), read as "\(f\) of \(x\)." The letter \(f\) is the name of the function (it can be \(g\), \(h\), \(P\), \(C\) — anything), and the \(x\) inside the brackets is a placeholder for whatever input you're about to substitute. This change of notation looks tiny but it is enormous: it lets you name many functions in one problem, talk about specific values like \(f(3)\) without ambiguity, and feed one function into another.

Think of \(f\) as a machine. You feed in a number; the machine processes it; out comes a number. \(f(x) = 2x^2 - 3x + 1\) is the instruction sheet for the machine. \(f(-2)\) is the result of feeding \(-2\) into that specific machine. The brackets are not multiplication — \(f(x)\) does not mean \(f\) times \(x\).

Once you're comfortable plugging in numbers, the same idea lets you plug in expressions: \(f(a+h)\), \(f(2x)\), \(f(x+1) - f(x)\). The last one is the key to the difference quotient, which becomes the derivative in calculus. Mastering substitution at this level pays off for years.

  • Function notation — write \(f(x)\) for the output of function \(f\) at input \(x\).
  • Argument — whatever sits inside the brackets (the input being fed in).
  • Evaluate — substitute a specific value or expression for the variable and simplify.
  • Independent variable — the input, usually \(x\); the value you choose.
  • Dependent variable — the output, \(f(x)\); its value depends on \(x\).

Example 1 — Let \(f(x) = 2x^2 - 3x + 1\). Compute \(f(-2)\), \(f(0)\), and \(f(\tfrac{1}{2})\).

\(f(-2) = 2(-2)^2 - 3(-2) + 1 = 2(4) + 6 + 1 = 8 + 6 + 1 = 15\).

\(f(0) = 2(0)^2 - 3(0) + 1 = 1\). (This is the \(y\)-intercept.)

\(f(\tfrac{1}{2}) = 2(\tfrac{1}{4}) - 3(\tfrac{1}{2}) + 1 = \tfrac{1}{2} - \tfrac{3}{2} + 1 = 0\). So \((\tfrac{1}{2}, 0)\) is on the graph.

Example 2 — For the same \(f\), simplify \(f(a+h) - f(a)\).

\(f(a+h) = 2(a+h)^2 - 3(a+h) + 1 = 2(a^2 + 2ah + h^2) - 3a - 3h + 1 = 2a^2 + 4ah + 2h^2 - 3a - 3h + 1\).

\(f(a) = 2a^2 - 3a + 1\).

\(f(a+h) - f(a) = (2a^2 + 4ah + 2h^2 - 3a - 3h + 1) - (2a^2 - 3a + 1) = 4ah + 2h^2 - 3h\).

This expression — and dividing it by \(h\) — is the average rate of change of \(f\) on \([a, a+h]\), foreshadowing calculus.

Example 3 — Given \(g(x) = \sqrt{x + 4}\), find \(g(5)\) and \(g(2t - 3)\).

\(g(5) = \sqrt{5 + 4} = \sqrt{9} = 3\).

\(g(2t - 3) = \sqrt{(2t - 3) + 4} = \sqrt{2t + 1}\).

Watch out for:

  • Reading \(f(x)\) as multiplication. It is a single symbol naming the output, not a product.
  • Forgetting parentheses when substituting negatives. \(f(-2)\) needs \((-2)^2 = 4\), not \(-2^2 = -4\).
  • When substituting an expression like \(a + h\), you must put it in brackets everywhere the variable appears, then expand carefully.
  • Mixing up "find \(x\) when \(f(x) = 5\)" (an equation to solve) with "find \(f(5)\)" (an evaluation).

Curriculum expectation: Strand A, Expectation A1.2 — "represent linear and quadratic functions using function notation, given their equations, tables of values, or graphs, and substitute into and evaluate functions [e.g., evaluate \(f(\tfrac{1}{2})\), given \(f(x) = 2x^2 + 3x - 1\)]."

1.3 — Domain & Range from Graphs and Equations
State domain and range using set-builder and interval notation; analyse restrictions from radicals (\( \sqrt{\,} \ge 0 \)) and rational expressions (denominator \( \ne 0 \)).
For \( f(x)=\sqrt{x-3} \): domain \( \{x\in\mathbb{R} \mid x\ge 3\} \), range \( \{y\in\mathbb{R}\mid y\ge 0\} \). For \( g(x)=\dfrac{1}{x-2} \): domain \( \{x\in\mathbb{R}\mid x\ne 2\} \). Always check both algebraic restrictions and the contextual domain when modelling.
📖 Lesson Notes — Ontario Curriculum

The domain of a function is the set of all inputs you're allowed to feed it. The range is the set of every output the function actually produces. Knowing both tells you the "shape" of the function before you even graph it: where it starts and ends, what gaps exist, and which heights are reachable.

Domain restrictions come from two main sources at this level. Square roots (and any even-index radical) need a non-negative radicand — you can't take a real square root of a negative number, so anything under \(\sqrt{\ \ }\) must be \(\ge 0\). Rational expressions can never have zero in the denominator — division by zero is undefined. A third source, which appears in word problems, is the context: time can't be negative, you can't ship a fractional number of crates, etc.

Range is usually best found from a graph or from a careful look at the function's behaviour: minimum/maximum value, asymptotes, end behaviour. There are two common notations: set-builder like \(\{x \in \mathbb{R} \mid x \ge 3\}\) and interval notation like \([3, \infty)\). Brackets \([\ ]\) include the endpoint; parentheses \((\ )\) exclude it. Infinity always gets a parenthesis because it isn't a real number.

  • Domain — set of all allowable inputs of a function.
  • Range — set of all outputs the function actually attains.
  • Set-builder notation — \(\{x \in \mathbb{R} \mid \text{condition}\}\); read "the set of \(x\) in the reals such that...".
  • Interval notation — \([a,b]\) closed, \((a,b)\) open, \([a,b)\) half-open; \(\infty\) is always paired with a parenthesis.
  • Restriction — an algebraic condition (e.g. denominator \(\ne 0\), radicand \(\ge 0\)) that excludes inputs from the domain.

Example 1 — State the domain and range of \(f(x) = \sqrt{x - 3}\).

Domain: the radicand must satisfy \(x - 3 \ge 0\), so \(x \ge 3\). In set-builder: \(\{x \in \mathbb{R} \mid x \ge 3\}\); in interval notation: \([3, \infty)\).

Range: \(\sqrt{\ \ }\) by convention returns the non-negative root, so outputs are \(\ge 0\). When \(x = 3\), \(f(x) = 0\); as \(x\) grows, \(f(x)\) grows without bound. Range: \([0, \infty)\).

Example 2 — State the domain and range of \(g(x) = \dfrac{1}{x - 2}\).

Domain: denominator cannot be zero, so \(x - 2 \ne 0\) means \(x \ne 2\). Domain: \(\{x \in \mathbb{R} \mid x \ne 2\}\), or \((-\infty, 2) \cup (2, \infty)\).

Range: \(\dfrac{1}{x-2}\) can be any real number except \(0\) (since the numerator is a nonzero constant). The horizontal asymptote is \(y = 0\), and the function approaches but never reaches it. Range: \(\{y \in \mathbb{R} \mid y \ne 0\}\), or \((-\infty, 0) \cup (0, \infty)\).

Example 3 — A ball is thrown with height \(h(t) = -5t^2 + 20t + 1.5\) (metres) where \(t\) is time in seconds.

Algebraically, \(h\) is defined for every real \(t\). But in context, time starts at \(t = 0\), and the ball stops when it hits the ground: solve \(-5t^2 + 20t + 1.5 = 0\) by the quadratic formula to get \(t \approx 4.07\) s.

Domain: \(\{t \in \mathbb{R} \mid 0 \le t \le 4.07\}\). Range: heights from \(0\) up to the vertex value of \(21.5\) m, i.e. \([0, 21.5]\).

Watch out for:

  • Forgetting to set the denominator equal to zero before declaring the domain. Always solve for the forbidden \(x\)-values.
  • Cube roots and other odd-index radicals accept negatives: domain of \(\sqrt[3]{x}\) is all real numbers.
  • Mixing up brackets and parentheses. Use \([\ ]\) only when the endpoint is included; never use \([\ ]\) with \(\infty\).
  • Stating algebraic domain but ignoring contextual restrictions in word problems.
  • Confusing "no \(x\)-intercept" with "no \(y\)-value zero in the range" — those are different statements.

Curriculum expectation: Strand A, Expectation A1.7 — "determine, through investigation, and using the function notation \(f(x)\), key features of the graphs of the functions defined by \(f(x) = x\), \(f(x) = x^2\), \(f(x) = \sqrt{x}\), \(f(x) = \dfrac{1}{x}\), and \(f(x) = \sin x\) (i.e., domain and range, intercepts, increasing/decreasing intervals, symmetry)."

1.4 — Inverse Functions \( f^{-1}(x) \)
Find inverses algebraically (swap \( x \) and \( y \), solve for \( y \)); reflect graphs across \( y = x \); identify when the inverse is itself a function.
For \( f(x)=3x-5 \), the inverse is found by writing \( y=3x-5 \), swapping to get \( x=3y-5 \), and solving: \( f^{-1}(x)=\dfrac{x+5}{3} \). The graph of \( f^{-1} \) is the reflection of \( f \) over the line \( y=x \). The inverse of \( f(x)=x^2 \) is not a function unless we restrict the domain to \( x\ge 0 \).
📖 Lesson Notes — Ontario Curriculum

The inverse of a function reverses what the function does. If \(f\) takes 3 and turns it into 7, then \(f^{-1}\) takes 7 and turns it back into 3. Symbolically, the rule is \(f^{-1}(f(x)) = x\) and \(f(f^{-1}(x)) = x\). Inverses matter because lots of real questions ask the reverse of the original: if revenue depends on price, what price gives a target revenue? If distance depends on time, when do you arrive at a given distance?

To find an inverse algebraically, swap \(x\) and \(y\) in the equation and solve for the new \(y\). The swap is the algebraic version of reflecting the graph in the line \(y = x\) — every point \((a, b)\) on \(f\) becomes \((b, a)\) on \(f^{-1}\). Because of this reflection, the domain of \(f\) becomes the range of \(f^{-1}\), and vice versa.

A critical caveat: the inverse of a function is itself a function only if the original passes the horizontal line test (every output comes from exactly one input — a one-to-one function). \(f(x) = x^2\) fails this: both \(x = 2\) and \(x = -2\) give output 4, so reversing isn't unique. To get a true inverse function we restrict the domain (e.g. \(x \ge 0\)) before inverting, yielding \(f^{-1}(x) = \sqrt{x}\).

  • Inverse function — \(f^{-1}\) reverses \(f\): \(f^{-1}(f(x)) = x\) for every \(x\) in the domain of \(f\).
  • One-to-one — each output comes from exactly one input; only one-to-one functions have functional inverses.
  • Horizontal line test — a graph passes if every horizontal line meets it at most once; equivalent to one-to-one.
  • Line of reflection — graph of \(y = f^{-1}(x)\) is the reflection of \(y = f(x)\) across \(y = x\).
  • Domain restriction — used to force a many-to-one function to become one-to-one before inverting.

Example 1 — Find the inverse of \(f(x) = 3x - 5\).

Step 1: write \(y = 3x - 5\).

Step 2: swap \(x\) and \(y\): \(x = 3y - 5\).

Step 3: solve for \(y\): \(x + 5 = 3y\), so \(y = \dfrac{x + 5}{3}\).

Therefore \(f^{-1}(x) = \dfrac{x + 5}{3}\).

Verify: \(f^{-1}(f(x)) = \dfrac{(3x - 5) + 5}{3} = \dfrac{3x}{3} = x\). Correct.

Example 2 — Find the inverse of \(f(x) = x^2 + 1\), restricted to \(x \ge 0\).

Step 1: \(y = x^2 + 1\).

Step 2: swap: \(x = y^2 + 1\).

Step 3: solve for \(y\): \(y^2 = x - 1\), so \(y = \pm\sqrt{x - 1}\). Because we restricted the original to \(x \ge 0\) (giving outputs \(y \ge 1\)), the range of the original becomes the domain of the inverse, and we take the positive root: \(f^{-1}(x) = \sqrt{x - 1}\).

Domain of \(f^{-1}\): \(x \ge 1\) (matches range of original). Range of \(f^{-1}\): \(y \ge 0\) (matches restricted domain of original).

Watch out for:

  • The symbol \(f^{-1}(x)\) does not mean \(\dfrac{1}{f(x)}\). The \(-1\) is a name, not an exponent. \(\dfrac{1}{f(x)}\) is the reciprocal function, a completely different thing.
  • Skipping the swap step or doing it after solving — always swap first, then solve.
  • Forgetting that the inverse may not be a function. Always check: does the original pass the horizontal line test?
  • When restricting the domain to make an inverse exist, the new range of the inverse must match the chosen restricted domain.

Curriculum expectation: Strand A, Expectation A1.8 — "determine, through investigation, the relationship between the graph of a function and the graph of its inverse (e.g., the graph of the inverse can be obtained by reflecting the graph of the function in the line \(y = x\); the graph of the inverse is the graph of a function only if the original function is one-to-one)."

1.5 — Transformations of Functions: \( a f(k(x-d)) + c \)
Identify and apply vertical/horizontal stretches and compressions, reflections, and translations on parent functions \( y=x^2 \), \( y=\sqrt{x} \), \( y=|x| \), \( y=1/x \).
In \( g(x)=a\,f(k(x-d))+c \): \( a \) controls vertical stretch (\(|a|>1\)) or compression and reflects in the x-axis if negative; \( k \) controls horizontal stretch (\(|k|<1\)) or compression and reflects in the y-axis if negative; \( d \) is a horizontal shift; \( c \) is a vertical shift. Always apply stretches and reflections before translations.
📖 Lesson Notes — Ontario Curriculum

A small number of parent functions — \(y = x\), \(y = x^2\), \(y = \sqrt{x}\), \(y = |x|\), \(y = \dfrac{1}{x}\), and later \(y = \sin x\) — generate an enormous family of related curves through four transformations: stretches, compressions, reflections, and translations. The general form bundles all four into one expression:

\[ g(x) = a\,f\bigl(k(x - d)\bigr) + c \]

Each parameter has a well-defined effect. The two outside the bracket affect \(y\)-values directly: \(a\) multiplies the output (vertical stretch by \(|a|\); if \(a<0\), reflect in the \(x\)-axis), and \(c\) adds to the output (vertical translation by \(c\)). The two inside the bracket affect \(x\)-values in the opposite way you might expect: \(k\) divides the \(x\)-coordinates by \(|k|\) (horizontal compression if \(|k|>1\), stretch if \(|k|<1\); reflect in the \(y\)-axis if \(k<0\)), and \(d\) adds to the \(x\)-coordinates (translation right by \(d\) when written as \((x - d)\)).

The fastest way to graph a transformation is to track a few "key points" on the parent and apply the rule \((x, y) \to \bigl(\tfrac{x}{k} + d, \, ay + c\bigr)\). Apply stretches and reflections first, then translations. If the inside isn't already in the factored form \(k(x - d)\), factor first.

  • Vertical stretch/compression — factor \(a\) acting on \(y\). \(|a|>1\) stretch, \(0<|a|<1\) compression, \(a<0\) reflect in \(x\)-axis.
  • Horizontal stretch/compression — factor \(k\) acting on \(x\). \(|k|>1\) compression, \(0<|k|<1\) stretch, \(k<0\) reflect in \(y\)-axis.
  • Translation — \(c\) shifts up (positive) or down (negative); \(d\) shifts right (positive) or left (negative).
  • Mapping rule — \((x, y) \to \bigl(\tfrac{x}{k} + d,\ ay + c\bigr)\) sends every parent point to its image.
  • Parent function — the simplest member of a family from which others are obtained by transformation.

Example 1 — Sketch \(g(x) = -2(x - 3)^2 + 4\) from the parent \(f(x) = x^2\).

Identify: \(a = -2\), \(k = 1\), \(d = 3\), \(c = 4\). So the transformation is: vertical stretch by 2, reflect in the \(x\)-axis, translate right 3, translate up 4.

Parent key points: \((-1, 1), (0, 0), (1, 1), (2, 4), (-2, 4)\).

Apply the mapping \((x, y) \to (x + 3, -2y + 4)\): \((-1,1) \to (2, 2)\), \((0,0)\to(3, 4)\), \((1,1)\to(4, 2)\), \((2,4)\to(5,-4)\), \((-2,4)\to(1,-4)\).

Vertex of \(g\) is \((3, 4)\), opens downward. Sketch a parabola through the mapped points.

Example 2 — Describe the transformations applied to \(y = \sqrt{x}\) to obtain \(y = \sqrt{-2x + 6} - 1\).

Inside the radical, factor out the coefficient of \(x\): \(-2x + 6 = -2(x - 3)\). So \(y = \sqrt{-2(x - 3)} - 1\).

Identify: \(a = 1\), \(k = -2\), \(d = 3\), \(c = -1\). Transformations: horizontal compression by factor 2, reflection in the \(y\)-axis, translation right 3, translation down 1.

Mapping: \((x, y) \to (-\tfrac{x}{2} + 3,\ y - 1)\). Parent point \((0,0) \to (3, -1)\); \((4, 2) \to (1, 1)\); \((9, 3) \to (\tfrac{3}{2}, 2)\). The graph "opens" to the left from \((3, -1)\) because of the reflection.

Watch out for:

  • Forgetting to factor \(k\) out of the bracket before reading off \(d\). \(\sqrt{2x - 6}\) is not "shift right 6" — first rewrite as \(\sqrt{2(x - 3)}\), then \(d = 3\).
  • Reversing horizontal effects: an inside factor of \(k = 2\) compresses, not stretches; \((x - 3)\) shifts right, not left.
  • Applying translations before stretches. Stretches/reflections are applied to the parent first; then the translations move the already-transformed graph.
  • Confusing \(-f(x)\) (reflect in \(x\)-axis) with \(f(-x)\) (reflect in \(y\)-axis). The minus sign's position changes everything.

Curriculum expectation: Strand A, Expectation A1.9 — "determine, through investigation using technology, the roles of the parameters \(a\), \(k\), \(d\), and \(c\) in functions of the form \(y = af(k(x - d)) + c\), and describe these roles in terms of transformations on the graphs of \(f(x) = x\), \(f(x) = x^2\), \(f(x) = \sqrt{x}\), \(f(x) = \sin x\), and \(f(x) = \dfrac{1}{x}\) [e.g., translations; reflections in the axes; vertical and horizontal stretches and compressions to and from the \(x\)- and \(y\)-axes]."

1.6 — Even and Odd Functions; Symmetry
Test symmetry algebraically: even iff \( f(-x)=f(x) \) (y-axis symmetric); odd iff \( f(-x)=-f(x) \) (origin symmetric).
\( f(x)=x^4-3x^2 \) is even because \( f(-x)=(-x)^4-3(-x)^2=x^4-3x^2=f(x) \). \( g(x)=x^3-x \) is odd because \( g(-x)=-x^3+x=-g(x) \). Functions like \( h(x)=x^2+x \) are neither.
📖 Lesson Notes — Ontario Curriculum

Symmetry is one of the most powerful organising ideas in mathematics, and functions exhibit two especially useful kinds of it. A function is even when its graph is symmetric in the \(y\)-axis — flipping the graph left-to-right leaves it unchanged. Algebraically this means replacing \(x\) with \(-x\) gives back the same function: \(f(-x) = f(x)\). The classic example is \(y = x^2\): squaring kills the sign of the input.

A function is odd when its graph has rotational symmetry about the origin — rotating 180° around the origin maps the graph onto itself. Algebraically: \(f(-x) = -f(x)\). The classic example is \(y = x^3\): cubing preserves the sign, but negating the input negates the whole output.

Why test for symmetry? Two reasons. First, it can cut your work in half — once you've graphed half of an even function, you mirror the rest; for odd, you rotate. Second, symmetry is a structural property that shows up across mathematics and physics. Fourier analysis, integration over symmetric intervals, and many physical models all exploit even/odd behaviour. A function can be even, odd, or neither — and most functions are neither.

  • Even function — \(f(-x) = f(x)\); graph is symmetric in the \(y\)-axis.
  • Odd function — \(f(-x) = -f(x)\); graph has 180° rotational symmetry about the origin.
  • Neither — \(f(-x)\) equals neither \(f(x)\) nor \(-f(x)\); most polynomials with mixed even-and-odd-degree terms fall here.
  • Polynomial heuristic — a polynomial is even when every term has even degree, odd when every term has odd degree (treating constants as degree 0, which is even).

Example 1 — Classify \(f(x) = x^4 - 3x^2\).

Compute \(f(-x) = (-x)^4 - 3(-x)^2 = x^4 - 3x^2\). This equals \(f(x)\). Even.

The polynomial heuristic also confirms this: every term has even degree (4 and 2).

Example 2 — Classify \(g(x) = x^3 - x\).

Compute \(g(-x) = (-x)^3 - (-x) = -x^3 + x = -(x^3 - x) = -g(x)\). Odd.

Both terms have odd degree (3 and 1), confirming the result.

Example 3 — Classify \(h(x) = x^2 + x\).

Compute \(h(-x) = (-x)^2 + (-x) = x^2 - x\). Compare to \(h(x) = x^2 + x\) and \(-h(x) = -x^2 - x\). It equals neither, so neither even nor odd.

The polynomial has a mix of even-degree (\(x^2\)) and odd-degree (\(x\)) terms — that mixing is exactly what breaks both symmetries.

Example 4 — Classify \(p(x) = \dfrac{x}{x^2 + 4}\).

\(p(-x) = \dfrac{-x}{(-x)^2 + 4} = \dfrac{-x}{x^2 + 4} = -p(x)\). Odd.

Watch out for:

  • Saying "if it has no \(y\)-axis symmetry it must be odd." Wrong — most functions are neither.
  • Misreading signs after substituting \(-x\). Especially in expressions like \((-x)^3 = -x^3\) vs. \((-x)^2 = x^2\).
  • Treating a constant function as odd. \(f(x) = c\) is even (\(c \ne 0\)) since \(f(-x) = c = f(x)\). Only \(f(x) = 0\) is both even and odd.
  • Forgetting that for an odd function, \(f(0) = 0\) (because \(f(0) = -f(0)\) forces \(f(0) = 0\)).

(i.e., domain and range, intercepts, increasing/decreasing intervals, symmetry)" — symmetry is interpreted here as the even/odd classification.
