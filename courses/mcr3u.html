1.1 — Relations vs. Functions; the Vertical Line Test
Define a relation; recognise a function from a set of ordered pairs, mapping diagram, table, or graph; apply the vertical line test.
A relation pairs inputs with outputs. A function is a relation in which each input has exactly one output. Graphically, a curve is a function iff every vertical line intersects it at most once. Common counter-example: \( x = y^2 \) (a sideways parabola) fails the test.
📖 Lesson Notes — Ontario Curriculum

A relation is any rule, table, set of ordered pairs, or graph that pairs values from one set (the inputs, or independent variable) with values from another set (the outputs, or dependent variable). A function is a special kind of relation — one with a strict promise: every input is paired with exactly one output. No input can produce two different outputs.

Why does this matter? Because a huge amount of mathematics relies on being able to ask "what is the output when the input is \(x\)?" and get a single, unambiguous answer. If a relation could spit out two values for the same input, you couldn't write \(y = f(x)\) — the symbol \(f(x)\) would be ambiguous. Functions give us a clean input-output machine, which is exactly what we need for modelling real-world processes (height vs. time, temperature vs. depth, cost vs. quantity).

The vertical line test turns this idea into a visual check. Draw any vertical line. If it ever crosses the graph in two or more places, then the \(x\)-value where it crosses is being mapped to multiple \(y\)-values — so the relation isn't a function. If every vertical line crosses the graph at most once, the relation is a function.

  • Relation — any pairing of inputs to outputs (a set of ordered pairs).
  • Function — a relation in which each input has exactly one output.
  • Domain — the set of all permissible inputs.
  • Range — the set of all outputs the function actually produces.
  • Vertical line test — a graph represents a function if and only if every vertical line meets the graph in at most one point.
  • Mapping diagram — a picture with arrows from inputs (left) to outputs (right); a function has at most one arrow leaving each input.

Example 1 — Decide whether each relation is a function.

(a) \(\{(1,2),(2,4),(3,6),(4,8)\}\). Every \(x\)-value is different, so each input has exactly one output. This is a function.

(b) \(\{(0,1),(1,2),(0,-1),(2,3)\}\). The input \(0\) is paired with both \(1\) and \(-1\). Not a function.

(c) The graph of \(x = y^2\) (a sideways parabola). At \(x = 4\), both \(y=2\) and \(y=-2\) lie on the curve. A vertical line at \(x = 4\) hits the curve twice. Not a function.

(d) The graph of \(y = x^2\). For every \(x\), there is exactly one \(y = x^2\). A vertical line meets the curve once. Is a function.

Example 2 — Does \(y^2 + x^2 = 25\) define \(y\) as a function of \(x\)?

Solving for \(y\): \(y = \pm\sqrt{25 - x^2}\). At \(x = 3\), \(y = \pm 4\) — two outputs for one input. The graph is a circle of radius 5, which fails the vertical line test. Not a function.

Watch out for:

  • Confusing the vertical line test with the horizontal line test — vertical decides "is this a function," horizontal decides "is the inverse a function."
  • Assuming any equation involving \(x\) and \(y\) is a function. Circles, sideways parabolas, and many implicit curves are not.
  • Forgetting that a function can repeat outputs — different inputs may share one output (e.g. \(f(-2) = f(2) = 4\) for \(y = x^2\)). What's forbidden is one input sharing two outputs.

1.2 — Function Notation \( f(x) \); Evaluation
Read and write \( f(x) \), \( g(t) \); evaluate at numerical and algebraic inputs; interpret the meaning of \( f(a) \) in context.
For \( f(x)=2x^2-3x+1 \): \( f(-2)=2(4)-3(-2)+1=8+6+1=15 \). For \( f(a+h) \), substitute \( a+h \) wherever \( x \) appears: \( f(a+h)=2(a+h)^2-3(a+h)+1 \). Function notation makes the input-output relationship explicit and is foundational for all later work in MHF4U and MCV4U.
📖 Lesson Notes — Ontario Curriculum

Up until now you've usually written equations as \(y = \text{(stuff with }x\text{)}\). Function notation just renames the output. Instead of \(y\), we write \(f(x)\), read as "\(f\) of \(x\)." The letter \(f\) is the name of the function (it can be \(g\), \(h\), \(P\), \(C\) — anything), and the \(x\) inside the brackets is a placeholder for whatever input you're about to substitute. This change of notation looks tiny but it is enormous: it lets you name many functions in one problem, talk about specific values like \(f(3)\) without ambiguity, and feed one function into another.

Think of \(f\) as a machine. You feed in a number; the machine processes it; out comes a number. \(f(x) = 2x^2 - 3x + 1\) is the instruction sheet for the machine. \(f(-2)\) is the result of feeding \(-2\) into that specific machine. The brackets are not multiplication — \(f(x)\) does not mean \(f\) times \(x\).

Once you're comfortable plugging in numbers, the same idea lets you plug in expressions: \(f(a+h)\), \(f(2x)\), \(f(x+1) - f(x)\). The last one is the key to the difference quotient, which becomes the derivative in calculus. Mastering substitution at this level pays off for years.

  • Function notation — write \(f(x)\) for the output of function \(f\) at input \(x\).
  • Argument — whatever sits inside the brackets (the input being fed in).
  • Evaluate — substitute a specific value or expression for the variable and simplify.
  • Independent variable — the input, usually \(x\); the value you choose.
  • Dependent variable — the output, \(f(x)\); its value depends on \(x\).

Example 1 — Let \(f(x) = 2x^2 - 3x + 1\). Compute \(f(-2)\), \(f(0)\), and \(f(\tfrac{1}{2})\).

\(f(-2) = 2(-2)^2 - 3(-2) + 1 = 2(4) + 6 + 1 = 8 + 6 + 1 = 15\).

\(f(0) = 2(0)^2 - 3(0) + 1 = 1\). (This is the \(y\)-intercept.)

\(f(\tfrac{1}{2}) = 2(\tfrac{1}{4}) - 3(\tfrac{1}{2}) + 1 = \tfrac{1}{2} - \tfrac{3}{2} + 1 = 0\). So \((\tfrac{1}{2}, 0)\) is on the graph.

Example 2 — For the same \(f\), simplify \(f(a+h) - f(a)\).

\(f(a+h) = 2(a+h)^2 - 3(a+h) + 1 = 2(a^2 + 2ah + h^2) - 3a - 3h + 1 = 2a^2 + 4ah + 2h^2 - 3a - 3h + 1\).

\(f(a) = 2a^2 - 3a + 1\).

\(f(a+h) - f(a) = (2a^2 + 4ah + 2h^2 - 3a - 3h + 1) - (2a^2 - 3a + 1) = 4ah + 2h^2 - 3h\).

This expression — and dividing it by \(h\) — is the average rate of change of \(f\) on \([a, a+h]\), foreshadowing calculus.

Example 3 — Given \(g(x) = \sqrt{x + 4}\), find \(g(5)\) and \(g(2t - 3)\).

\(g(5) = \sqrt{5 + 4} = \sqrt{9} = 3\).

\(g(2t - 3) = \sqrt{(2t - 3) + 4} = \sqrt{2t + 1}\).

Watch out for:

  • Reading \(f(x)\) as multiplication. It is a single symbol naming the output, not a product.
  • Forgetting parentheses when substituting negatives. \(f(-2)\) needs \((-2)^2 = 4\), not \(-2^2 = -4\).
  • When substituting an expression like \(a + h\), you must put it in brackets everywhere the variable appears, then expand carefully.
  • Mixing up "find \(x\) when \(f(x) = 5\)" (an equation to solve) with "find \(f(5)\)" (an evaluation).

1.3 — Domain & Range from Graphs and Equations
State domain and range using set-builder and interval notation; analyse restrictions from radicals (\( \sqrt{\,} \ge 0 \)) and rational expressions (denominator \( \ne 0 \)).
For \( f(x)=\sqrt{x-3} \): domain \( \{x\in\mathbb{R} \mid x\ge 3\} \), range \( \{y\in\mathbb{R}\mid y\ge 0\} \). For \( g(x)=\dfrac{1}{x-2} \): domain \( \{x\in\mathbb{R}\mid x\ne 2\} \). Always check both algebraic restrictions and the contextual domain when modelling.
📖 Lesson Notes — Ontario Curriculum

The domain of a function is the set of all inputs you're allowed to feed it. The range is the set of every output the function actually produces. Knowing both tells you the "shape" of the function before you even graph it: where it starts and ends, what gaps exist, and which heights are reachable.

Domain restrictions come from two main sources at this level. Square roots (and any even-index radical) need a non-negative radicand — you can't take a real square root of a negative number, so anything under \(\sqrt{\ \ }\) must be \(\ge 0\). Rational expressions can never have zero in the denominator — division by zero is undefined. A third source, which appears in word problems, is the context: time can't be negative, you can't ship a fractional number of crates, etc.

Range is usually best found from a graph or from a careful look at the function's behaviour: minimum/maximum value, asymptotes, end behaviour. There are two common notations: set-builder like \(\{x \in \mathbb{R} \mid x \ge 3\}\) and interval notation like \([3, \infty)\). Brackets \([\ ]\) include the endpoint; parentheses \((\ )\) exclude it. Infinity always gets a parenthesis because it isn't a real number.

  • Domain — set of all allowable inputs of a function.
  • Range — set of all outputs the function actually attains.
  • Set-builder notation — \(\{x \in \mathbb{R} \mid \text{condition}\}\); read "the set of \(x\) in the reals such that...".
  • Interval notation — \([a,b]\) closed, \((a,b)\) open, \([a,b)\) half-open; \(\infty\) is always paired with a parenthesis.
  • Restriction — an algebraic condition (e.g. denominator \(\ne 0\), radicand \(\ge 0\)) that excludes inputs from the domain.

Example 1 — State the domain and range of \(f(x) = \sqrt{x - 3}\).

Domain: the radicand must satisfy \(x - 3 \ge 0\), so \(x \ge 3\). In set-builder: \(\{x \in \mathbb{R} \mid x \ge 3\}\); in interval notation: \([3, \infty)\).

Range: \(\sqrt{\ \ }\) by convention returns the non-negative root, so outputs are \(\ge 0\). When \(x = 3\), \(f(x) = 0\); as \(x\) grows, \(f(x)\) grows without bound. Range: \([0, \infty)\).

Example 2 — State the domain and range of \(g(x) = \dfrac{1}{x - 2}\).

Domain: denominator cannot be zero, so \(x - 2 \ne 0\) means \(x \ne 2\). Domain: \(\{x \in \mathbb{R} \mid x \ne 2\}\), or \((-\infty, 2) \cup (2, \infty)\).

Range: \(\dfrac{1}{x-2}\) can be any real number except \(0\) (since the numerator is a nonzero constant). The horizontal asymptote is \(y = 0\), and the function approaches but never reaches it. Range: \(\{y \in \mathbb{R} \mid y \ne 0\}\), or \((-\infty, 0) \cup (0, \infty)\).

Example 3 — A ball is thrown with height \(h(t) = -5t^2 + 20t + 1.5\) (metres) where \(t\) is time in seconds.

Algebraically, \(h\) is defined for every real \(t\). But in context, time starts at \(t = 0\), and the ball stops when it hits the ground: solve \(-5t^2 + 20t + 1.5 = 0\) by the quadratic formula to get \(t \approx 4.07\) s.

Domain: \(\{t \in \mathbb{R} \mid 0 \le t \le 4.07\}\). Range: heights from \(0\) up to the vertex value of \(21.5\) m, i.e. \([0, 21.5]\).

Watch out for:

  • Forgetting to set the denominator equal to zero before declaring the domain. Always solve for the forbidden \(x\)-values.
  • Cube roots and other odd-index radicals accept negatives: domain of \(\sqrt[3]{x}\) is all real numbers.
  • Mixing up brackets and parentheses. Use \([\ ]\) only when the endpoint is included; never use \([\ ]\) with \(\infty\).
  • Stating algebraic domain but ignoring contextual restrictions in word problems.
  • Confusing "no \(x\)-intercept" with "no \(y\)-value zero in the range" — those are different statements.

Curriculum expectation: Strand A, Expectation A1.7 — "determine, through investigation, and using the function notation \(f(x)\), key features of the graphs of the functions defined by \(f(x) = x\), \(f(x) = x^2\), \(f(x) = \sqrt{x}\), \(f(x) = \dfrac{1}{x}\), and \(f(x) = \sin x\) (i.e., domain and range, intercepts, increasing/decreasing intervals, symmetry)."

1.4 — Inverse Functions \( f^{-1}(x) \)
Find inverses algebraically (swap \( x \) and \( y \), solve for \( y \)); reflect graphs across \( y = x \); identify when the inverse is itself a function.
For \( f(x)=3x-5 \), the inverse is found by writing \( y=3x-5 \), swapping to get \( x=3y-5 \), and solving: \( f^{-1}(x)=\dfrac{x+5}{3} \). The graph of \( f^{-1} \) is the reflection of \( f \) over the line \( y=x \). The inverse of \( f(x)=x^2 \) is not a function unless we restrict the domain to \( x\ge 0 \).
📖 Lesson Notes — Ontario Curriculum

The inverse of a function reverses what the function does. If \(f\) takes 3 and turns it into 7, then \(f^{-1}\) takes 7 and turns it back into 3. Symbolically, the rule is \(f^{-1}(f(x)) = x\) and \(f(f^{-1}(x)) = x\). Inverses matter because lots of real questions ask the reverse of the original: if revenue depends on price, what price gives a target revenue? If distance depends on time, when do you arrive at a given distance?

To find an inverse algebraically, swap \(x\) and \(y\) in the equation and solve for the new \(y\). The swap is the algebraic version of reflecting the graph in the line \(y = x\) — every point \((a, b)\) on \(f\) becomes \((b, a)\) on \(f^{-1}\). Because of this reflection, the domain of \(f\) becomes the range of \(f^{-1}\), and vice versa.

A critical caveat: the inverse of a function is itself a function only if the original passes the horizontal line test (every output comes from exactly one input — a one-to-one function). \(f(x) = x^2\) fails this: both \(x = 2\) and \(x = -2\) give output 4, so reversing isn't unique. To get a true inverse function we restrict the domain (e.g. \(x \ge 0\)) before inverting, yielding \(f^{-1}(x) = \sqrt{x}\).

  • Inverse function — \(f^{-1}\) reverses \(f\): \(f^{-1}(f(x)) = x\) for every \(x\) in the domain of \(f\).
  • One-to-one — each output comes from exactly one input; only one-to-one functions have functional inverses.
  • Horizontal line test — a graph passes if every horizontal line meets it at most once; equivalent to one-to-one.
  • Line of reflection — graph of \(y = f^{-1}(x)\) is the reflection of \(y = f(x)\) across \(y = x\).
  • Domain restriction — used to force a many-to-one function to become one-to-one before inverting.

Example 1 — Find the inverse of \(f(x) = 3x - 5\).

Step 1: write \(y = 3x - 5\).

Step 2: swap \(x\) and \(y\): \(x = 3y - 5\).

Step 3: solve for \(y\): \(x + 5 = 3y\), so \(y = \dfrac{x + 5}{3}\).

Therefore \(f^{-1}(x) = \dfrac{x + 5}{3}\).

Verify: \(f^{-1}(f(x)) = \dfrac{(3x - 5) + 5}{3} = \dfrac{3x}{3} = x\). Correct.

Example 2 — Find the inverse of \(f(x) = x^2 + 1\), restricted to \(x \ge 0\).

Step 1: \(y = x^2 + 1\).

Step 2: swap: \(x = y^2 + 1\).

Step 3: solve for \(y\): \(y^2 = x - 1\), so \(y = \pm\sqrt{x - 1}\). Because we restricted the original to \(x \ge 0\) (giving outputs \(y \ge 1\)), the range of the original becomes the domain of the inverse, and we take the positive root: \(f^{-1}(x) = \sqrt{x - 1}\).

Domain of \(f^{-1}\): \(x \ge 1\) (matches range of original). Range of \(f^{-1}\): \(y \ge 0\) (matches restricted domain of original).

Watch out for:

  • The symbol \(f^{-1}(x)\) does not mean \(\dfrac{1}{f(x)}\). The \(-1\) is a name, not an exponent. \(\dfrac{1}{f(x)}\) is the reciprocal function, a completely different thing.
  • Skipping the swap step or doing it after solving — always swap first, then solve.
  • Forgetting that the inverse may not be a function. Always check: does the original pass the horizontal line test?
  • When restricting the domain to make an inverse exist, the new range of the inverse must match the chosen restricted domain.

1.5 — Transformations of Functions: \( a f(k(x-d)) + c \)
Identify and apply vertical/horizontal stretches and compressions, reflections, and translations on parent functions \( y=x^2 \), \( y=\sqrt{x} \), \( y=|x| \), \( y=1/x \).
In \( g(x)=a\,f(k(x-d))+c \): \( a \) controls vertical stretch (\(|a|>1\)) or compression and reflects in the x-axis if negative; \( k \) controls horizontal stretch (\(|k|<1\)) or compression and reflects in the y-axis if negative; \( d \) is a horizontal shift; \( c \) is a vertical shift. Always apply stretches and reflections before translations.
📖 Lesson Notes — Ontario Curriculum

A small number of parent functions — \(y = x\), \(y = x^2\), \(y = \sqrt{x}\), \(y = |x|\), \(y = \dfrac{1}{x}\), and later \(y = \sin x\) — generate an enormous family of related curves through four transformations: stretches, compressions, reflections, and translations. The general form bundles all four into one expression:

\[ g(x) = a\,f\bigl(k(x - d)\bigr) + c \]

Each parameter has a well-defined effect. The two outside the bracket affect \(y\)-values directly: \(a\) multiplies the output (vertical stretch by \(|a|\); if \(a<0\), reflect in the \(x\)-axis), and \(c\) adds to the output (vertical translation by \(c\)). The two inside the bracket affect \(x\)-values in the opposite way you might expect: \(k\) divides the \(x\)-coordinates by \(|k|\) (horizontal compression if \(|k|>1\), stretch if \(|k|<1\); reflect in the \(y\)-axis if \(k<0\)), and \(d\) adds to the \(x\)-coordinates (translation right by \(d\) when written as \((x - d)\)).

The fastest way to graph a transformation is to track a few "key points" on the parent and apply the rule \((x, y) \to \bigl(\tfrac{x}{k} + d, \, ay + c\bigr)\). Apply stretches and reflections first, then translations. If the inside isn't already in the factored form \(k(x - d)\), factor first.

  • Vertical stretch/compression — factor \(a\) acting on \(y\). \(|a|>1\) stretch, \(0<|a|<1\) compression, \(a<0\) reflect in \(x\)-axis.
  • Horizontal stretch/compression — factor \(k\) acting on \(x\). \(|k|>1\) compression, \(0<|k|<1\) stretch, \(k<0\) reflect in \(y\)-axis.
  • Translation — \(c\) shifts up (positive) or down (negative); \(d\) shifts right (positive) or left (negative).
  • Mapping rule — \((x, y) \to \bigl(\tfrac{x}{k} + d,\ ay + c\bigr)\) sends every parent point to its image.
  • Parent function — the simplest member of a family from which others are obtained by transformation.

Example 1 — Sketch \(g(x) = -2(x - 3)^2 + 4\) from the parent \(f(x) = x^2\).

Identify: \(a = -2\), \(k = 1\), \(d = 3\), \(c = 4\). So the transformation is: vertical stretch by 2, reflect in the \(x\)-axis, translate right 3, translate up 4.

Parent key points: \((-1, 1), (0, 0), (1, 1), (2, 4), (-2, 4)\).

Apply the mapping \((x, y) \to (x + 3, -2y + 4)\): \((-1,1) \to (2, 2)\), \((0,0)\to(3, 4)\), \((1,1)\to(4, 2)\), \((2,4)\to(5,-4)\), \((-2,4)\to(1,-4)\).

Vertex of \(g\) is \((3, 4)\), opens downward. Sketch a parabola through the mapped points.

Example 2 — Describe the transformations applied to \(y = \sqrt{x}\) to obtain \(y = \sqrt{-2x + 6} - 1\).

Inside the radical, factor out the coefficient of \(x\): \(-2x + 6 = -2(x - 3)\). So \(y = \sqrt{-2(x - 3)} - 1\).

Identify: \(a = 1\), \(k = -2\), \(d = 3\), \(c = -1\). Transformations: horizontal compression by factor 2, reflection in the \(y\)-axis, translation right 3, translation down 1.

Mapping: \((x, y) \to (-\tfrac{x}{2} + 3,\ y - 1)\). Parent point \((0,0) \to (3, -1)\); \((4, 2) \to (1, 1)\); \((9, 3) \to (\tfrac{3}{2}, 2)\). The graph "opens" to the left from \((3, -1)\) because of the reflection.

Watch out for:

  • Forgetting to factor \(k\) out of the bracket before reading off \(d\). \(\sqrt{2x - 6}\) is not "shift right 6" — first rewrite as \(\sqrt{2(x - 3)}\), then \(d = 3\).
  • Reversing horizontal effects: an inside factor of \(k = 2\) compresses, not stretches; \((x - 3)\) shifts right, not left.
  • Applying translations before stretches. Stretches/reflections are applied to the parent first; then the translations move the already-transformed graph.
  • Confusing \(-f(x)\) (reflect in \(x\)-axis) with \(f(-x)\) (reflect in \(y\)-axis). The minus sign's position changes everything.

1.6 — Even and Odd Functions; Symmetry
Test symmetry algebraically: even iff \( f(-x)=f(x) \) (y-axis symmetric); odd iff \( f(-x)=-f(x) \) (origin symmetric).
\( f(x)=x^4-3x^2 \) is even because \( f(-x)=(-x)^4-3(-x)^2=x^4-3x^2=f(x) \). \( g(x)=x^3-x \) is odd because \( g(-x)=-x^3+x=-g(x) \). Functions like \( h(x)=x^2+x \) are neither.
📖 Lesson Notes — Ontario Curriculum

Symmetry is one of the most powerful organising ideas in mathematics, and functions exhibit two especially useful kinds of it. A function is even when its graph is symmetric in the \(y\)-axis — flipping the graph left-to-right leaves it unchanged. Algebraically this means replacing \(x\) with \(-x\) gives back the same function: \(f(-x) = f(x)\). The classic example is \(y = x^2\): squaring kills the sign of the input.

A function is odd when its graph has rotational symmetry about the origin — rotating 180° around the origin maps the graph onto itself. Algebraically: \(f(-x) = -f(x)\). The classic example is \(y = x^3\): cubing preserves the sign, but negating the input negates the whole output.

Why test for symmetry? Two reasons. First, it can cut your work in half — once you've graphed half of an even function, you mirror the rest; for odd, you rotate. Second, symmetry is a structural property that shows up across mathematics and physics. Fourier analysis, integration over symmetric intervals, and many physical models all exploit even/odd behaviour. A function can be even, odd, or neither — and most functions are neither.

  • Even function — \(f(-x) = f(x)\); graph is symmetric in the \(y\)-axis.
  • Odd function — \(f(-x) = -f(x)\); graph has 180° rotational symmetry about the origin.
  • Neither — \(f(-x)\) equals neither \(f(x)\) nor \(-f(x)\); most polynomials with mixed even-and-odd-degree terms fall here.
  • Polynomial heuristic — a polynomial is even when every term has even degree, odd when every term has odd degree (treating constants as degree 0, which is even).

Example 1 — Classify \(f(x) = x^4 - 3x^2\).

Compute \(f(-x) = (-x)^4 - 3(-x)^2 = x^4 - 3x^2\). This equals \(f(x)\). Even.

The polynomial heuristic also confirms this: every term has even degree (4 and 2).

Example 2 — Classify \(g(x) = x^3 - x\).

Compute \(g(-x) = (-x)^3 - (-x) = -x^3 + x = -(x^3 - x) = -g(x)\). Odd.

Both terms have odd degree (3 and 1), confirming the result.

Example 3 — Classify \(h(x) = x^2 + x\).

Compute \(h(-x) = (-x)^2 + (-x) = x^2 - x\). Compare to \(h(x) = x^2 + x\) and \(-h(x) = -x^2 - x\). It equals neither, so neither even nor odd.

The polynomial has a mix of even-degree (\(x^2\)) and odd-degree (\(x\)) terms — that mixing is exactly what breaks both symmetries.

Example 4 — Classify \(p(x) = \dfrac{x}{x^2 + 4}\).

\(p(-x) = \dfrac{-x}{(-x)^2 + 4} = \dfrac{-x}{x^2 + 4} = -p(x)\). Odd.

Watch out for:

  • Saying "if it has no \(y\)-axis symmetry it must be odd." Wrong — most functions are neither.
  • Misreading signs after substituting \(-x\). Especially in expressions like \((-x)^3 = -x^3\) vs. \((-x)^2 = x^2\).
  • Treating a constant function as odd. \(f(x) = c\) is even (\(c \ne 0\)) since \(f(-x) = c = f(x)\). Only \(f(x) = 0\) is both even and odd.
  • Forgetting that for an odd function, \(f(0) = 0\) (because \(f(0) = -f(0)\) forces \(f(0) = 0\)).

2.1 — Quadratic Functions Review: Three Forms
Move fluently between standard \( y=ax^2+bx+c \), vertex \( y=a(x-h)^2+k \), and factored \( y=a(x-r)(x-s) \) forms.
Each form reveals different features: standard gives the y-intercept (\( c \)) immediately; vertex gives the vertex \( (h,k) \); factored gives the x-intercepts \( (r,0), (s,0) \). Conversion between forms is a critical skill: expand factored to get standard; complete the square on standard to get vertex.
📖 Lesson Notes — Ontario Curriculum

A quadratic function — one whose highest power of \(x\) is 2 — can be written in three algebraically equivalent forms, each emphasising different features of the parabola. Choosing the right form for a problem is half the battle; converting between them is the other half.

Standard form \(y = ax^2 + bx + c\) is the one that drops out of expansion. It immediately reveals the \(y\)-intercept (when \(x = 0\), \(y = c\)) and the direction of opening (\(a > 0\) up, \(a < 0\) down). The axis of symmetry sits at \(x = -\dfrac{b}{2a}\). This form is the one most often used in the quadratic formula.

Vertex form \(y = a(x - h)^2 + k\) puts the vertex \((h, k)\) on display and connects directly to function transformations (Chapter 1.5): the parent \(y = x^2\) shifted right \(h\), up \(k\), and stretched by \(a\). This form is best when you need the maximum or minimum value or want to graph by transformation.

Factored form \(y = a(x - r)(x - s)\) shows the \(x\)-intercepts (zeros) \((r, 0)\) and \((s, 0)\) — set each factor to zero and read off the roots. The axis of symmetry is the midpoint \(x = \dfrac{r + s}{2}\); use it to find the vertex without completing the square. Factored form is ideal whenever you know or can find the zeros.

  • Standard form — \(y = ax^2 + bx + c\); reveals \(y\)-intercept.
  • Vertex form — \(y = a(x - h)^2 + k\); reveals vertex \((h, k)\) and axis \(x = h\).
  • Factored form — \(y = a(x - r)(x - s)\); reveals \(x\)-intercepts \(r, s\).
  • Axis of symmetry — vertical line \(x = h\) through the vertex; midpoint of the two zeros.
  • Leading coefficient — \(a\); same value in all three forms; determines direction and stretch.

Example 1 — Convert \(y = 2(x - 1)(x + 3)\) to standard and vertex form.

To standard: expand. \(y = 2(x^2 + 3x - x - 3) = 2(x^2 + 2x - 3) = 2x^2 + 4x - 6\).

To vertex: axis of symmetry is midway between \(x = 1\) and \(x = -3\): \(x = -1\). Then \(y(-1) = 2(-1 - 1)(-1 + 3) = 2(-2)(2) = -8\). So vertex \((-1, -8)\) and \(y = 2(x + 1)^2 - 8\).

Example 2 — Convert \(y = -3(x - 2)^2 + 12\) to standard and factored form.

To standard: \(y = -3(x^2 - 4x + 4) + 12 = -3x^2 + 12x - 12 + 12 = -3x^2 + 12x\).

To factored: factor out \(-3x\): \(y = -3x(x - 4)\). Zeros: \(x = 0, 4\).

Check: from factored, axis is \(x = (0+4)/2 = 2\) — matches the vertex form's \(h = 2\). ✓

Example 3 — A parabola has zeros at \(x = -2\) and \(x = 6\) and passes through \((1, -27)\). Find its equation in all three forms.

Factored: \(y = a(x + 2)(x - 6)\). Substitute \((1, -27)\): \(-27 = a(3)(-5) = -15a\), so \(a = \tfrac{9}{5}\). Hmm, let me redo: \(-27 / -15 = 1.8 = 9/5\). Factored: \(y = \tfrac{9}{5}(x + 2)(x - 6)\).

Standard: expand: \(y = \tfrac{9}{5}(x^2 - 4x - 12) = \tfrac{9}{5}x^2 - \tfrac{36}{5}x - \tfrac{108}{5}\).

Vertex: axis at \(x = (-2 + 6)/2 = 2\); \(y(2) = \tfrac{9}{5}(4)(-4) = -\tfrac{144}{5}\). Vertex form: \(y = \tfrac{9}{5}(x - 2)^2 - \tfrac{144}{5}\).

Watch out for:

  • Forgetting to multiply \(a\) through when converting factored to standard. \((x - r)(x - s)\) expanded does not equal \(a(x-r)(x-s)\) unless you keep the \(a\).
  • Sign errors in vertex form: \(y = (x - 3)^2\) has vertex \((3, 0)\), not \((-3, 0)\).
  • Assuming every quadratic factors over the rationals. Some don't (e.g. \(x^2 + 1\)), and conversion to factored form requires the quadratic formula or completing the square.

2.2 — Factoring Trinomials & Special Patterns
Factor \( ax^2+bx+c \), difference of squares \( a^2-b^2=(a-b)(a+b) \), perfect-square trinomials, and grouping; factor by GCF first.
Examples: \( 6x^2+11x-10=(3x-2)(2x+5) \) (decomposition); \( 9x^2-25=(3x-5)(3x+5) \) (difference of squares); \( x^2+10x+25=(x+5)^2 \) (perfect square). Always factor out a GCF before attempting any other technique.
📖 Lesson Notes — Ontario Curriculum

Factoring is the reverse of expanding. Where expansion turns \((x + 2)(x + 3)\) into \(x^2 + 5x + 6\), factoring goes the other way. Factoring a quadratic is how you turn standard form into factored form, which immediately reveals the zeros — the heart of solving quadratic equations and analysing parabolas.

There is a strict checklist to follow whenever you see a polynomial to factor:

Step 1: Common factor. Look for the greatest common factor across all terms (numerical and variable). Pull it out front. This step is non-negotiable — skipping it usually means the rest of your work gets harder, and it can be the only step needed.

Step 2: Count terms and pick a strategy. Two terms suggest a difference of squares \(a^2 - b^2 = (a - b)(a + b)\). Three terms could be a trinomial: for \(x^2 + bx + c\), find two numbers that multiply to \(c\) and add to \(b\); for \(ax^2 + bx + c\) with \(a \ne 1\), use decomposition — find two numbers that multiply to \(ac\) and add to \(b\), split the middle term, then factor by grouping. Four terms call for grouping.

Step 3: Check for special patterns. Perfect-square trinomials \(a^2 \pm 2ab + b^2 = (a \pm b)^2\) factor instantly. Recognising them saves time.

  • GCF — greatest common factor; the largest expression that divides every term evenly.
  • Decomposition — splitting the middle term so the trinomial can be factored by grouping.
  • Difference of squares — \(a^2 - b^2 = (a - b)(a + b)\); only works for subtraction of two squares.
  • Perfect square trinomial — \(a^2 \pm 2ab + b^2 = (a \pm b)^2\); the middle coefficient is twice the product of the square roots of the outer terms.
  • Sum of squares — \(a^2 + b^2\) does not factor over the real numbers.

Example 1 — Factor \(6x^2 + 11x - 10\) by decomposition.

Compute \(ac = 6 \cdot (-10) = -60\). Find two numbers multiplying to \(-60\) and summing to \(11\): \(15\) and \(-4\).

Split the middle term: \(6x^2 + 15x - 4x - 10\).

Group: \((6x^2 + 15x) + (-4x - 10) = 3x(2x + 5) - 2(2x + 5)\).

Factor out the common binomial: \((2x + 5)(3x - 2)\).

Example 2 — Factor \(9x^2 - 25\) (difference of squares).

Recognise: \(9x^2 = (3x)^2\) and \(25 = 5^2\). So \(9x^2 - 25 = (3x - 5)(3x + 5)\).

Example 3 — Factor \(4x^3 - 36x\) (GCF first, then difference of squares).

GCF: \(4x\). Pull it out: \(4x(x^2 - 9)\).

Now \(x^2 - 9 = (x - 3)(x + 3)\), so the full factorization is \(4x(x - 3)(x + 3)\).

(Skipping the GCF would leave you trying to factor a cubic — the GCF made it easy.)

Example 4 — Factor \(x^2 + 10x + 25\) (perfect square).

Check: \(\sqrt{x^2} = x\), \(\sqrt{25} = 5\), and \(2(x)(5) = 10x\) matches the middle term. So \(x^2 + 10x + 25 = (x + 5)^2\).

Watch out for:

  • Skipping the GCF step. Many "impossible" trinomials become easy once a common factor is removed.
  • Trying to factor a sum of squares like \(x^2 + 9\) — it doesn't factor over the real numbers.
  • Sign errors in decomposition: the two numbers must multiply to \(ac\) (not just \(c\)) and sum to the middle coefficient.
  • Forgetting to write the leading coefficient out front when only the trinomial in brackets factors. After GCF \(2\) from \(2x^2 - 8x + 6\), you get \(2(x^2 - 4x + 3) = 2(x - 1)(x - 3)\) — keep the 2.

2.3 — Completing the Square & the Quadratic Formula
Convert standard form to vertex form by completing the square; derive and apply \( x = \dfrac{-b\pm\sqrt{b^2-4ac}}{2a} \); use the discriminant \( \Delta=b^2-4ac \) to count real roots.
For \( y=2x^2+8x-3 \): factor 2 from x-terms: \( 2(x^2+4x)-3 \); add and subtract 4 inside: \( 2(x^2+4x+4-4)-3 = 2(x+2)^2 - 11 \). Vertex \( (-2,-11) \). The discriminant \( \Delta>0 \) means two real roots; \( \Delta=0 \) one (a double root); \( \Delta<0 \) no real roots.
📖 Lesson Notes — Ontario Curriculum

Completing the square is the algebraic move that converts a quadratic in standard form to vertex form. The vertex form makes the vertex visible, and — applied generally — produces the quadratic formula itself. So this technique is two things at once: a tool for graphing, and a derivation engine for the most important formula in Grade 11.

The idea: \(x^2 + bx\) is "almost" a perfect square. To finish the square, add \(\left(\dfrac{b}{2}\right)^2\) — that's exactly the constant that makes \(x^2 + bx + \left(\dfrac{b}{2}\right)^2 = \left(x + \dfrac{b}{2}\right)^2\). But you can't just add a number to an expression without changing its value, so you add and subtract the same amount, keeping the equation balanced.

If the leading coefficient \(a\) is not 1, factor \(a\) out of the \(x\)-terms first, complete the square inside the bracket, and then multiply back out. When the procedure is applied symbolically to \(ax^2 + bx + c = 0\), the result is the quadratic formula:

\[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]

The expression under the radical, \(\Delta = b^2 - 4ac\), is the discriminant. It tells you immediately how many real roots the equation has — without solving anything: \(\Delta > 0\) gives two real roots, \(\Delta = 0\) gives one repeated root (the vertex sits on the \(x\)-axis), and \(\Delta < 0\) gives no real roots (the parabola never crosses the \(x\)-axis).

  • Completing the square — adding and subtracting \((b/2)^2\) to turn \(x^2 + bx\) into \((x + b/2)^2\).
  • Vertex form — the goal of completing the square: \(y = a(x - h)^2 + k\), vertex \((h, k)\).
  • Quadratic formula — \(x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}\), the general solver for \(ax^2 + bx + c = 0\).
  • Discriminant \(\Delta = b^2 - 4ac\) — sign decides the number of real roots: \(>0\) two; \(=0\) one (double); \(<0\) none.

Example 1 — Convert \(y = 2x^2 + 8x - 3\) to vertex form by completing the square.

Factor 2 from the \(x\)-terms only: \(y = 2(x^2 + 4x) - 3\).

Half of 4 is 2; square it to get 4. Add and subtract 4 inside the bracket: \(y = 2(x^2 + 4x + 4 - 4) - 3 = 2[(x + 2)^2 - 4] - 3\).

Distribute the 2: \(y = 2(x + 2)^2 - 8 - 3 = 2(x + 2)^2 - 11\).

Vertex: \((-2, -11)\); parabola opens upward.

Example 2 — Solve \(3x^2 - 5x - 2 = 0\) using the quadratic formula.

Identify \(a = 3\), \(b = -5\), \(c = -2\). Discriminant: \(\Delta = (-5)^2 - 4(3)(-2) = 25 + 24 = 49\). Since \(\Delta > 0\), expect two real roots.

\(x = \dfrac{-(-5) \pm \sqrt{49}}{2(3)} = \dfrac{5 \pm 7}{6}\). So \(x = \dfrac{12}{6} = 2\) or \(x = \dfrac{-2}{6} = -\dfrac{1}{3}\).

Example 3 — Use the discriminant to determine the nature of the roots of \(x^2 - 6x + 9 = 0\) and \(x^2 + x + 1 = 0\) without solving.

First: \(\Delta = 36 - 36 = 0\), so one repeated real root (the parabola touches the \(x\)-axis at one point).

Second: \(\Delta = 1 - 4 = -3 < 0\), so no real roots (the parabola never crosses the \(x\)-axis).

Watch out for:

  • Forgetting to multiply the subtracted constant by \(a\) when distributing. In Example 1, \(2 \times (-4) = -8\), not \(-4\).
  • Halving \(b\) incorrectly when \(b\) is odd or fractional — keep it as a fraction. For \(x^2 + 5x\), half of 5 is \(\tfrac{5}{2}\), square is \(\tfrac{25}{4}\).
  • Sign mistakes when reading off the vertex: \((x + 2)^2\) corresponds to \(h = -2\), not \(+2\).
  • Misreading the discriminant. A negative discriminant doesn't mean "error" — it means "no real solutions" (the roots are complex, studied in MHF4U/MCV4U).

2.4 — Solving Quadratic Equations
Solve quadratic equations by factoring, by completing the square, or by the quadratic formula; identify which method is most efficient.
Solve \( 2x^2-7x+3=0 \): factor as \( (2x-1)(x-3)=0 \), so \( x=\tfrac{1}{2} \) or \( x=3 \). When factoring fails (e.g. \( x^2+4x+1=0 \)), use the quadratic formula: \( x=\frac{-4\pm\sqrt{12}}{2}=-2\pm\sqrt{3} \).
📖 Lesson Notes — Ontario Curriculum

Solving a quadratic equation means finding the values of \(x\) that make \(ax^2 + bx + c = 0\). Graphically, these are the \(x\)-intercepts of the parabola. There are three methods, and a key skill is choosing the most efficient one for the equation in front of you.

Method 1 — Factoring. If you can factor the quadratic over the rationals, use the zero-product property: if \(PQ = 0\), then \(P = 0\) or \(Q = 0\). Fast, exact, and clean. Try this first when the coefficients are small integers and a quick mental factorisation seems likely.

Method 2 — Completing the square. Convert to vertex form, then solve by isolating the square term. Most useful when an exact-radical answer is needed and factoring fails, or when the problem also asks for the vertex.

Method 3 — Quadratic formula. The universal fallback: \(x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}\) works on every quadratic. Use it whenever factoring is awkward, especially when the discriminant isn't a perfect square.

Before solving by any method, make sure the equation is written as \(ax^2 + bx + c = 0\). Move every term to one side; the other side must be zero. The zero-product property only works on a zero.

  • Root / zero / solution — same idea: a value of \(x\) making the equation true; the \(x\)-intercept of \(y = ax^2 + bx + c\).
  • Zero-product property — \(PQ = 0 \Rightarrow P = 0 \text{ or } Q = 0\). Requires the product to equal zero.
  • Double root — a single value occurring twice when the discriminant is zero; the parabola is tangent to the \(x\)-axis.
  • Extraneous solution — a value satisfying a transformed equation but not the original (rare in pure quadratics; common when squaring both sides of a radical equation).

Example 1 — Solve \(2x^2 - 7x + 3 = 0\) by factoring.

Decompose: \(ac = 6\), find two numbers multiplying to 6, summing to \(-7\): \(-6\) and \(-1\). Split: \(2x^2 - 6x - x + 3 = 0\). Group: \(2x(x - 3) - 1(x - 3) = 0\), so \((2x - 1)(x - 3) = 0\). Zero-product: \(2x - 1 = 0\) gives \(x = \tfrac{1}{2}\); \(x - 3 = 0\) gives \(x = 3\). Solutions: \(x = \tfrac{1}{2}, 3\).

Example 2 — Solve \(x^2 + 4x + 1 = 0\) using the quadratic formula.

\(a = 1, b = 4, c = 1\). Discriminant: \(16 - 4 = 12\).

\(x = \dfrac{-4 \pm \sqrt{12}}{2} = \dfrac{-4 \pm 2\sqrt{3}}{2} = -2 \pm \sqrt{3}\).

Two irrational roots: \(x = -2 + \sqrt{3} \approx -0.27\) and \(x = -2 - \sqrt{3} \approx -3.73\).

Example 3 — Solve \((x + 1)(x - 4) = 6\).

Common mistake: setting each factor equal to 6. The zero-product property requires zero on the right.

Expand and rearrange: \(x^2 - 3x - 4 = 6 \Rightarrow x^2 - 3x - 10 = 0\). Factor: \((x - 5)(x + 2) = 0\). So \(x = 5\) or \(x = -2\).

Example 4 — Solve \(3x^2 - 5x = 0\).

Factor out the GCF: \(x(3x - 5) = 0\). Solutions: \(x = 0\) or \(x = \tfrac{5}{3}\).

Tempting wrong move: divide both sides by \(x\). That throws away the solution \(x = 0\). Never divide by a variable unless you know it's nonzero.

Watch out for:

  • Applying the zero-product property when the right side isn't zero. Move everything to one side first.
  • Dividing both sides by \(x\) (or any variable). You lose the \(x = 0\) root — always factor out \(x\) instead.
  • Sign mistakes in the quadratic formula. Write \(a, b, c\) carefully, including signs. Use brackets when substituting.
  • Simplifying \(\sqrt{12}\) to \(\sqrt{12}\) and leaving it. Always reduce radicals: \(\sqrt{12} = 2\sqrt{3}\).

2.5 — Quadratic Applications: Optimization & Projectiles
Set up quadratic models for projectile motion, area maximization, and revenue/profit; identify the vertex as the optimal value.
A ball is thrown so its height (m) is \( h(t)=-5t^2+20t+1.5 \). Maximum height occurs at \( t=-\frac{b}{2a}=\frac{-20}{-10}=2 \) s; \( h(2)=-20+40+1.5=21.5 \) m. The ball hits the ground when \( h(t)=0 \), solved by the quadratic formula.
📖 Lesson Notes — Ontario Curriculum

Many physical and economic situations have a hidden parabolic shape. Projectiles (anything thrown, kicked, or launched near Earth) trace parabolic paths because gravity is a constant downward acceleration. Maximising an enclosed area with a fixed perimeter leads to a parabola in width-vs-area space. Revenue and profit problems pair a linear price-vs-demand relationship with a quadratic revenue function. The key skill is to set up the quadratic model and then use the algebra of Chapter 2 to find the answer.

The vertex is the optimal value — the maximum (parabola opening down, \(a < 0\)) or minimum (opening up, \(a > 0\)). The fastest way to find it once the model is in standard form \(y = ax^2 + bx + c\) is the formula \(x = -\dfrac{b}{2a}\) for the \(x\)-coordinate; then substitute to get the \(y\)-coordinate. If the equation is in factored form, average the zeros instead. If in vertex form, just read it off.

Always interpret your answer in context. "Maximum at \(t = 2\)" is incomplete — say "the ball reaches its maximum height of 21.5 m at \(t = 2\) s." Always check that the optimal \(x\) lies in the contextual domain, and write any zero-finding result with appropriate units and rounding.

  • Projectile motion — in metric, near Earth, height (m) at time \(t\) (s): \(h(t) = -\tfrac{1}{2}gt^2 + v_0 t + h_0\) with \(g \approx 9.8\) (or \(\approx 10\) for tidy numbers in MCR3U).
  • Optimization — finding the maximum or minimum value of a quadratic model; equivalent to locating the vertex.
  • Revenue = price × quantity; if price and quantity are linearly related, revenue is quadratic.
  • Profit = revenue \(-\) cost; usually quadratic when revenue is quadratic and cost is linear.
  • Contextual domain — the realistic range of inputs (e.g. \(t \ge 0\)); the algebraic answer must fall inside.

Example 1 — A ball is launched upward. Its height (in metres) after \(t\) seconds is \(h(t) = -5t^2 + 20t + 1.5\). Find the maximum height, when it is reached, and when the ball hits the ground.

Maximum: \(t = -\dfrac{b}{2a} = -\dfrac{20}{2(-5)} = 2\) s.

\(h(2) = -5(4) + 20(2) + 1.5 = -20 + 40 + 1.5 = 21.5\) m.

Hits the ground when \(h(t) = 0\): solve \(-5t^2 + 20t + 1.5 = 0\). Use the quadratic formula with \(a = -5, b = 20, c = 1.5\): discriminant \(= 400 + 30 = 430\); \(t = \dfrac{-20 \pm \sqrt{430}}{-10}\). The positive root is \(t = \dfrac{-20 - \sqrt{430}}{-10} = \dfrac{20 + \sqrt{430}}{10} \approx 4.07\) s.

The ball reaches a maximum height of 21.5 m at \(t = 2\) s and lands after about 4.07 s.

Example 2 — A farmer has 100 m of fencing to enclose a rectangular pen against a barn (only three sides need fencing). Find the dimensions of maximum area.

Let \(w\) be the width perpendicular to the barn and \(\ell\) the length parallel. Fencing: \(2w + \ell = 100\), so \(\ell = 100 - 2w\).

Area: \(A(w) = w \cdot (100 - 2w) = 100w - 2w^2 = -2w^2 + 100w\).

Vertex: \(w = -\dfrac{100}{2(-2)} = 25\). Then \(\ell = 100 - 50 = 50\). Maximum area \(= 25 \cdot 50 = 1250\) m\(^2\).

Example 3 — A shop sells \(150\) hats per week at \$20 each. Market research suggests each \$1 price increase loses 5 sales per week. What price maximises weekly revenue?

Let \(x\) be the number of \$1 increases. Price \(= 20 + x\). Quantity \(= 150 - 5x\). Revenue \(R(x) = (20 + x)(150 - 5x) = 3000 - 100x + 150x - 5x^2 = -5x^2 + 50x + 3000\).

Vertex: \(x = -\dfrac{50}{2(-5)} = 5\). Optimal price \(= 20 + 5 = \$25\). Maximum revenue: \(R(5) = -125 + 250 + 3000 = \$3{,}125\) per week.

Watch out for:

  • Forgetting to substitute back to get the \(y\)-coordinate of the vertex once you've found the \(x\)-coordinate.
  • Choosing the wrong root from the quadratic formula (e.g. negative time). Always check the contextual domain.
  • Skipping units. "21.5" without "m" is incomplete; a marker will deduct.
  • Setting up the wrong quadratic — read the problem carefully and label all variables before writing equations.

3.1 — Exponent Laws & Integer Exponents
Apply the laws of exponents: product, quotient, power-of-a-power, zero, and negative exponents.
Key laws: \( a^m\cdot a^n=a^{m+n} \); \( \dfrac{a^m}{a^n}=a^{m-n} \); \( (a^m)^n=a^{mn} \); \( a^0=1 \) (\(a\ne 0\)); \( a^{-n}=\dfrac{1}{a^n} \). Example: \( \dfrac{(2x^3)^2 \cdot x^{-4}}{x^2} = \dfrac{4x^6 x^{-4}}{x^2}=4x^{6-4-2}=4x^0=4 \).
📖 Lesson Notes — Ontario Curriculum

Exponents are repeated multiplication: \(a^n = a \cdot a \cdot a \cdots a\) (\(n\) factors of \(a\)). All the exponent laws come from this single idea. If you forget a law, write it out longhand and the rule will fall out by counting factors.

The product law \(a^m \cdot a^n = a^{m+n}\) follows because multiplying \(a \cdot a \cdot a\) (3 factors) by \(a \cdot a\) (2 factors) gives 5 factors. The quotient law \(a^m / a^n = a^{m - n}\) follows because dividing cancels factors: 5 factors on top, 2 below, leaves 3 on top. The power-of-a-power law \((a^m)^n = a^{mn}\) follows because raising \((a^m)\) to \(n\) means multiplying \(n\) copies of \(a^m\) — which is \(a^{m + m + \cdots + m} = a^{mn}\).

The trickier extensions are zero and negative exponents. To keep the quotient law consistent, we define \(a^0 = 1\) (because \(a^m / a^m\) must equal both 1 and \(a^{m - m} = a^0\)). Similarly we define \(a^{-n} = \dfrac{1}{a^n}\) because \(a^0 / a^n\) must equal both \(a^{-n}\) and \(\dfrac{1}{a^n}\). These aren't separate laws — they're the only definitions that keep the laws working.

  • Base — the number being multiplied; exponent — how many times.
  • Product law — \(a^m \cdot a^n = a^{m+n}\).
  • Quotient law — \(\dfrac{a^m}{a^n} = a^{m-n}\) (\(a \ne 0\)).
  • Power of a power — \((a^m)^n = a^{mn}\).
  • Power of a product — \((ab)^n = a^n b^n\).
  • Power of a quotient — \(\left(\dfrac{a}{b}\right)^n = \dfrac{a^n}{b^n}\) (\(b \ne 0\)).
  • Zero exponent — \(a^0 = 1\) (\(a \ne 0\)).
  • Negative exponent — \(a^{-n} = \dfrac{1}{a^n}\) (\(a \ne 0\)).

Example 1 — Simplify \(\dfrac{(2x^3)^2 \cdot x^{-4}}{x^2}\), writing the answer with positive exponents.

Apply power of a product to the numerator: \((2x^3)^2 = 4x^6\). So the expression becomes \(\dfrac{4x^6 \cdot x^{-4}}{x^2}\).

Combine in the numerator: \(x^6 \cdot x^{-4} = x^{6 + (-4)} = x^2\). Now \(\dfrac{4x^2}{x^2} = 4 \cdot x^{2-2} = 4 \cdot x^0 = 4\).

Example 2 — Simplify \(\left(\dfrac{3a^{-2}b^3}{6ab^{-1}}\right)^{-2}\).

Simplify inside first: \(\dfrac{3}{6} = \dfrac{1}{2}\), \(\dfrac{a^{-2}}{a} = a^{-2-1} = a^{-3}\), \(\dfrac{b^3}{b^{-1}} = b^{3-(-1)} = b^4\). Inside becomes \(\dfrac{a^{-3} b^4}{2}\).

Now apply the outer \(-2\): \(\left(\dfrac{a^{-3}b^4}{2}\right)^{-2} = \dfrac{a^{6} b^{-8}}{2^{-2}} = \dfrac{4 a^6}{b^8}\).

Example 3 — Evaluate \(2^{-3} + 5^0 - (-3)^2\).

\(2^{-3} = \dfrac{1}{8}\); \(5^0 = 1\); \((-3)^2 = 9\). Sum: \(\dfrac{1}{8} + 1 - 9 = \dfrac{1}{8} - 8 = -\dfrac{63}{8}\).

Watch out for:

  • Multiplying the base by the exponent. \(2^3\) is \(2 \cdot 2 \cdot 2 = 8\), not \(2 \cdot 3 = 6\).
  • \(-3^2\) versus \((-3)^2\). The first is \(-(3^2) = -9\); the second is \(9\). Brackets matter.
  • Trying to use the product law when bases are different. \(2^3 \cdot 3^2\) does not equal \(5^5\) or \(6^5\) — the laws apply only to identical bases.
  • Treating \(0^0\) as 1; it's an indeterminate form (you won't be tested on it at this level, but don't claim it equals 1).
  • Negative exponents do not flip signs of bases; they flip locations: \(a^{-n} = \tfrac{1}{a^n}\), still positive if \(a > 0\).

3.2 — Rational Exponents & Radicals
Convert between radical and rational-exponent form: \( a^{1/n} = \sqrt[n]{a} \) and \( a^{m/n} = \sqrt[n]{a^m} = (\sqrt[n]{a})^m \).
\( 27^{2/3} = (\sqrt[3]{27})^2 = 3^2 = 9 \). \( 16^{-3/4} = \dfrac{1}{16^{3/4}} = \dfrac{1}{(\sqrt[4]{16})^3} = \dfrac{1}{2^3} = \dfrac{1}{8} \). Rational exponents allow us to extend exponent laws beyond integer powers and are essential for working with exponential functions of any base.
📖 Lesson Notes — Ontario Curriculum

You've already learned that \(a^{-n} = \dfrac{1}{a^n}\) and \(a^0 = 1\). Rational exponents finish the job by giving meaning to expressions like \(a^{1/2}\), \(a^{2/3}\), \(a^{0.25}\). The definition is forced on us by the power-of-a-power law: if we want \((a^{1/2})^2 = a^{1/2 \cdot 2} = a^1 = a\), then \(a^{1/2}\) must be a number that squares to give \(a\) — which is exactly \(\sqrt{a}\).

Generalising: \(a^{1/n} = \sqrt[n]{a}\), the \(n\)th root of \(a\). And \(a^{m/n}\) means take the \(n\)th root and raise to the \(m\)th power (or vice versa — they give the same result): \(a^{m/n} = \sqrt[n]{a^m} = (\sqrt[n]{a})^m\). For positive bases this works for any rational exponent. Combine with the negative-exponent rule and you can handle every rational power: \(a^{-m/n} = \dfrac{1}{a^{m/n}}\).

Which order is easier — root first or power first? Almost always root first: small numbers are easier to square than to take square roots of. \(8^{2/3}\) is easier as \((\sqrt[3]{8})^2 = 2^2 = 4\) than as \(\sqrt[3]{64} = 4\), though both work.

  • Rational exponent — an exponent that is a fraction \(m/n\) with integer numerator and positive integer denominator.
  • Root form — \(a^{1/n} = \sqrt[n]{a}\); the index of the radical is the denominator of the exponent.
  • Mixed power-root — \(a^{m/n} = \sqrt[n]{a^m} = (\sqrt[n]{a})^m\).
  • Restriction on negative bases — \(a^{m/n}\) is well-defined for negative \(a\) only when \(n\) is odd (you can take odd roots of negatives, not even roots).
  • Principal root — by convention \(\sqrt[n]{a} \ge 0\) when \(n\) is even and \(a \ge 0\).

Example 1 — Evaluate \(27^{2/3}\).

Root first: \(\sqrt[3]{27} = 3\) (since \(3^3 = 27\)). Then power: \(3^2 = 9\). So \(27^{2/3} = 9\).

Example 2 — Evaluate \(16^{-3/4}\).

Handle the negative first: \(16^{-3/4} = \dfrac{1}{16^{3/4}}\).

\(16^{3/4} = (\sqrt[4]{16})^3 = 2^3 = 8\).

So \(16^{-3/4} = \dfrac{1}{8}\).

Example 3 — Simplify \(\dfrac{x^{1/2} \cdot x^{2/3}}{x^{-1/6}}\), writing the answer as a single power.

Add exponents in the numerator: \(\dfrac{1}{2} + \dfrac{2}{3} = \dfrac{3}{6} + \dfrac{4}{6} = \dfrac{7}{6}\). Numerator: \(x^{7/6}\).

Subtract the denominator's exponent: \(\dfrac{7}{6} - (-\dfrac{1}{6}) = \dfrac{8}{6} = \dfrac{4}{3}\).

Answer: \(x^{4/3}\), or equivalently \(\sqrt[3]{x^4}\).

Example 4 — Rewrite \(\sqrt[5]{x^3}\) using a rational exponent and simplify \(\left(\sqrt[5]{x^3}\right)^{10}\).

\(\sqrt[5]{x^3} = x^{3/5}\). Then \(\left(x^{3/5}\right)^{10} = x^{3/5 \cdot 10} = x^6\).

Watch out for:

  • Reversing numerator and denominator in the exponent. The denominator is the root index; the numerator is the power. So \(a^{2/3}\) is a cube root, not a square root.
  • Taking even roots of negative numbers in the real-number system. \((-16)^{1/2}\) is undefined over the reals.
  • Forgetting to handle negative exponents before doing arithmetic. Convert to fraction first: \(a^{-m/n} = \tfrac{1}{a^{m/n}}\).
  • Reducing \(\dfrac{a^{1/2}}{a^{1/3}}\) by subtracting unlike fractions. Find a common denominator first: \(\dfrac{3}{6} - \dfrac{2}{6} = \dfrac{1}{6}\), so the result is \(a^{1/6}\).

3.3 — The Exponential Function \( y = b^x \)
Graph parent exponentials \( y=2^x, y=10^x, y=(\tfrac{1}{2})^x \); identify domain, range, asymptote, intercepts; distinguish growth (\(b>1\)) from decay (\(0
For \( y=b^x \) with \( b>0, b\ne 1 \): domain \( \mathbb{R} \); range \( y>0 \); horizontal asymptote \( y=0 \); y-intercept always \( (0,1) \) since \( b^0=1 \); no x-intercept. Growth functions rise to the right; decay functions fall to the right.
📖 Lesson Notes — Ontario Curriculum

The exponential function \(f(x) = b^x\) has the variable in the exponent — a profound change from the polynomial functions you've studied. Where polynomials grow at a rate proportional to a power of \(x\), exponentials grow at a rate proportional to themselves: a population that is twice as big today grows twice as fast today. That's the signature of exponential change, and it shows up everywhere — compound interest, populations, radioactivity, viral spread, cooling, learning curves.

The base \(b\) is restricted: we require \(b > 0\) and \(b \ne 1\). If \(b = 1\), the function is constant (\(1^x = 1\) for any \(x\)). If \(b \le 0\), \(b^x\) is not consistently defined for all real \(x\) (e.g. \((-2)^{1/2}\) is not real). With \(b > 0\) and \(b \ne 1\), the function is one-to-one with a clean, continuous graph.

Key features of every \(y = b^x\):

  • Domain: all real numbers, \((-\infty, \infty)\).
  • Range: positive reals, \((0, \infty)\). The graph never touches the \(x\)-axis.
  • \(y\)-intercept: \((0, 1)\) — because \(b^0 = 1\) for any allowed \(b\).
  • Horizontal asymptote: \(y = 0\). For growth (\(b > 1\)), approached as \(x \to -\infty\); for decay (\(0 < b < 1\)), as \(x \to +\infty\).
  • No \(x\)-intercept.
  • Growth when \(b > 1\): rises to the right, falls to the left.
  • Decay when \(0 < b < 1\): falls to the right, rises to the left.
  • One-to-one: so its inverse (the logarithm, studied in MHF4U) is a function.

The bigger \(b\) is (for growth), the steeper the rise. The closer \(b\) is to 1, the slower the change. Decay with base \(b\) and growth with base \(1/b\) are reflections of each other across the \(y\)-axis.

  • Base — the constant \(b\); determines growth vs decay and the rate.
  • Asymptote — a line the curve approaches arbitrarily closely but never reaches. For \(y = b^x\), it's the \(x\)-axis.
  • Common base \(e \approx 2.71828\) — natural exponential, central in calculus; encountered fully in MHF4U.
  • Growth factor — \(b > 1\); each unit increase in \(x\) multiplies \(y\) by \(b\).
  • Decay factor — \(0 < b < 1\); each unit increase in \(x\) multiplies \(y\) by \(b\), shrinking it.

Example 1 — Sketch \(y = 2^x\). Identify domain, range, intercepts, and asymptote.

Make a small table: \(x = -2: y = 1/4\); \(x = -1: y = 1/2\); \(x = 0: y = 1\); \(x = 1: y = 2\); \(x = 2: y = 4\); \(x = 3: y = 8\). Plot and connect smoothly.

Domain \(\mathbb{R}\); range \((0, \infty)\); \(y\)-intercept \((0,1)\); no \(x\)-intercept; horizontal asymptote \(y = 0\) (approached as \(x \to -\infty\)).

Example 2 — Compare \(y = 2^x\) and \(y = (1/2)^x\).

Table for \(y = (1/2)^x\): \(x = -2: y = 4\); \(x = -1: y = 2\); \(x = 0: y = 1\); \(x = 1: y = 1/2\); \(x = 2: y = 1/4\).

Compared to \(2^x\), every \(y\)-value swaps from \(x = a\) to \(x = -a\): \(2^x\) at \(x = 1\) is 2; \((1/2)^x\) at \(x = -1\) is 2. So \((1/2)^x = 2^{-x}\). The graph of \((1/2)^x\) is the reflection of \(2^x\) in the \(y\)-axis. Both share the same horizontal asymptote and \(y\)-intercept.

Example 3 — A function \(f(x) = b^x\) passes through \((2, 9)\). Find \(b\).

\(9 = b^2\), so \(b = 3\) (taking the positive root since \(b > 0\)). The function is \(y = 3^x\).

Watch out for:

  • Confusing exponential growth with a quadratic. \(y = x^2\) doubles slowly; \(y = 2^x\) doubles every unit — by \(x = 10\) it's already 1024 and racing away.
  • Forgetting the horizontal asymptote when sketching — the curve flattens toward \(y = 0\) but doesn't touch it.
  • Mistakenly writing \((1/2)^x\) as \(2^{-x}\) and treating them as different functions — they're identical.
  • Allowing the base to be negative. \(b\) must be positive (and \(\ne 1\)) for an exponential function.

3.4 — Transformations of Exponential Functions
Apply \( y=a\cdot b^{k(x-d)}+c \): identify the new asymptote (\( y=c \)), the y-intercept, and effects of each parameter.
For \( y=2(3)^{x-1}-4 \): the parent \( y=3^x \) is stretched vertically by 2, shifted right 1, and down 4. New horizontal asymptote: \( y=-4 \). At \( x=1 \): \( y=2(3)^0-4=-2 \). At \( x=0 \): \( y=2(3)^{-1}-4=\tfrac{2}{3}-4=-\tfrac{10}{3} \).
📖 Lesson Notes — Ontario Curriculum

Exponential functions follow the same transformation framework as every other function family (Chapter 1.5). The general form is:

\[ y = a \cdot b^{k(x - d)} + c \]

where \(b\) is the original base (\(b > 0, b \ne 1\)) and \(a, k, d, c\) play their usual roles. The only new feature is the horizontal asymptote: the parent \(y = b^x\) has asymptote \(y = 0\), and the vertical translation \(c\) moves it to \(y = c\). This is unique to functions like exponentials and rationals — translating up or down literally moves the asymptote.

The \(y\)-intercept changes in a predictable way. For the parent, \(y(0) = b^0 = 1\). For \(y = a \cdot b^{k(x - d)} + c\), substitute \(x = 0\): \(y(0) = a \cdot b^{-kd} + c\). The \(x\)-intercept may now exist (since the asymptote has moved) — solve \(y = 0\) to find it.

The mapping rule for a key point \((x, y)\) on the parent is:

\[ (x, y) \mapsto \left(\frac{x}{k} + d,\ ay + c\right) \]

Use it on memorable points like \((-1, 1/b), (0, 1), (1, b)\) on the parent.

  • Vertical translation \(c\) — moves the entire graph (including asymptote) up by \(c\).
  • Horizontal translation \(d\) — shifts right by \(d\) when written as \(b^{k(x - d)}\); the asymptote is unchanged.
  • Vertical stretch \(a\) — multiplies all \(y\)-values by \(a\); if \(a < 0\), reflect in \(x\)-axis and the curve dips below the asymptote.
  • Horizontal compression / reflection \(k\) — divides \(x\)-coordinates by \(|k|\); if \(k < 0\), reflect in \(y\)-axis (swaps growth and decay).
  • New asymptote — \(y = c\) after the vertical shift.

Example 1 — Analyse \(y = 2(3)^{x - 1} - 4\). State the transformations, asymptote, \(y\)-intercept, and sketch.

Parent: \(y = 3^x\). Parameters: \(a = 2\), \(k = 1\), \(d = 1\), \(c = -4\). Transformations: vertical stretch by 2; translate right 1; translate down 4.

New asymptote: \(y = -4\).

\(y\)-intercept: \(x = 0\): \(y = 2 \cdot 3^{-1} - 4 = \dfrac{2}{3} - 4 = -\dfrac{10}{3}\). So \((0, -\tfrac{10}{3})\).

Point on parent: \((0, 1)\) maps to \((0 + 1, 2 \cdot 1 - 4) = (1, -2)\). \((1, 3) \mapsto (2, 2)\). \((2, 9)\mapsto (3, 14)\). Sketch through these points with asymptote \(y = -4\).

Example 2 — Find the \(x\)-intercept of \(y = 2(3)^{x - 1} - 4\).

Set \(y = 0\): \(2 \cdot 3^{x-1} - 4 = 0\), so \(3^{x-1} = 2\). Same-base method doesn't help (different bases). Logarithms (MHF4U) would let us solve exactly: \(x - 1 = \log_3 2\), so \(x = 1 + \log_3 2 \approx 1.631\). At this MCR3U level we'd estimate graphically — \(3^{0.6} \approx 1.93\) and \(3^{0.7} \approx 2.16\), so \(x \approx 1.63\).

Example 3 — Describe the transformations applied to \(y = 5^x\) to obtain \(y = -3 \cdot 5^{-2(x + 1)} + 1\).

\(a = -3\) (vertical stretch by 3 and reflection in \(x\)-axis); \(k = -2\) (horizontal compression by factor 2 and reflection in \(y\)-axis); \(d = -1\) (translate left 1); \(c = 1\) (translate up 1). New asymptote: \(y = 1\), and because of the negative \(a\) the curve sits below the asymptote.

Watch out for:

  • Forgetting to factor \(k\) out of the exponent before reading \(d\). \(b^{2x - 4}\) is \(b^{2(x - 2)}\), so \(d = 2\), not 4.
  • Saying the asymptote is always \(y = 0\). It's \(y = c\) after the vertical translation.
  • Skipping reflections when \(a\) or \(k\) is negative.
  • Mixing decay vs growth. The reflection \(b^{-x} = (1/b)^x\) converts growth to decay (and vice versa).

3.5 — Comparing Linear, Quadratic, & Exponential Growth
Compare rates of growth using finite differences and tables; understand why exponential eventually beats any polynomial.
Linear functions have constant first differences; quadratics have constant second differences; exponentials have a constant ratio of consecutive y-values. For large \( x \), \( 2^x \) eventually exceeds \( x^{100} \) — a key intuition for limits and rates of change later in MHF4U.
📖 Lesson Notes — Ontario Curriculum

Linear, quadratic, and exponential functions are the three workhorses of Grade 11 modelling. Recognising which one fits a data set is a high-value skill, and there's a simple table-based test that distinguishes them.

Linear functions \(y = mx + b\) have a constant first difference: every unit step in \(x\) changes \(y\) by the same amount \(m\). Subtract consecutive \(y\)-values; if you get the same number every time, it's linear.

Quadratic functions \(y = ax^2 + bx + c\) have variable first differences but constant second differences. Take differences of the differences; if those are constant, it's quadratic. The constant second difference equals \(2a \cdot (\Delta x)^2\).

Exponential functions \(y = a \cdot b^x\) have a constant ratio: divide consecutive \(y\)-values rather than subtract. \(y_{n+1}/y_n = b\) for every \(n\). No matter how far down the table you go, the multiplier stays the same.

This distinction matters because exponential growth eventually beats any polynomial growth. For small \(x\), \(x^{10}\) towers over \(2^x\) — at \(x = 10\), \(x^{10} = 10^{10} \approx 10^{10}\) versus \(2^{10} = 1024\). But by \(x \approx 60\), \(2^x\) has overtaken \(x^{10}\). For \(x^{100}\) the crossover is later but inevitable. In the long run, no polynomial can keep up with an exponential — a fact that makes compound-interest schemes, viral spread, and computational complexity (e.g. why \(2^n\) algorithms are infeasible) so dramatic.

  • First difference — \(\Delta y = y_{n+1} - y_n\). Constant first difference ⇒ linear.
  • Second difference — \(\Delta(\Delta y)\). Constant second difference ⇒ quadratic.
  • Common ratio — \(y_{n+1}/y_n\). Constant common ratio ⇒ exponential.
  • End behaviour — describes \(y\) as \(x \to \pm \infty\). Exponential growth dominates polynomial growth as \(x \to \infty\).

Example 1 — Classify each table.

(a) \(x: 0, 1, 2, 3, 4\); \(y: 1, 3, 5, 7, 9\). First differences: \(2, 2, 2, 2\) — constant. Linear: \(y = 2x + 1\).

(b) \(x: 0, 1, 2, 3, 4\); \(y: 1, 2, 5, 10, 17\). First differences: \(1, 3, 5, 7\) — not constant. Second differences: \(2, 2, 2\) — constant. Quadratic: \(y = x^2 + 1\).

(c) \(x: 0, 1, 2, 3, 4\); \(y: 1, 2, 4, 8, 16\). Differences: \(1, 2, 4, 8\) — not constant. Ratios: \(2/1, 4/2, 8/4, 16/8 = 2\) — constant. Exponential: \(y = 2^x\).

Example 2 — A bacterial culture grows according to the table:

\(t\) (hours): 0, 1, 2, 3, 4; population: 500, 600, 720, 864, 1036.8.

Ratios: \(600/500 = 1.2\); \(720/600 = 1.2\); \(864/720 = 1.2\); \(1036.8/864 = 1.2\). Constant ratio \(1.2\). The model is exponential: \(P(t) = 500 \cdot 1.2^t\) — a 20% per-hour growth rate.

Example 3 — Compare long-run growth: which is larger at \(x = 100\), \(y = x^3\) or \(y = 1.05^x\)?

At \(x = 100\): \(x^3 = 10^6 = 1{,}000{,}000\). \(1.05^{100} \approx 131.5\). Polynomial is much bigger.

But at \(x = 500\): \(x^3 = 1.25 \times 10^8\). \(1.05^{500} \approx 3.93 \times 10^{10}\) — exponential dominates.

Even a modest base like 1.05 (5%/period growth) eventually overtakes a cubic. This is the moral lesson of compound interest and a foundation for limits in MHF4U.

Watch out for:

  • Stopping after first differences — if they're not constant, you haven't ruled out quadratic. Compute second differences before claiming "not polynomial."
  • Computing ratios using subtraction. The exponential test is division, not difference.
  • Comparing growth only on small intervals. Exponential lag at first; their dominance shows only as \(x\) grows.
  • Assuming an arithmetic-looking sequence is linear without checking. Sometimes data is approximately linear over the visible range but actually exponential or quadratic.

4.1 — Solving Exponential Equations (Same-Base Method)
Rewrite both sides with a common base, then equate exponents; recognise when the same-base technique applies.
Solve \( 4^{x+1} = 8^{x-2} \): rewrite as \( (2^2)^{x+1}=(2^3)^{x-2} \), so \( 2^{2x+2}=2^{3x-6} \). Equate exponents: \( 2x+2=3x-6 \), giving \( x=8 \). When no common base exists (e.g. \( 2^x=7 \)), logarithms are required — that's MHF4U.
📖 Lesson Notes — Ontario Curriculum

Equations like \(2^x = 8\) or \(4^{x + 1} = 8^{x - 2}\) hide a simple algebraic trick. Because the exponential function \(b^x\) is one-to-one (every output comes from exactly one input), the equation \(b^P = b^Q\) is equivalent to \(P = Q\) — once the bases match, you can drop the bases and solve the resulting equation in the exponents.

The strategy: rewrite both sides with a common base, then equate exponents and solve. The work breaks into three steps.

Step 1: Identify a common base. If both sides are powers of 2 (e.g. 2, 4, 8, 16, 32), use 2. If they're powers of 3, use 3. Sometimes the equation hides a common base behind a different label — \(125 = 5^3\), \(\dfrac{1}{16} = 2^{-4}\), \(\sqrt{27} = 27^{1/2} = 3^{3/2}\).

Step 2: Use the power-of-a-power law \((b^m)^n = b^{mn}\) to express each side as a single power of the common base.

Step 3: Equate the exponents. You now have an ordinary linear (or sometimes quadratic) equation.

This same-base technique handles many MCR3U-level problems. When no common base exists — say \(2^x = 7\) — logarithms are needed, and you'll meet those in MHF4U.

  • One-to-one property — for \(b > 0\), \(b \ne 1\): \(b^P = b^Q\) iff \(P = Q\).
  • Common base — a single base \(b\) into which all numerical bases in the equation can be rewritten as integer or rational powers.
  • Reciprocal as negative exponent — \(\dfrac{1}{b^n} = b^{-n}\); often the bridge that exposes a common base.
  • Radical as fractional exponent — \(\sqrt[n]{b^m} = b^{m/n}\); another bridge.

Example 1 — Solve \(2^x = 32\).

Rewrite 32 as a power of 2: \(32 = 2^5\). Equation: \(2^x = 2^5\), so \(x = 5\).

Example 2 — Solve \(4^{x + 1} = 8^{x - 2}\).

Common base 2: \(4 = 2^2\), \(8 = 2^3\). Rewrite: \((2^2)^{x + 1} = (2^3)^{x - 2}\), so \(2^{2x + 2} = 2^{3x - 6}\).

Equate exponents: \(2x + 2 = 3x - 6\). Solve: \(8 = x\), so \(x = 8\).

Check: \(4^9 = 262144\); \(8^6 = 262144\). ✓

Example 3 — Solve \(9^{x} = \dfrac{1}{27^{x - 1}}\).

Rewrite both sides as powers of 3: \(9 = 3^2\), \(27 = 3^3\). So \((3^2)^x = \dfrac{1}{(3^3)^{x - 1}} = (3^3)^{-(x - 1)} = 3^{-3(x - 1)}\).

Simplify exponents: \(3^{2x} = 3^{-3x + 3}\). Equate: \(2x = -3x + 3\), so \(5x = 3\), \(x = \dfrac{3}{5}\).

Example 4 — Solve \(2^{x^2} = 2^{3x + 4}\).

Equate exponents directly: \(x^2 = 3x + 4\), so \(x^2 - 3x - 4 = 0\). Factor: \((x - 4)(x + 1) = 0\). Solutions: \(x = 4\) or \(x = -1\). Both are valid (the exponent law works for any real exponent), so two solutions.

Watch out for:

  • Trying to equate bases when only the exponents match. You can drop the bases only when the bases are equal (and positive and \(\ne 1\)).
  • Distributing the exponent incorrectly: \((b^m)^n = b^{mn}\), not \(b^{m + n}\).
  • Missing a negative when handling reciprocals. \(\dfrac{1}{b^{x - 1}} = b^{-(x - 1)} = b^{1 - x}\), and the minus sign distributes.
  • Forgetting to check solutions in the original equation, especially when squaring or moving terms across the equation.

4.2 — Exponential Growth Models
Use \( A(t)=A_0(1+r)^t \) and \( A(t)=A_0\,b^{t/T} \) for population growth, viral spread, and continuous appreciation.
A culture starts with 500 cells and doubles every 3 hours. After \( t \) hours: \( A(t)=500\cdot 2^{t/3} \). After 12 hours: \( A(12)=500\cdot 2^4=8000 \) cells. After 1 day: \( A(24)=500\cdot 2^8=128{,}000 \). The form \( b^{t/T} \) is read “b raised to (time per period)” — the doubling time form.
📖 Lesson Notes — Ontario Curriculum

Exponential growth describes any quantity that increases by the same proportion (rather than the same amount) in each fixed time unit. Examples include population growth, viral infections, savings under compound interest, and bacteria in a culture. The defining feature is that the rate of growth is proportional to the current size — a population of 1000 grows ten times as fast as a population of 100.

Two equivalent forms appear in MCR3U:

Rate form: \(A(t) = A_0(1 + r)^t\), where \(A_0\) is the initial amount, \(r\) is the growth rate per period (as a decimal), and \(t\) is the number of periods. Use this when the percent growth per period is given.

Period form: \(A(t) = A_0 \cdot b^{t/T}\), where \(b\) is a growth factor (most often 2 for doubling) and \(T\) is the time it takes the quantity to multiply by \(b\). Use this when the doubling/tripling period is given. The exponent \(t/T\) counts how many \(T\)-long periods have passed.

Both forms are exponentials with positive base; they describe the same kind of growth. Translation between them uses an exponent law: \((1 + r)^t = (1 + r)^{T \cdot t/T} = [(1 + r)^T]^{t/T} = b^{t/T}\) where \(b = (1 + r)^T\). At MCR3U you'll typically be given a model in one form and asked to use it; at MHF4U you'll convert freely with logarithms.

  • Initial value \(A_0\) — the amount at \(t = 0\).
  • Growth rate \(r\) — proportional increase per period (e.g. \(r = 0.05\) for 5%/period).
  • Growth factor \(1 + r\) — multiplier per period; for a 5% increase, factor is 1.05.
  • Doubling time \(T\) — time for the quantity to double.
  • Period — the time unit of \(r\); must match the units of \(t\) (years with years, hours with hours).

Example 1 — A bacterial culture starts with 500 cells and doubles every 3 hours. Find the population after 12 hours and after 1 day.

Doubling form: \(A(t) = 500 \cdot 2^{t/3}\) where \(t\) is in hours.

At \(t = 12\): \(A(12) = 500 \cdot 2^{12/3} = 500 \cdot 2^4 = 500 \cdot 16 = 8000\) cells.

At \(t = 24\): \(A(24) = 500 \cdot 2^{24/3} = 500 \cdot 2^8 = 500 \cdot 256 = 128{,}000\) cells.

Example 2 — A town of 12{,}000 grows at 3% per year. Find the population after 10 years and after 25 years.

Rate form: \(P(t) = 12000(1.03)^t\).

At \(t = 10\): \(P(10) = 12000(1.03)^{10} \approx 12000 \cdot 1.3439 \approx 16{,}127\).

At \(t = 25\): \(P(25) = 12000(1.03)^{25} \approx 12000 \cdot 2.0938 \approx 25{,}126\).

Notice the population a bit more than doubled in 25 years — consistent with a "rule of 70" estimate (70/3 ≈ 23 years to double).

Example 3 — A virus is spreading so the number of cases triples every 4 days. There are 200 cases today. Predict the number after 20 days.

Tripling form: \(C(t) = 200 \cdot 3^{t/4}\).

At \(t = 20\): \(C(20) = 200 \cdot 3^{5} = 200 \cdot 243 = 48{,}600\) cases. The geometric explosion is the hallmark of unchecked exponential growth.

Watch out for:

  • Confusing the growth rate \(r\) with the growth factor \(1 + r\). A 5% increase has \(r = 0.05\) and factor \(1.05\).
  • Mismatched time units. If \(r\) is per year, \(t\) must be in years; otherwise convert.
  • Reading "doubles every 3 hours" as \(2^t\) (forgetting the \(/3\)). At \(t = 1\) hour, the population has grown only by a factor of \(2^{1/3} \approx 1.26\), not by 2.
  • Compounding "5% per year for 10 years" linearly as \(50\%\). Exponential compounding gives \(1.05^{10} - 1 \approx 63\%\), not 50%.

4.3 — Exponential Decay & Half-Life
Use \( A(t)=A_0(\tfrac{1}{2})^{t/h} \) for radioactive decay, drug elimination, and cooling; relate half-life to the decay constant.
Carbon-14 has a half-life of 5730 years. A 100 g sample after \( t \) years: \( A(t)=100(\tfrac{1}{2})^{t/5730} \). After 11{,}460 years: \( A=100(\tfrac{1}{2})^2=25 \) g. Equivalently: \( A(t)=100(0.5)^{t/5730} \). Decay models always have base \( 0
📖 Lesson Notes — Ontario Curriculum

Exponential decay is the mirror image of exponential growth. A quantity decays exponentially when it shrinks by the same proportion in each fixed time unit. The same formulas apply, but with a decay rate or a base less than 1. Classic examples: radioactive isotopes, drug concentrations in the bloodstream, the temperature difference between an object and its surroundings (Newton's law of cooling), and tyre pressure leaking through a slow puncture.

The most common model uses the half-life \(h\) — the time for the quantity to fall to half its current value:

\[ A(t) = A_0 \left(\frac{1}{2}\right)^{t/h} \]

The exponent \(t/h\) counts how many half-lives have passed. After 1 half-life, the amount is \(A_0/2\); after 2, \(A_0/4\); after 3, \(A_0/8\). The "halving" structure is exact, regardless of how much time has passed.

The equivalent rate form is \(A(t) = A_0(1 - r)^t\) when the decay rate per period is \(r\) (e.g. \(r = 0.10\) for 10%/period loss). The growth factor \(1 - r\) is positive and less than 1; that's the algebraic signature of decay. You can also write decay using base \(b\) directly: any \(0 < b < 1\) gives a decay function as \(t\) increases.

Half-life problems are central to radiometric dating (carbon-14 for ages up to ~50{,}000 years, uranium-lead for billions), to pharmacology (drug dosing intervals), and to nuclear safety. The mathematics is identical to compound interest with a negative rate.

  • Half-life \(h\) — time for the quantity to reduce to half its current value.
  • Decay rate \(r\) — proportional decrease per period; \(0 < r < 1\).
  • Decay factor \(1 - r\) — multiplier per period; less than 1.
  • Asymptote \(y = 0\) — decay functions approach but never reach zero; mathematically the quantity is never exactly zero, though physically it becomes negligibly small.

Example 1 — Carbon-14 has a half-life of 5730 years. A fossil starts with 100 g of C-14. How much remains after 11{,}460 years? After 17{,}190 years?

Model: \(A(t) = 100 \left(\dfrac{1}{2}\right)^{t/5730}\).

At \(t = 11{,}460\): exponent is \(11460/5730 = 2\). \(A = 100 \cdot (1/2)^2 = 25\) g.

At \(t = 17{,}190\): exponent is \(3\). \(A = 100 \cdot (1/2)^3 = 12.5\) g.

Example 2 — A car loses 15% of its value each year. If it's worth \$25{,}000 today, what's it worth in 5 years? In 10 years?

Rate form: \(V(t) = 25000 (1 - 0.15)^t = 25000(0.85)^t\).

At \(t = 5\): \(V(5) = 25000(0.85)^5 \approx 25000 \cdot 0.4437 \approx \$11{,}092\).

At \(t = 10\): \(V(10) = 25000(0.85)^{10} \approx 25000 \cdot 0.1969 \approx \$4{,}922\).

Example 3 — A drug has a half-life of 6 hours in the bloodstream. A patient receives a 240 mg dose. Estimate the amount remaining after 6, 12, 24, and 48 hours.

Model: \(D(t) = 240(1/2)^{t/6}\).

\(t = 6\): one half-life, 120 mg.

\(t = 12\): two half-lives, 60 mg.

\(t = 24\): four half-lives, \(240/16 = 15\) mg.

\(t = 48\): eight half-lives, \(240/256 \approx 0.94\) mg — clinically negligible.

Watch out for:

  • Using a growth rate when you mean decay. Decay needs base \(b < 1\) or \(1 - r\) (subtraction), not \(1 + r\).
  • Inserting the half-life into the wrong place: \(t/h\), not \(h \cdot t\).
  • Reading "decreased by 15%" as "now equals 15%". It's a 15% loss; the remaining factor is 0.85, not 0.15.
  • Computing decay for negative \(t\): the formula still works mathematically, but means "how much existed in the past" — physically requires care.

4.4 — Compound Interest as an Exponential Application
Apply \( A=P(1+i)^n \) where \( i \) is the rate per compounding period and \( n \) is the number of compounding periods; preview the formal treatment in Ch 6.
$1000 invested at 6% per annum compounded quarterly for 5 years: \( i = 0.06/4=0.015 \), \( n=5\cdot 4=20 \). \( A=1000(1.015)^{20}\approx \$1{,}346.86 \). Exponential growth at work — and the foundation for the more rigorous treatment of present value, future value, and annuities in Chapter 6.
📖 Lesson Notes — Ontario Curriculum

Compound interest is the most important real-world example of exponential growth. Unlike simple interest, where interest is calculated only on the original principal, compound interest earns interest on previously-earned interest. Each compounding period, the new balance is the old balance times \((1 + i)\), where \(i\) is the rate per period. After \(n\) periods, the balance has been multiplied by \((1 + i)\) a total of \(n\) times:

\[ A = P(1 + i)^n \]

Here \(P\) is the principal (initial deposit), \(A\) is the amount after \(n\) periods, \(i\) is the periodic rate (the annual rate divided by the number of compoundings per year), and \(n\) is the total number of compounding periods.

The "trick" — and the reason compound interest builds wealth quietly over decades — is that the interest earned each period sits in the account and earns its own interest the next period. Linear (simple) interest \(\$P + Pin\) is easy to predict but loses out badly over time compared with the exponential model.

This topic previews Chapter 6 (Financial Math). What you need now is the formula, the substitution rules for \(i\) and \(n\), and an understanding that compound interest is just \(y = a(1 + i)^t\) in a financial costume.

  • Principal \(P\) — the initial deposit or loan.
  • Amount \(A\) — total value at the end (principal + interest).
  • Nominal annual rate — the stated yearly rate (e.g. 6%/yr). The periodic rate is this divided by the number of compoundings per year.
  • Periodic rate \(i\) — interest rate per compounding period; \(i = (\text{annual rate}) / (\text{compoundings per year})\).
  • Number of periods \(n\) — total compoundings; \(n = (\text{years}) \times (\text{compoundings per year})\).
  • Compounding frequency — annually (\(m = 1\)), semi-annually (2), quarterly (4), monthly (12), daily (365).

Example 1 — \$1000 invested at 6% per annum compounded quarterly for 5 years.

Periodic rate: \(i = 0.06 / 4 = 0.015\). Number of periods: \(n = 5 \times 4 = 20\).

\(A = 1000(1.015)^{20}\). Compute \(1.015^{20} \approx 1.34686\). So \(A \approx \$1{,}346.86\). Total interest: \$346.86.

Example 2 — Compare \$5000 at 4% annually for 20 years, compounded annually vs. monthly.

Annually: \(A = 5000(1.04)^{20} \approx 5000 \cdot 2.1911 \approx \$10{,}955.62\).

Monthly: \(i = 0.04/12 \approx 0.003333\), \(n = 240\). \(A = 5000(1.003333)^{240} \approx 5000 \cdot 2.2226 \approx \$11{,}112.97\).

Monthly compounding earns about \$157 more — small but real, and it scales.

Example 3 — Compound vs simple. \$2000 at 5%/yr for 10 years.

Simple interest: \(I = Prt = 2000 \cdot 0.05 \cdot 10 = \$1000\). Total: \$3000.

Compound (annually): \(A = 2000(1.05)^{10} \approx 2000 \cdot 1.6289 \approx \$3{,}257.79\). Total interest \$1257.79.

Compounding earned an extra \$257.79 over 10 years. Over 30 years the gap widens to thousands.

Watch out for:

  • Using the annual rate as \(i\) when the problem says "compounded quarterly/monthly." Always convert.
  • Forgetting to multiply years by the compounding frequency to get \(n\).
  • Treating "5%" as 5 instead of 0.05.
  • Reaching for simple-interest \(I = Prt\) in a compound problem. Use compound formulas unless the question explicitly says "simple interest."

5.1 — Sequences: Recursive vs. General-Term Definitions
Define a sequence as a function with domain \( \mathbb{N} \); express explicitly (\( t_n = \) formula in \( n \)) or recursively (\( t_n=t_{n-1}+\dots \)).
The Fibonacci sequence \( 1,1,2,3,5,8,\ldots \) is recursive: \( t_n=t_{n-1}+t_{n-2}, \, t_1=t_2=1 \). The same sequence has no simple closed-form. By contrast \( t_n=2n+1 \) gives \( 3,5,7,9,\ldots \) — both forms describe the sequence; they emphasise different patterns.
📖 Lesson Notes — Ontario Curriculum

A sequence is an ordered list of numbers. Each entry is a term, and we usually label them \(t_1, t_2, t_3, \ldots, t_n, \ldots\) — the subscript counts which term we're talking about. A sequence is really a function whose domain is the positive integers (or sometimes \(\{0, 1, 2, \ldots\}\)): you give it a position \(n\) and it returns a term \(t_n\). That's why we call them "discrete functions" — they map integers to numbers.

There are two main ways to describe a sequence:

Explicit (general-term) formula: a formula for \(t_n\) directly in terms of \(n\). Example: \(t_n = 2n + 1\) gives 3, 5, 7, 9, ... — you can jump straight to \(t_{100} = 201\) without computing any other term.

Recursive formula: a rule that defines each term using earlier terms, plus an explicit value for the first term (or two). Example: \(t_1 = 3\), \(t_n = t_{n-1} + 2\) defines the same sequence. To find \(t_{100}\) you'd need to compute every term before it (or recognise it's arithmetic and use a closed form). The Fibonacci sequence \(t_1 = t_2 = 1\), \(t_n = t_{n-1} + t_{n-2}\) is famously recursive — every term depends on the previous two.

Some sequences have both forms; some have only one. Arithmetic and geometric sequences (the next two topics) have clean closed forms. Fibonacci has a closed form (Binet's formula) but it's complicated. Recursive definitions are powerful — they capture natural growth processes — but they're computationally expensive without the closed form.

  • Sequence — an ordered list of numbers, indexed by positive integers (or sometimes starting at 0).
  • Term \(t_n\) — the \(n\)th entry; \(n\) is the term number, \(t_n\) is its value.
  • Explicit (general) formula — \(t_n = \text{(formula in } n)\); allows direct calculation of any term.
  • Recursive formula — defines \(t_n\) using \(t_{n-1}\) (and possibly earlier terms), with one or more initial values.
  • Finite vs infinite — a finite sequence has a last term; an infinite sequence goes on forever.

Example 1 — Find the first five terms of \(t_n = 2n + 1\).

\(t_1 = 3, t_2 = 5, t_3 = 7, t_4 = 9, t_5 = 11\). The sequence increases by 2 each step.

Example 2 — Write a recursive formula for the sequence 5, 10, 20, 40, 80, ...

Each term is twice the previous. Recursive: \(t_1 = 5\), \(t_n = 2 \cdot t_{n-1}\).

Explicit form: \(t_n = 5 \cdot 2^{n-1}\). Check: \(t_3 = 5 \cdot 4 = 20\). ✓

Example 3 — Compute the first six Fibonacci numbers given \(t_1 = t_2 = 1\), \(t_n = t_{n-1} + t_{n-2}\).

\(t_1 = 1, t_2 = 1, t_3 = 1+1 = 2, t_4 = 1+2 = 3, t_5 = 2+3 = 5, t_6 = 3+5 = 8\).

The sequence 1, 1, 2, 3, 5, 8, 13, 21, ... appears in pinecone spirals, sunflower seed heads, rabbit population growth (Fibonacci's original 1202 example), and Pascal's triangle.

Example 4 — The sequence \(t_n = \dfrac{n}{n + 1}\) starts \(\dfrac{1}{2}, \dfrac{2}{3}, \dfrac{3}{4}, \dfrac{4}{5}, \ldots\). Describe its long-term behaviour.

As \(n\) grows, the fraction approaches 1 (since the numerator and denominator differ by 1). The sequence converges to 1 — a concept formalised in calculus.

Watch out for:

  • Mixing up 0-indexing and 1-indexing. The problem statement decides whether \(t_0\) or \(t_1\) is the first term — read carefully.
  • For a recursive definition, forgetting to state the initial term(s). Without \(t_1\), \(t_n = t_{n-1} + 2\) describes infinitely many sequences.
  • Assuming "next term" means "add a constant." Could be add, multiply, square, or anything; look for a pattern.
  • Trying to find a closed form for every sequence. Some don't have nice explicit formulas.

5.2 — Arithmetic Sequences
Identify a constant common difference \( d \); use \( t_n = a + (n-1)d \) where \( a=t_1 \).
Sequence: \( 7, 11, 15, 19, \ldots \); \( a=7, d=4 \). The 25th term is \( t_{25}=7+(25-1)(4)=7+96=103 \). To find which term equals 167: \( 167=7+(n-1)(4) \) gives \( n=41 \). The constant first difference is the function-class signature of an arithmetic (linear) sequence.
📖 Lesson Notes — Ontario Curriculum

An arithmetic sequence is one in which every consecutive pair of terms differs by the same constant. That constant is the common difference, denoted \(d\). To test whether a sequence is arithmetic, subtract consecutive terms; if you always get the same result, it's arithmetic. The first term is denoted \(a\) (or \(t_1\)).

From \(a\) and \(d\) you can write the explicit formula:

\[ t_n = a + (n - 1)d \]

The logic: to get from term 1 to term \(n\) you add \(d\) a total of \(n - 1\) times (one less than the term number, because the first term needs zero additions). This formula lets you jump to any term without writing out the sequence.

Arithmetic sequences are discrete linear functions. Plotting \(t_n\) versus \(n\) gives a line with slope \(d\) and \(t\)-intercept \(a - d\). The recursive form is \(t_1 = a\), \(t_n = t_{n-1} + d\) for \(n \ge 2\).

Real-world arithmetic sequences appear whenever something increases by a fixed amount each period: simple-interest balances (interest is calculated on the original principal only), the seats in successive rows of a stadium, the cost of taxi fares with per-kilometre charges, salary scales with annual fixed-dollar raises.

  • Arithmetic sequence — sequence with a constant common difference.
  • First term \(a\) (or \(t_1\)) — the starting value.
  • Common difference \(d\) — the constant added at each step; equals \(t_n - t_{n-1}\).
  • General term formula — \(t_n = a + (n - 1)d\).
  • Discrete linear function — the graph is a set of dots on a straight line; the slope is \(d\).

Example 1 — Given the sequence 7, 11, 15, 19, 23, ..., find \(t_{25}\) and determine which term equals 167.

\(a = 7\), \(d = 11 - 7 = 4\).

\(t_{25} = 7 + (25 - 1)(4) = 7 + 96 = 103\).

For 167: \(167 = 7 + (n - 1)(4)\), so \(160 = 4(n - 1)\), \(n - 1 = 40\), \(n = 41\). The 41st term is 167.

Example 2 — Find the formula for the arithmetic sequence with \(t_5 = 22\) and \(t_{15} = 62\).

Set up two equations:

\(t_5 = a + 4d = 22\); \(t_{15} = a + 14d = 62\).

Subtract: \(10d = 40\), so \(d = 4\). Then \(a = 22 - 16 = 6\).

Formula: \(t_n = 6 + (n - 1)(4) = 4n + 2\).

Check: \(t_5 = 22\) ✓, \(t_{15} = 62\) ✓.

Example 3 — A pile of bricks is stacked so that there are 50 bricks in the bottom row, 47 in the second, 44 in the third, and so on. How many bricks are in the 12th row, and which row has 5 bricks?

\(a = 50\), \(d = -3\). \(t_{12} = 50 + 11(-3) = 50 - 33 = 17\) bricks.

For 5 bricks: \(5 = 50 + (n - 1)(-3)\), so \(-45 = -3(n - 1)\), \(n - 1 = 15\), \(n = 16\). The 16th row.

Watch out for:

  • Using \(t_n = a + nd\) (off by one). Always use \(n - 1\), because the first term needs zero additions of \(d\).
  • Confusing arithmetic with geometric. Arithmetic = constant difference; geometric = constant ratio.
  • Treating a decreasing sequence as not arithmetic. A negative \(d\) is fine.
  • For "between two given terms" problems, always set up two equations and subtract — this isolates \(d\) immediately.

5.3 — Geometric Sequences

Sequence: \( 3, 6, 12, 24, \ldots \); \( a=3, r=2 \). The 10th term is \( t_{10}=3\cdot 2^9=3\cdot 512=1536 \). Geometric sequences are discrete exponential functions — the link between Strands B and C.
📖 Lesson Notes — Ontario Curriculum

A geometric sequence is one in which every consecutive pair of terms has the same ratio. That ratio is the common ratio, denoted \(r\). To test whether a sequence is geometric, divide consecutive terms; if you always get the same number, it's geometric. The first term is \(a\) (or \(t_1\)).

The explicit formula:

\[ t_n = a \cdot r^{n - 1} \]

To get from term 1 to term \(n\), multiply by \(r\) a total of \(n - 1\) times. The structure is the discrete analogue of an exponential function: where the exponential \(y = a \cdot b^x\) is defined for all real \(x\), the geometric sequence is the same expression sampled at integer \(x = n - 1\).

The recursive form: \(t_1 = a\), \(t_n = r \cdot t_{n-1}\).

If \(|r| > 1\), the magnitude of the terms grows without bound (compound interest, population doubling, viral spread). If \(0 < |r| < 1\), they shrink toward zero (radioactive decay, drug elimination, bouncing-ball heights). If \(r = 1\), the sequence is constant. If \(r < 0\), consecutive terms alternate signs (e.g. \(2, -6, 18, -54, \ldots\) with \(r = -3\)).

  • Geometric sequence — sequence with a constant common ratio.
  • First term \(a\) — the starting value.
  • Common ratio \(r\) — multiplier between consecutive terms; \(r = t_n / t_{n-1}\).
  • General term formula — \(t_n = ar^{n - 1}\).
  • Discrete exponential function — plotting \(t_n\) vs \(n\) traces an exponential curve at integer inputs.

Example 1 — For the sequence 3, 6, 12, 24, 48, ..., find \(t_{10}\) and the 7th term.

\(a = 3\), \(r = 6/3 = 2\). \(t_{10} = 3 \cdot 2^{9} = 3 \cdot 512 = 1536\).

\(t_7 = 3 \cdot 2^6 = 3 \cdot 64 = 192\).

Example 2 — A geometric sequence has \(t_3 = 18\) and \(t_6 = 486\). Find \(a\) and \(r\).

\(t_3 = ar^2 = 18\); \(t_6 = ar^5 = 486\).

Divide: \(\dfrac{ar^5}{ar^2} = r^3 = \dfrac{486}{18} = 27\), so \(r = 3\).

Then \(a \cdot 9 = 18\), \(a = 2\). Formula: \(t_n = 2 \cdot 3^{n - 1}\).

Example 3 — A ball is dropped from 200 cm and rebounds to 80% of its previous height each bounce. How high does it bounce on the 5th bounce?

Heights form a geometric sequence with \(a = 200\), \(r = 0.8\). The "5th bounce" is term \(t_6\) if we let \(t_1 = 200\) be the initial drop; or term 5 if \(t_1\) is the first rebound. Reading conventionally: first rebound = \(200 \cdot 0.8 = 160\). The 5th rebound: \(t_5 = 200 \cdot 0.8^5 = 200 \cdot 0.32768 \approx 65.5\) cm.

Example 4 — Identify each sequence as arithmetic, geometric, or neither.

(a) 2, 5, 8, 11, 14. Differences \(3, 3, 3, 3\) constant → arithmetic.

(b) 2, 6, 18, 54, 162. Ratios \(3, 3, 3, 3\) constant → geometric.

(c) 1, 1, 2, 3, 5. Neither constant difference nor constant ratio → neither (Fibonacci).

Watch out for:

  • Using \(t_n = ar^n\). The correct exponent is \(n - 1\) — verify by plugging \(n = 1\).
  • Computing the ratio by subtraction. The geometric test is division.
  • Forgetting alternating signs when \(r < 0\). Track the sign through each step.
  • Mistaking a slowly-growing geometric sequence for an arithmetic one. Early terms can look similar; only the ratio test is definitive.

5.4 — Arithmetic Series \( S_n \)
Sum the first \( n \) terms of an arithmetic sequence: \( S_n = \dfrac{n}{2}\,(2a+(n-1)d) = \dfrac{n}{2}(t_1+t_n) \).
Sum the first 50 odd numbers: \( a=1, d=2 \). \( S_{50}=\tfrac{50}{2}(2(1)+49(2))=25(100)=2500 \). The famous “Gauss trick” — pair the first and last terms — produces this same formula.
📖 Lesson Notes — Ontario Curriculum

A series is the sum of the terms of a sequence. If the sequence is arithmetic, the resulting sum is an arithmetic series. We write \(S_n\) for the sum of the first \(n\) terms:

\[ S_n = t_1 + t_2 + t_3 + \cdots + t_n \]

For arithmetic sequences there are two beautiful formulas, both giving \(S_n\):

\[ S_n = \frac{n}{2}\bigl(2a + (n - 1)d\bigr) \quad \text{or} \quad S_n = \frac{n}{2}(t_1 + t_n) \]

The second form is the cleanest: the sum equals the number of terms times the average of the first and last terms. This works because in an arithmetic sequence the terms are evenly spaced — pair the first and last, the second and second-last, and so on, and each pair has the same sum.

This pairing trick is the famous "Gauss trick" — supposedly an 8-year-old Carl Friedrich Gauss summed 1 + 2 + 3 + ... + 100 in seconds by noticing the 50 pairs each summing to 101: \(50 \cdot 101 = 5050\).

Which formula should you use? If the problem gives you \(a, d, n\), use the first form directly. If it gives you the first and last terms, use the second. Convert between using \(t_n = a + (n - 1)d\) as needed.

  • Series — the sum of terms in a sequence.
  • Partial sum \(S_n\) — the sum of the first \(n\) terms.
  • Pairing trick — \(t_1 + t_n = t_2 + t_{n-1} = \cdots\); each pair has the same sum, so \(S_n = \tfrac{n}{2}(t_1 + t_n)\).
  • Average-term form — \(S_n = n \cdot \overline{t}\), where \(\overline{t} = (t_1 + t_n)/2\) is the average of the first and last terms.

Example 1 — Sum the first 50 odd positive integers \(1 + 3 + 5 + \ldots\)

\(a = 1, d = 2, n = 50\). Using the first formula: \(S_{50} = \dfrac{50}{2}\bigl(2(1) + 49(2)\bigr) = 25(2 + 98) = 25 \cdot 100 = 2500\).

Equivalently: \(t_{50} = 1 + 49(2) = 99\). Then \(S_{50} = \dfrac{50}{2}(1 + 99) = 25 \cdot 100 = 2500\). ✓

(In fact, the sum of the first \(n\) odd integers is always \(n^2\) — a famous identity. Here \(50^2 = 2500\).)

Example 2 — Sum the arithmetic series \(7 + 11 + 15 + \ldots + 103\).

\(a = 7, d = 4, t_n = 103\). Find \(n\) first: \(103 = 7 + (n - 1)(4)\), so \(96 = 4(n - 1)\), \(n - 1 = 24\), \(n = 25\).

\(S_{25} = \dfrac{25}{2}(7 + 103) = \dfrac{25 \cdot 110}{2} = 1375\).

Example 3 — A theatre has 20 rows, the first row with 15 seats and each subsequent row with 2 more seats than the one in front of it. How many seats in total?

\(a = 15, d = 2, n = 20\). \(S_{20} = \dfrac{20}{2}(2(15) + 19(2)) = 10(30 + 38) = 10 \cdot 68 = 680\) seats.

Example 4 — Find \(n\) such that \(2 + 5 + 8 + \ldots\) (arithmetic, \(d = 3\)) sums to 287.

\(a = 2, d = 3\). \(S_n = \dfrac{n}{2}(2(2) + (n - 1)(3)) = \dfrac{n}{2}(4 + 3n - 3) = \dfrac{n}{2}(3n + 1)\).

Set \(\dfrac{n(3n + 1)}{2} = 287\): \(n(3n + 1) = 574\), \(3n^2 + n - 574 = 0\). Quadratic formula: \(n = \dfrac{-1 + \sqrt{1 + 6888}}{6} = \dfrac{-1 + 83}{6} = \dfrac{82}{6}\). Not an integer — let me recheck. Quadratic: \(3n^2 + n - 574 = 0\); \(\Delta = 1 + 4(3)(574) = 6889 = 83^2\). \(n = \dfrac{-1 + 83}{6} = \dfrac{82}{6} \approx 13.67\). Not an integer, so no positive integer \(n\) gives exactly 287. \(S_{13} = 13 \cdot 40 / 2 = 260\); \(S_{14} = 14 \cdot 43 / 2 = 301\). Indeed 287 sits between.

Watch out for:

  • Mixing up \(t_n\) (a single term) with \(S_n\) (the sum of \(n\) terms). \(t_5\) is the 5th term; \(S_5\) is \(t_1 + t_2 + t_3 + t_4 + t_5\).
  • Plugging the last value of the variable instead of the number of terms into the formula. Always compute \(n\) explicitly first.
  • Forgetting that an arithmetic series can have a negative common difference, giving decreasing sums.
  • Using the geometric formula by accident. Arithmetic series formulas have an \(n - 1\) inside a bracket, not a \(r^n\).

5.5 — Geometric Series & Sigma Notation
Sum the first \( n \) terms of a geometric sequence: \( S_n = \dfrac{a(r^n - 1)}{r-1} \) (\( r\ne 1 \)); read and write series in sigma notation.
Sum: \( 2+6+18+\ldots+2(3)^9 \). Here \( a=2, r=3, n=10 \). \( S_{10}=\dfrac{2(3^{10}-1)}{3-1}=\dfrac{2(59048)}{2}=59048 \). In sigma notation: \( \displaystyle\sum_{k=1}^{10} 2\cdot 3^{k-1} \).
📖 Lesson Notes — Ontario Curriculum

A geometric series is the sum of consecutive terms of a geometric sequence. Like the arithmetic case, there is a closed-form formula that avoids actually adding term-by-term — and the derivation is clever and worth seeing once.

Let \(S_n = a + ar + ar^2 + \cdots + ar^{n-1}\). Multiply by \(r\): \(rS_n = ar + ar^2 + \cdots + ar^n\). Subtract: \(S_n - rS_n = a - ar^n\), so \(S_n(1 - r) = a(1 - r^n)\), giving:

\[ S_n = \frac{a(r^n - 1)}{r - 1} = \frac{a(1 - r^n)}{1 - r}, \quad r \ne 1 \]

The two forms are algebraically identical; use whichever keeps signs friendly. For \(|r| > 1\) the first form usually works better; for \(|r| < 1\) the second.

If \(r = 1\), the sequence is constant and \(S_n = na\) directly.

Sigma notation is a compact way to write series. The capital Greek letter \(\Sigma\) (sigma) means "sum." A bound under and over it controls the running index. For example:

\[ \sum_{k = 1}^{n} ar^{k - 1} = a + ar + ar^2 + \cdots + ar^{n - 1} \]

Read aloud: "the sum from \(k\) equal 1 to \(n\) of \(a\) times \(r\) to the \(k - 1\)." The index variable (here \(k\)) is a dummy — you can rename it without changing the sum.

  • Geometric series — sum of consecutive terms of a geometric sequence.
  • Closed form — \(S_n = \dfrac{a(r^n - 1)}{r - 1}\) (\(r \ne 1\)).
  • Sigma notation \(\sum_{k = m}^{n} f(k)\) — sum \(f(m) + f(m+1) + \cdots + f(n)\).
  • Lower / upper index — the values where the summation starts and ends.
  • Index of summation — the variable (often \(k, i, n, j\)) that takes each integer value from lower to upper.

Example 1 — Compute \(2 + 6 + 18 + \ldots + 2 \cdot 3^9\) (10 terms).

\(a = 2, r = 3, n = 10\).

\(S_{10} = \dfrac{2(3^{10} - 1)}{3 - 1} = \dfrac{2(59049 - 1)}{2} = 59048\).

In sigma notation: \(\displaystyle\sum_{k = 1}^{10} 2 \cdot 3^{k - 1} = 59{,}048\).

Example 2 — Compute \(\displaystyle\sum_{k = 1}^{8} 5 \cdot (1/2)^{k - 1}\).

This is geometric with \(a = 5\), \(r = 1/2\), \(n = 8\).

\(S_8 = \dfrac{5 \bigl(1 - (1/2)^8\bigr)}{1 - 1/2} = \dfrac{5(1 - 1/256)}{1/2} = 10 \cdot \dfrac{255}{256} = \dfrac{2550}{256} = \dfrac{1275}{128} \approx 9.961\).

Example 3 — Expand and evaluate \(\displaystyle\sum_{k = 2}^{5} (k^2 + 1)\).

This isn't geometric or arithmetic; expand directly: \(k = 2: 5; k = 3: 10; k = 4: 17; k = 5: 26\). Sum: \(5 + 10 + 17 + 26 = 58\).

Example 4 — A chain letter starts with one person sending the letter to 4 friends, each of whom sends it to 4 more, and so on. How many letters in total are sent after 6 rounds?

Round 1: 4 letters. Round 2: 16. Round 3: 64. Geometric with \(a = 4, r = 4\).

Total after 6 rounds: \(S_6 = \dfrac{4(4^6 - 1)}{4 - 1} = \dfrac{4(4096 - 1)}{3} = \dfrac{4 \cdot 4095}{3} = \dfrac{16380}{3} = 5460\) letters.

Watch out for:

  • Using \(r^{n - 1}\) instead of \(r^n\) inside the formula. The formula has \(r^n\) — that's the exponent of the first omitted term, not the last included term.
  • Dividing by \(r - 1\) when \(r = 1\). For \(r = 1\), the formula is undefined; use \(S_n = na\) instead.
  • Sigma notation off-by-one errors. The number of terms in \(\sum_{k = m}^{n}\) is \(n - m + 1\), not \(n - m\).
  • Interpreting a non-geometric series with the geometric formula. Always verify the ratio is constant before applying the closed form.

6.1 — Compound Interest: Future Value & Present Value
Apply \( A=P(1+i)^n \); solve for \( P \) (present value) by rearranging: \( P=A(1+i)^{-n} \).
How much to invest now to have \$10{,}000 in 8 years at 5%/yr compounded annually? \( P=10000(1.05)^{-8}\approx \$6{,}768.39 \). Present value answers “what's it worth today?”; future value answers “what will it grow to?”.
📖 Lesson Notes — Ontario Curriculum

Compound interest is built on a single formula: \(A = P(1 + i)^n\), where \(P\) is principal, \(i\) is the rate per compounding period, \(n\) is the number of compoundings, and \(A\) is the future value. We already saw this in Chapter 4 as an application of exponential functions. Chapter 6 sharpens the language and starts working backwards: given a target future value, what principal do you need today?

That backwards question is the present value question, and it's just algebra. Rearrange \(A = P(1 + i)^n\) for \(P\):

\[ P = A(1 + i)^{-n} = \frac{A}{(1 + i)^n} \]

The two forms are identical. Both express the principle that a future dollar is worth less than a dollar today — money in your hand can earn interest, so to compensate for waiting, future money must be discounted.

The conceptual upshot: present value (PV) answers "what is a future amount worth today?"; future value (FV) answers "what will a present amount grow to?". They are inverses — multiply by \((1 + i)^n\) to go forward in time; multiply by \((1 + i)^{-n}\) to go backward.

This idea — the time value of money — is the foundation of all of finance. Equating present values is how we compare a lump sum with a stream of payments, evaluate loans, price bonds, and make any decision that involves money over time.

  • Principal \(P\) — initial deposit (or the unknown in a present-value problem).
  • Amount \(A\) — final value (often FV).
  • Future value (FV) — value at a future date; \(FV = PV(1 + i)^n\).
  • Present value (PV) — value today; \(PV = FV(1 + i)^{-n}\).
  • Discount factor — \((1 + i)^{-n}\); shrinks future money to its present equivalent.
  • Time value of money — a dollar today is worth more than a dollar tomorrow.

Example 1 — How much should you invest today to have \$10{,}000 in 8 years at 5%/yr compounded annually?

\(A = 10000, i = 0.05, n = 8\).

\(P = 10000(1.05)^{-8} = \dfrac{10000}{1.05^8} = \dfrac{10000}{1.47746} \approx \$6{,}768.39\).

Example 2 — Bob's parents want \$50{,}000 available for his tuition in 18 years. The bank offers 4%/yr compounded semi-annually. What lump sum should they invest today?

\(A = 50000\), periodic rate \(i = 0.04/2 = 0.02\), \(n = 18 \times 2 = 36\) compoundings.

\(P = 50000(1.02)^{-36} = \dfrac{50000}{1.02^{36}} \approx \dfrac{50000}{2.0399} \approx \$24{,}511\).

Example 3 — \$4500 invested today at 3.5%/yr compounded quarterly. What's it worth in 12 years?

\(i = 0.035/4 = 0.00875\), \(n = 48\).

\(A = 4500(1.00875)^{48} \approx 4500 \cdot 1.5198 \approx \$6{,}839.18\).

Example 4 — Which is the better deal today: \$8000 now or \$10{,}000 in 6 years? Use 4.5%/yr compounded annually for comparison.

Convert the \$10{,}000 to present value: \(PV = 10000(1.045)^{-6} \approx 10000 / 1.3023 \approx \$7{,}679.10\).

\$8000 today is worth more than \$10{,}000 in 6 years (at this rate). Take the \$8000 now.

Watch out for:

  • Using the annual rate as \(i\) when compounding is more frequent. Always set \(i = (\text{annual rate}) / (\text{periods per year})\).
  • Forgetting the negative exponent when discounting. \(PV = FV(1 + i)^{-n}\), not \((1 + i)^n\).
  • Comparing dollar amounts at different dates without first converting to a common point in time.
  • Round-off errors in long compoundings — keep at least 6 decimal places in intermediate work, then round at the end.

6.2 — Compounding Periods & Effective Rates
Convert a nominal annual rate to a periodic rate \( i = j/m \) for \( m \) compounding periods per year; compare effective annual yields.
A 6% annual rate compounded monthly: \( i=0.06/12=0.005 \); after 1 year, factor is \( (1.005)^{12}\approx 1.0617 \), so the effective annual rate is \( \approx 6.17\% \). More frequent compounding → higher effective yield.
📖 Lesson Notes — Ontario Curriculum

Two banks each advertise "6% per year" on a savings account. One compounds annually; the other monthly. They look the same on paper, but they don't give you the same return. The more often interest is compounded, the more times you earn interest-on-interest within a year, and the higher the real annual yield. To compare fairly, we use the effective annual rate (EAR).

A nominal annual rate \(j\) compounded \(m\) times per year produces a periodic rate \(i = j/m\) per period. After one full year (\(m\) periods), \$1 grows to:

\[ (1 + i)^m = \left(1 + \frac{j}{m}\right)^m \]

The effective annual rate is this growth factor minus 1:

\[ \text{EAR} = \left(1 + \frac{j}{m}\right)^m - 1 \]

So 6% compounded annually gives EAR = 6%. 6% compounded monthly gives EAR = \((1.005)^{12} - 1 \approx 6.168\%\). 6% compounded daily gives EAR = \((1 + 0.06/365)^{365} - 1 \approx 6.183\%\). The more frequent the compounding, the higher the EAR — but with rapidly diminishing returns. As \(m \to \infty\) the limit is \(e^{0.06} - 1 \approx 6.184\%\) (continuous compounding, a Grade 12 topic).

EAR is the consumer-facing number you compare. In Canada, mortgages must by law disclose the rate compounded semi-annually (a regulation). Investments may use any frequency. Knowing how to convert between nominal and effective rates is essential for honest comparisons.

  • Nominal annual rate \(j\) — the stated yearly rate, before accounting for compounding.
  • Compounding frequency \(m\) — number of times per year interest is added; \(m = 1\) (annual), 2 (semi-annual), 4 (quarterly), 12 (monthly), 52 (weekly), 365 (daily).
  • Periodic rate \(i\) — \(j/m\), the rate per compounding period.
  • Effective annual rate (EAR) — actual annual return: \((1 + j/m)^m - 1\).
  • Equivalent rates — two different (nominal, frequency) pairs are equivalent if they have the same EAR.

Example 1 — Find the EAR for a 6% nominal rate compounded monthly, quarterly, semi-annually, and annually.

Monthly: \(i = 0.005\), \((1.005)^{12} \approx 1.06168\), EAR \(\approx 6.17\%\).

Quarterly: \(i = 0.015\), \((1.015)^4 \approx 1.06136\), EAR \(\approx 6.14\%\).

Semi-annually: \(i = 0.03\), \((1.03)^2 = 1.0609\), EAR = \(6.09\%\).

Annually: EAR \(= 6.00\%\).

The pattern: more frequent compounding → higher EAR; the gap is small at this rate.

Example 2 — Convert 8% compounded semi-annually to its monthly-compounded equivalent.

EAR: \((1.04)^2 - 1 = 0.0816\) (i.e. 8.16%).

Find monthly \(i\) such that \((1 + i)^{12} = 1.0816\): \(1 + i = 1.0816^{1/12} \approx 1.006558\), so \(i \approx 0.006558\). The equivalent monthly nominal rate is \(12i \approx 0.0787 = 7.87\%/\text{yr}\) compounded monthly.

(This kind of conversion is exactly what's done to find the monthly rate on a Canadian mortgage that's quoted semi-annually.)

Example 3 — Bank A offers 4.9% compounded daily; Bank B offers 5% compounded annually. Which is the better savings rate?

A's EAR: \((1 + 0.049/365)^{365} - 1 \approx 0.05022 = 5.02\%\).

B's EAR: \(5.00\%\) (already annual).

Bank A is slightly better — the more frequent compounding overcomes the lower nominal rate.

Watch out for:

  • Comparing nominal rates with different compounding frequencies. Always convert to EAR first.
  • Forgetting that \(m\) is per year, not the number of periods in the whole problem.
  • Confusing the periodic rate with the effective rate. Periodic = nominal/\(m\); effective = \((1 + i)^m - 1\).
  • Rounding intermediate calculations too aggressively — small rate errors compound into noticeable dollar errors over long time horizons.

6.3 — Future Value of an Ordinary Annuity
Use \( FV = R\,\dfrac{(1+i)^n - 1}{i} \) for equal periodic deposits \( R \) made at the end of each period.
Save \$200 monthly for 30 years at 6%/yr compounded monthly: \( i=0.005, n=360, R=200 \). \( FV=200\cdot\dfrac{1.005^{360}-1}{0.005}\approx \$200{,}903.01 \). Notice the formula is a finite geometric series — Ch 5 in disguise.
📖 Lesson Notes — Ontario Curriculum

An annuity is a stream of equal periodic payments. You contribute \$200 a month to an RRSP; an insurance company pays a retiree \$1500 a month; you make a fixed mortgage payment every month. The mathematics is the same in each case: you're either adding to or drawing from an account that earns interest between payments.

The simplest case is an ordinary simple annuity: payments are made at the end of each compounding period (hence "ordinary"), and the payment period equals the compounding period (hence "simple"). For example, \$200 deposited at the end of every month into an account compounded monthly.

The future value of an ordinary simple annuity is the total amount accumulated just after the final payment. Each payment grows separately by compound interest until that final date, and then they're added together. The formula:

\[ FV = R \cdot \frac{(1 + i)^n - 1}{i} \]

where \(R\) is the regular payment, \(i\) is the periodic interest rate, and \(n\) is the total number of payments.

Where does this formula come from? The first payment compounds for \(n - 1\) periods (growing to \(R(1 + i)^{n - 1}\)); the second for \(n - 2\) (\(R(1 + i)^{n - 2}\)); ...; the last payment for 0 periods (\(R\)). Sum: \(R(1 + i)^{n - 1} + R(1 + i)^{n - 2} + \cdots + R\) — that's a geometric series with \(a = R\), ratio \(1 + i\), \(n\) terms. The geometric-series formula collapses it to the boxed expression. Chapter 5 in disguise.

  • Annuity — series of equal periodic payments.
  • Ordinary annuity — payments at the end of each period.
  • Simple annuity — payment period equals compounding period.
  • Future value of annuity \(FV\) — total just after the last payment.
  • Annuity factor — \(\dfrac{(1 + i)^n - 1}{i}\); the value of one dollar paid every period for \(n\) periods, accumulated to the end.

Example 1 — Save \$200 at the end of every month for 30 years in an account paying 6%/yr compounded monthly. What's the value at the end?

\(R = 200, i = 0.06/12 = 0.005, n = 30 \times 12 = 360\).

\(FV = 200 \cdot \dfrac{(1.005)^{360} - 1}{0.005}\). Compute \(1.005^{360} \approx 6.02258\). Numerator: \(6.02258 - 1 = 5.02258\). Quotient: \(5.02258 / 0.005 = 1004.515\). Then \(FV \approx 200 \cdot 1004.52 \approx \$200{,}903.01\).

Total contributions: \(360 \cdot 200 = \$72{,}000\). Interest earned: \(\approx \$128{,}903\). Most of the final value is compound interest — the magic of starting early.

Example 2 — A teen-saver deposits \$50 each month for 5 years at 3%/yr compounded monthly. Find FV.

\(R = 50, i = 0.0025, n = 60\).

\(FV = 50 \cdot \dfrac{(1.0025)^{60} - 1}{0.0025}\). \(1.0025^{60} \approx 1.16162\). Numerator \(\approx 0.16162\). Quotient \(\approx 64.647\). \(FV \approx 50 \cdot 64.65 \approx \$3{,}232\).

Total contributions: \(50 \cdot 60 = \$3{,}000\). Interest \(\approx \$232\) — modest, but the habit matters more than the rate at this stage.

Example 3 — What monthly payment is needed to reach \$1{,}000{,}000 in 40 years at 7%/yr compounded monthly?

Rearrange the formula for \(R\): \(R = \dfrac{FV \cdot i}{(1 + i)^n - 1}\).

\(i = 7/12 / 100 \approx 0.005833\), \(n = 480\). \((1.005833)^{480} \approx 16.4257\). Denominator: \(15.4257\). Annuity factor: \(15.4257 / 0.005833 \approx 2644.5\).

\(R = 1{,}000{,}000 / 2644.5 \approx \$378.16\) per month.

Compounded over 40 years, this turns about \$181,000 of deposits into a million. Compound interest, again.

Watch out for:

  • Using the annual rate as \(i\). Always use the periodic rate.
  • Counting years instead of payments for \(n\). \(n\) is the total number of payments.
  • Forgetting that the formula assumes payments at the end of each period. "Annuity due" (payment at the beginning) requires multiplying FV by an extra factor of \((1 + i)\).
  • Confusing FV (the goal of saving) with PV (the lump sum needed today).

6.4 — Present Value of an Ordinary Annuity
Use \( PV = R\,\dfrac{1-(1+i)^{-n}}{i} \) to find the lump sum equivalent to a stream of equal future payments.
A retiree wants \$2{,}000/month for 25 years at 4%/yr compounded monthly: \( i=0.04/12 \approx 0.00333, n=300 \). \( PV \approx 2000\cdot \dfrac{1-(1.00333)^{-300}}{0.00333}\approx \$378{,}824 \). The required lump-sum invested today.
📖 Lesson Notes — Ontario Curriculum

The future value of an annuity tells you what a stream of payments grows to. The present value of an annuity tells you what a stream of future payments is worth today — the lump sum that, invested at the same rate, would generate exactly that stream of payments.

The formula for an ordinary simple annuity (payments at the end of each period):

\[ PV = R \cdot \frac{1 - (1 + i)^{-n}}{i} \]

where \(R\) is the regular payment, \(i\) is the periodic rate, \(n\) is the total number of payments.

The reasoning is the same time-value-of-money idea, applied in reverse. The first payment of \(R\) will arrive at the end of period 1; its present value is \(R(1 + i)^{-1}\). The second arrives at end of period 2; PV is \(R(1 + i)^{-2}\). And so on. Sum the PVs: \(R(1 + i)^{-1} + R(1 + i)^{-2} + \cdots + R(1 + i)^{-n}\). That's a geometric series with first term \(R(1 + i)^{-1}\) and ratio \((1 + i)^{-1}\). Summing it produces the boxed formula.

PV-of-annuity problems are everywhere: how much do you need in your retirement account at age 65 to fund \$3{,}000/month for 30 years? How much can you borrow if you can afford \$500/month payments for 5 years? What's a "lottery prize of \$1 million paid as \$50{,}000/yr for 20 years" actually worth today? All are present-value computations.

  • Present value of an annuity \(PV\) — lump sum equivalent to all future payments, discounted to today.
  • Discount factor — \((1 + i)^{-n}\) appears inside the formula.
  • Annuity present-value factor — \(\dfrac{1 - (1 + i)^{-n}}{i}\); the PV of one dollar paid every period for \(n\) periods.
  • Lottery vs lump sum — advertised "millions over 20 years" is always worth less than the headline number due to discounting.

Example 1 — A retiree wants to withdraw \$2{,}000 at the end of every month for 25 years from an account paying 4%/yr compounded monthly. How much must be in the account at retirement?

\(R = 2000, i = 0.04/12 = 0.003333\), \(n = 25 \cdot 12 = 300\).

\(PV = 2000 \cdot \dfrac{1 - (1.003333)^{-300}}{0.003333}\).

Compute \((1.003333)^{300} \approx 2.7138\), so \((1.003333)^{-300} \approx 0.3685\). Numerator: \(1 - 0.3685 = 0.6315\). Quotient: \(0.6315 / 0.003333 \approx 189.45\). Then \(PV \approx 2000 \cdot 189.45 \approx \$378{,}824\).

Example 2 — A car loan offers payments of \$385 per month for 5 years at 6.9%/yr compounded monthly. How much is being borrowed?

\(R = 385, i = 0.069/12 = 0.00575\), \(n = 60\).

\(PV = 385 \cdot \dfrac{1 - (1.00575)^{-60}}{0.00575}\).

\(1.00575^{60} \approx 1.41203\); \((1.00575)^{-60} \approx 0.70820\). Numerator: \(0.29180\). Quotient: \(0.29180 / 0.00575 \approx 50.748\). Then \(PV \approx 385 \cdot 50.75 \approx \$19{,}538\).

Total paid over 5 years: \(385 \cdot 60 = \$23{,}100\). Interest: \$23,100 − \$19,538 ≈ \$3,562.

Example 3 — A lottery advertises a prize of \$50{,}000 per year for 20 years (\$1 million total). With money worth 5%/yr compounded annually, what's the lump-sum equivalent today?

\(R = 50000, i = 0.05, n = 20\).

\(PV = 50000 \cdot \dfrac{1 - (1.05)^{-20}}{0.05} \approx 50000 \cdot \dfrac{1 - 0.3769}{0.05} \approx 50000 \cdot 12.462 \approx \$623{,}111\).

The "\$1 million" prize is actually worth about \$623{,}000 today — a 38% haircut due to discounting.

Watch out for:

  • Forgetting the negative exponent. PV uses \((1 + i)^{-n}\); FV uses \((1 + i)^n\). The minus sign matters.
  • Mixing FV and PV formulas in the same problem.
  • Using the annual rate or counting years instead of payments.
  • Forgetting that the formula assumes the first payment is one period in the future (ordinary annuity). An annuity due (payments at the beginning of the period) multiplies the formula by \((1 + i)\).

6.5 — Mortgages & Amortization
Apply PV-of-annuity to find mortgage payment \( R \); construct an amortization schedule (interest vs. principal).
A \$300{,}000 mortgage at 5%/yr compounded semi-annually, 25-year amortization: convert to monthly equivalent rate, solve \( 300{,}000=R\cdot\dfrac{1-(1+i)^{-300}}{i} \) for \( R\approx \$1{,}744.81 \) per month. Early payments are mostly interest; later payments are mostly principal.
📖 Lesson Notes — Ontario Curriculum

A mortgage is a long-term loan secured against real estate. From a mathematical standpoint it's just a present-value annuity calculation viewed from the lender's perspective: the bank gives you a lump sum today (the loan amount), and you pay it back as a stream of equal monthly payments over the amortization period (commonly 25 years in Canada).

Canadian mortgages have a quirky rate convention: by federal regulation they're quoted with semi-annual compounding, but payments are usually monthly. So the first step is converting the semi-annual rate to its monthly equivalent — find \(i\) such that \((1 + i)^{12} = (1 + j/2)^{2}\), where \(j\) is the quoted annual rate. Then apply the PV-of-annuity formula to solve for the payment \(R\).

Rearranging the PV formula gives the payment:

\[ R = \frac{PV \cdot i}{1 - (1 + i)^{-n}} \]

Once you know \(R\), you can construct an amortization schedule — a table showing how each payment splits between interest and principal. The pattern is striking: early payments are mostly interest; later payments are mostly principal. That's because interest is charged on the outstanding balance, which is largest at the start. The payment is constant, but its split shifts dramatically over time.

Each row of an amortization schedule has four numbers: payment, interest portion (previous balance × \(i\)), principal portion (payment − interest), new balance (previous − principal portion). The final row should end with a balance of zero.

  • Mortgage — loan secured by real property, repaid in equal periodic payments.
  • Amortization period — total time to pay off the loan in full (often 25 or 30 years).
  • Term — length of the interest-rate agreement, often shorter than amortization (e.g. 5-year term within a 25-year amortization); at the end of the term, the rate is renegotiated.
  • Equivalent rate conversion — Canadian convention: \((1 + i_m)^{12} = (1 + i_s)^2\), where \(i_m\) is monthly and \(i_s\) is semi-annual.
  • Amortization schedule — table of every payment, showing interest, principal, and remaining balance.

Example 1 — A \$300{,}000 mortgage at 5%/yr compounded semi-annually, 25-year amortization with monthly payments. Find the monthly payment.

Convert the rate: semi-annual periodic \(= 0.025\); we need monthly \(i\) so \((1 + i)^{12} = (1.025)^2 = 1.050625\). Then \(1 + i = 1.050625^{1/12} \approx 1.0041239\), so \(i \approx 0.0041239\).

\(n = 25 \cdot 12 = 300\).

\(R = \dfrac{300000 \cdot 0.0041239}{1 - (1.0041239)^{-300}}\). Compute \(1.0041239^{300} \approx 3.4378\), so the denominator's discount factor \((1.0041239)^{-300} \approx 0.29089\), and \(1 - 0.29089 = 0.70911\). Numerator: \(300000 \cdot 0.0041239 \approx 1237.18\). Then \(R \approx 1237.18 / 0.70911 \approx \$1{,}744.81\) per month.

Total paid over 25 years: \(300 \cdot 1744.81 \approx \$523{,}443\). Interest paid: \$223,443 — almost as much as the original loan.

Example 2 — First two rows of the amortization schedule for the loan in Example 1.

Month 1: balance \$300{,}000. Interest: \(300000 \cdot 0.0041239 \approx \$1{,}237.17\). Principal: \(1744.81 - 1237.17 \approx \$507.64\). New balance: \(300000 - 507.64 = \$299{,}492.36\).

Month 2: balance \$299{,}492.36. Interest: \(299492.36 \cdot 0.0041239 \approx \$1{,}235.07\). Principal: \(1744.81 - 1235.07 \approx \$509.74\). New balance: \(\approx \$298{,}982.62\).

Notice: of \$1{,}744.81 paid each month, almost \$1,235 disappears as interest in month 2; only \$510 reduces principal. Twenty years in, the split will be reversed.

Example 3 — Same loan, but 20-year amortization. How much do you save in total interest?

\(n = 240\). Repeating the calculation: \(1.0041239^{240} \approx 2.6856\), \((1.0041239)^{-240} \approx 0.37236\), \(1 - 0.37236 = 0.62764\); \(R = 300000 \cdot 0.0041239 / 0.62764 \approx \$1{,}971.04\).

Total paid: \(240 \cdot 1971.04 \approx \$473{,}050\). Interest: \$173,050. Savings versus 25-year: \$223,443 − \$173,050 = \$50,393. A shorter amortization saves real money.

Watch out for:

  • Forgetting the semi-annual-compounding regulation. In Canadian MCR3U problems, the rate is quoted compounded semi-annually unless stated otherwise.
  • Mixing up term and amortization. The 5-year term lets the bank reset the rate every 5 years, but the loan is fully paid only at the end of the (longer) amortization.
  • Using the annual rate for the monthly calculation directly.
  • Rounding the monthly rate too early; small errors balloon over hundreds of payments.

7.1 — Primary & Reciprocal Trigonometric Ratios
Define \( \sin\theta = \tfrac{\text{opp}}{\text{hyp}}, \cos\theta = \tfrac{\text{adj}}{\text{hyp}}, \tan\theta = \tfrac{\text{opp}}{\text{adj}} \) and the reciprocals \( \csc, \sec, \cot \) for acute angles.
In a right triangle with opposite 5, adjacent 12, hypotenuse 13: \( \sin\theta=\tfrac{5}{13}, \cos\theta=\tfrac{12}{13}, \tan\theta=\tfrac{5}{12} \) and \( \csc\theta=\tfrac{13}{5}, \sec\theta=\tfrac{13}{12}, \cot\theta=\tfrac{12}{5} \). Reciprocal ratios extend the toolkit beyond the three primary ratios.
📖 Lesson Notes — Ontario Curriculum

The three primary trigonometric ratios connect the angles of a right triangle to the lengths of its sides. For an acute angle \(\theta\), define:

\[ \sin\theta = \frac{\text{opposite}}{\text{hypotenuse}}, \quad \cos\theta = \frac{\text{adjacent}}{\text{hypotenuse}}, \quad \tan\theta = \frac{\text{opposite}}{\text{adjacent}} \]

Remember "SOH-CAH-TOA." The opposite side is across from \(\theta\); the adjacent is the leg touching \(\theta\) (not the hypotenuse); the hypotenuse is the longest side, opposite the right angle. Each ratio depends only on the angle \(\theta\), not on the triangle's size — similar triangles share trig ratios, which is why these are well-defined functions of \(\theta\).

The three reciprocal ratios flip the primaries:

\[ \csc\theta = \frac{1}{\sin\theta} = \frac{\text{hyp}}{\text{opp}}, \quad \sec\theta = \frac{1}{\cos\theta} = \frac{\text{hyp}}{\text{adj}}, \quad \cot\theta = \frac{1}{\tan\theta} = \frac{\text{adj}}{\text{opp}} \]

Memorisation aid: csc reciprocates sin, sec reciprocates cos, cot reciprocates tan — pairing each "co-" with the opposite primary.

Why six ratios? In solving any triangle you may need ratios both ways: sometimes you know opposite-and-hypotenuse and want \(\theta\); sometimes you know hypotenuse-and-opposite and want a different ratio expressed cleanly. The reciprocals also let you write identities and formulas more compactly.

  • SOH-CAH-TOA — mnemonic for the three primary ratios.
  • Opposite, adjacent, hypotenuse — three labelled sides of a right triangle; opposite/adjacent are relative to the angle of interest.
  • Reciprocal ratios — cosecant (csc), secant (sec), cotangent (cot); each is one over a primary ratio.
  • Pythagorean identity — \(\sin^2\theta + \cos^2\theta = 1\); follows from Pythagoras \(a^2 + b^2 = c^2\) divided by \(c^2\).
  • Quotient identity — \(\tan\theta = \dfrac{\sin\theta}{\cos\theta}\); follows directly from the definitions.

Example 1 — A right triangle has opposite leg 5, adjacent leg 12, hypotenuse 13 (a Pythagorean triple). Find all six trig ratios of the angle \(\theta\) opposite the side of length 5.

\(\sin\theta = \dfrac{5}{13}, \cos\theta = \dfrac{12}{13}, \tan\theta = \dfrac{5}{12}\).

\(\csc\theta = \dfrac{13}{5}, \sec\theta = \dfrac{13}{12}, \cot\theta = \dfrac{12}{5}\).

Verify Pythagorean identity: \(\sin^2\theta + \cos^2\theta = \dfrac{25}{169} + \dfrac{144}{169} = \dfrac{169}{169} = 1\). ✓

Example 2 — Given that \(\sin\theta = \dfrac{3}{5}\) and \(\theta\) is acute, find \(\cos\theta\), \(\tan\theta\), and \(\sec\theta\) without a calculator.

Sketch a right triangle: opposite = 3, hypotenuse = 5. By Pythagoras: adjacent \(= \sqrt{25 - 9} = 4\).

\(\cos\theta = \dfrac{4}{5}\); \(\tan\theta = \dfrac{3}{4}\); \(\sec\theta = \dfrac{5}{4}\).

Example 3 — A ladder leans against a wall. The foot is 1.8 m from the wall and the angle with the ground is 70°. How long is the ladder, and how high up the wall does it reach?

The ladder is the hypotenuse \(h\). Adjacent to the 70° angle is 1.8 m (along the ground).

\(\cos 70° = \dfrac{1.8}{h}\), so \(h = \dfrac{1.8}{\cos 70°} \approx \dfrac{1.8}{0.342} \approx 5.26\) m.

Wall height (opposite to 70°): \(\tan 70° = \dfrac{\text{height}}{1.8}\), so height \(\approx 1.8 \cdot 2.747 \approx 4.94\) m.

Watch out for:

  • Confusing opposite and adjacent. Always identify them relative to the angle of interest; rotating which angle you focus on swaps which leg is which.
  • Using degrees vs radians. MCR3U is degree-mode; make sure your calculator agrees.
  • Computing \(\sin^{-1}\) as \(\dfrac{1}{\sin}\) — they're different things. \(\sin^{-1}\) is inverse sine (gives an angle); \(\dfrac{1}{\sin}\) is the reciprocal (cosecant).
  • Forgetting to write units (degrees, metres) in final answers.

7.2 — Trigonometric Ratios of Any Angle (0°–360°)
Use the unit circle and the CAST rule to determine the sign of each trig ratio; find related/reference angles.
For \( \theta \) in standard position with terminal arm passing through \( (-3, 4) \): \( r=5 \), so \( \sin\theta=4/5, \cos\theta=-3/5, \tan\theta=-4/3 \). The reference angle is \( \tan^{-1}(4/3)\approx 53.13° \); the principal angle is \( 180°-53.13°=126.87° \). Quadrant II: only sine positive (CAST).
📖 Lesson Notes — Ontario Curriculum

Right-triangle trig only works for acute angles between 0° and 90°. To extend the trig ratios to any angle — including 0°, 90°, 180°, negative angles, angles bigger than 360° — we use the unit circle and the idea of an angle in standard position.

An angle is in standard position when its vertex is at the origin and its initial arm lies along the positive \(x\)-axis. The terminal arm is rotated counter-clockwise by the angle measure (clockwise for negative angles). The terminal arm crosses points \((x, y)\); at distance \(r = \sqrt{x^2 + y^2}\) from the origin, we define:

\[ \sin\theta = \frac{y}{r}, \quad \cos\theta = \frac{x}{r}, \quad \tan\theta = \frac{y}{x} \]

For acute angles in the first quadrant this matches SOH-CAH-TOA. But now negative \(x\) or \(y\) values are allowed, so the ratios can be positive or negative depending on which quadrant the terminal arm lands in.

The CAST rule tells you which ratios are positive in each quadrant. Starting from Quadrant IV and going counter-clockwise: C (Q IV, cosine positive), A (Q I, all positive), S (Q II, sine positive), T (Q III, tangent positive). The other ratios in each quadrant are negative.

The reference angle is the acute angle the terminal arm makes with the \(x\)-axis (positive or negative). Trig values for any angle in standard position equal \(\pm\) the trig values of its reference angle — the sign is determined by CAST.

  • Standard position — vertex at origin, initial arm along positive \(x\)-axis.
  • Terminal arm — the rotated ray that defines the angle's measure.
  • \(r\) — distance from origin to a point on the terminal arm, \(r = \sqrt{x^2 + y^2}\); always positive.
  • Reference angle \(\beta\) — acute angle between the terminal arm and the \(x\)-axis; \(0° < \beta \le 90°\).
  • CAST rule — quadrant signs: Q I all positive, Q II sin, Q III tan, Q IV cos.
  • Principal angle — angle in standard position between 0° and 360°.

Example 1 — The terminal arm of an angle \(\theta\) in standard position passes through \((-3, 4)\). Find the six trig ratios, the reference angle, and the principal angle.

\(x = -3, y = 4, r = \sqrt{9 + 16} = 5\).

\(\sin\theta = \dfrac{4}{5}, \cos\theta = -\dfrac{3}{5}, \tan\theta = -\dfrac{4}{3}\).

\(\csc\theta = \dfrac{5}{4}, \sec\theta = -\dfrac{5}{3}, \cot\theta = -\dfrac{3}{4}\).

Reference angle: \(\tan\beta = \dfrac{|y|}{|x|} = \dfrac{4}{3}\), so \(\beta \approx 53.13°\).

Principal angle: terminal arm is in Quadrant II (\(x < 0, y > 0\)). Principal \(\theta = 180° - 53.13° = 126.87°\).

CAST check: Q II → only sine is positive. ✓

Example 2 — Find both angles between 0° and 360° for which \(\sin\theta = -\dfrac{1}{2}\).

Sine is negative in Q III and Q IV. Reference angle: \(\sin\beta = \dfrac{1}{2}\) gives \(\beta = 30°\).

Q III: \(\theta = 180° + 30° = 210°\). Q IV: \(\theta = 360° - 30° = 330°\).

Solutions: \(\theta = 210°, 330°\).

Example 3 — Evaluate \(\cos 240°\).

240° is in Q III. Reference angle: \(240° - 180° = 60°\). In Q III, cosine is negative.

\(\cos 240° = -\cos 60° = -\dfrac{1}{2}\).

Example 4 — If \(\cos\theta = -\dfrac{5}{13}\) and \(\theta\) is in Quadrant III, find \(\sin\theta\) and \(\tan\theta\).

In Q III, \(x < 0, y < 0\). \(x = -5, r = 13\), so \(y = -\sqrt{r^2 - x^2} = -12\).

\(\sin\theta = \dfrac{y}{r} = -\dfrac{12}{13}\); \(\tan\theta = \dfrac{y}{x} = \dfrac{-12}{-5} = \dfrac{12}{5}\). (Sine negative, tangent positive — matches CAST for Q III.)

Watch out for:

  • Forgetting to use CAST when reading signs. Calculator gives only one solution; you may need others.
  • Confusing reference angle with principal angle. The reference is always between 0° and 90°; the principal is anywhere 0° to 360°.
  • Treating \(r\) as negative. \(r\) is a distance — always positive. Signs come from \(x\) and \(y\).
  • Calculators returning a Q I or Q IV answer for \(\sin^{-1}\). You may have to add or subtract from 180° (or 360°) to find the answer in the correct quadrant.

7.3 — Special Angles & Exact Values
Memorise the exact ratios for 0°, 30°, 45°, 60°, 90° and their multiples in standard position.
Key exact values: \( \sin 30°=\tfrac{1}{2}, \cos 30°=\tfrac{\sqrt{3}}{2}, \tan 30°=\tfrac{1}{\sqrt{3}} \); \( \sin 45°=\cos 45°=\tfrac{\sqrt{2}}{2}, \tan 45°=1 \); \( \sin 60°=\tfrac{\sqrt{3}}{2}, \cos 60°=\tfrac{1}{2}, \tan 60°=\sqrt{3} \). Derived from 30-60-90 and 45-45-90 reference triangles.
📖 Lesson Notes — Ontario Curriculum

Calculators give decimal approximations for trig ratios. For certain "special" angles, however, the exact values are pleasant fractions or surds. Memorising these — and being able to derive them on the fly from two reference triangles — pays huge dividends in MCR3U, MHF4U, and MCV4U, where exact answers are often required.

The two key triangles:

30°-60°-90° triangle. Start with an equilateral triangle of side 2 and bisect it. You get a right triangle with hypotenuse 2, shorter leg 1 (opposite the 30°), and longer leg \(\sqrt{3}\) (opposite the 60°, by Pythagoras: \(\sqrt{2^2 - 1^2} = \sqrt{3}\)).

45°-45°-90° triangle. Take an isosceles right triangle with legs 1. Hypotenuse = \(\sqrt{1^2 + 1^2} = \sqrt{2}\). Both acute angles are 45°.

From these two triangles, every special-angle ratio drops out:

  • \(\sin 30° = \dfrac{1}{2}, \cos 30° = \dfrac{\sqrt{3}}{2}, \tan 30° = \dfrac{1}{\sqrt{3}} = \dfrac{\sqrt{3}}{3}\).
  • \(\sin 45° = \dfrac{1}{\sqrt{2}} = \dfrac{\sqrt{2}}{2}, \cos 45° = \dfrac{\sqrt{2}}{2}, \tan 45° = 1\).
  • \(\sin 60° = \dfrac{\sqrt{3}}{2}, \cos 60° = \dfrac{1}{2}, \tan 60° = \sqrt{3}\).
  • \(\sin 0° = 0, \cos 0° = 1, \tan 0° = 0\) (degenerate triangle: terminal arm on \(+x\)-axis).
  • \(\sin 90° = 1, \cos 90° = 0, \tan 90°\) is undefined (terminal arm on \(+y\)-axis; \(x = 0\) so division by zero).

For angles in Quadrants II, III, IV, use the reference angle (one of 30°, 45°, 60°, 90°) together with the CAST sign rule to write the exact value.

  • Special angles — 0°, 30°, 45°, 60°, 90° and their reflections in other quadrants.
  • 30-60-90 triangle — sides in ratio \(1 : \sqrt{3} : 2\).
  • 45-45-90 triangle — sides in ratio \(1 : 1 : \sqrt{2}\).
  • Rationalising the denominator — preferred form for fractions like \(\dfrac{1}{\sqrt{3}}\): multiply top and bottom by \(\sqrt{3}\) to get \(\dfrac{\sqrt{3}}{3}\).
  • Undefined — \(\tan 90°\), \(\sec 90°\), etc., correspond to vertical terminal arms where division by zero would occur.

Example 1 — Find the exact values of \(\sin 150°\), \(\cos 225°\), and \(\tan 300°\).

150° in Q II: reference 30°. Sine positive in Q II: \(\sin 150° = \sin 30° = \dfrac{1}{2}\).

225° in Q III: reference 45°. Cosine negative in Q III: \(\cos 225° = -\cos 45° = -\dfrac{\sqrt{2}}{2}\).

300° in Q IV: reference 60°. Tangent negative in Q IV: \(\tan 300° = -\tan 60° = -\sqrt{3}\).

Example 2 — Evaluate \(\sin^2 60° + \cos^2 60°\).

\(\sin 60° = \dfrac{\sqrt{3}}{2}\), so \(\sin^2 60° = \dfrac{3}{4}\). \(\cos 60° = \dfrac{1}{2}\), so \(\cos^2 60° = \dfrac{1}{4}\). Sum: \(\dfrac{3}{4} + \dfrac{1}{4} = 1\). ✓ (Verifies the Pythagorean identity.)

Example 3 — Simplify \(\sin 30° \cos 60° + \cos 30° \sin 60°\) and interpret.

\(= \dfrac{1}{2} \cdot \dfrac{1}{2} + \dfrac{\sqrt{3}}{2} \cdot \dfrac{\sqrt{3}}{2} = \dfrac{1}{4} + \dfrac{3}{4} = 1\).

This equals \(\sin(30° + 60°) = \sin 90° = 1\). The expression is an instance of the sine sum identity \(\sin(A + B) = \sin A \cos B + \cos A \sin B\), studied formally in MHF4U.

Watch out for:

  • Mixing up \(\sin 30°\) and \(\sin 60°\). Memorise the position of the 30° (short side) vs 60° (long side) in the 30-60-90 triangle.
  • Leaving \(\dfrac{1}{\sqrt{3}}\) unsimplified. Most teachers prefer the rationalised form \(\dfrac{\sqrt{3}}{3}\).
  • Forgetting to apply the CAST sign rule for angles outside Q I.
  • Writing \(\sin 90° = 0\) or \(\cos 0° = 0\) — the special boundary cases are easy to confuse. \(\sin 90° = 1, \cos 90° = 0, \sin 0° = 0, \cos 0° = 1\).

7.4 — The Sine Law & the Ambiguous Case
Apply \( \dfrac{a}{\sin A}=\dfrac{b}{\sin B}=\dfrac{c}{\sin C} \) to non-right triangles; recognise SSA situations producing 0, 1, or 2 triangles.
In \( \triangle ABC \): \( a=10, b=14, A=35° \). By sine law: \( \sin B=\dfrac{14\sin 35°}{10}\approx 0.803 \), so \( B\approx 53.4° \) or \( B\approx 126.6° \) — two triangles satisfy the data. Ambiguous case (SSA): always check whether the supplementary angle also gives a valid triangle (i.e. \( A+B<180° \)).
📖 Lesson Notes — Ontario Curriculum

Right-triangle trig only handles right triangles. The sine law works for any triangle. In a triangle \(ABC\) with side \(a\) opposite angle \(A\), side \(b\) opposite \(B\), side \(c\) opposite \(C\):

\[ \frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} \]

All three ratios are equal. (Their common value, geometrically, is the diameter of the triangle's circumscribed circle — a fact you'll see in MHF4U.) The sine law is used in two main cases:

AAS / ASA: two angles and a side. Find the third angle (angles sum to 180°), then use the law to find the remaining sides.

SSA: two sides and a non-included angle. This is the ambiguous case — depending on the data you can get 0, 1, or 2 triangles. The sine law gives \(\sin B = \dfrac{b \sin A}{a}\), and:

  • If \(\sin B > 1\), no triangle exists.
  • If \(\sin B = 1\), exactly one (right) triangle.
  • If \(\sin B < 1\), there are two candidate angles for \(B\) (the inverse sine and its supplement, \(180° - B_1\)). Both must be checked: a candidate is valid only if the angles still sum to under 180° (i.e. \(A + B < 180°\)).

When in SSA territory, never trust your calculator's first answer. Always test the supplementary angle. If only one of the two candidates gives a valid triangle, you have one solution; if both do, you have two distinct triangles.

  • Sine law — \(\dfrac{a}{\sin A} = \dfrac{b}{\sin B} = \dfrac{c}{\sin C}\), valid in any triangle.
  • AAS / ASA — angle-angle-side or angle-side-angle; sine law gives a unique solution.
  • SSA — side-side-angle (also called the ambiguous case); may give 0, 1, or 2 triangles.
  • Supplementary angles — pair \(B\) and \(180° - B\) have the same sine.
  • Triangle inequality — the sum of any two angles must be less than 180° for a valid triangle.

Example 1 (ASA) — In \(\triangle ABC\): \(A = 40°, B = 75°, c = 12\). Solve the triangle.

Third angle: \(C = 180° - 40° - 75° = 65°\).

By sine law: \(\dfrac{a}{\sin 40°} = \dfrac{12}{\sin 65°}\), so \(a = \dfrac{12 \sin 40°}{\sin 65°} \approx \dfrac{12 \cdot 0.6428}{0.9063} \approx 8.51\).

\(b = \dfrac{12 \sin 75°}{\sin 65°} \approx \dfrac{12 \cdot 0.9659}{0.9063} \approx 12.79\).

Example 2 (SSA, ambiguous case) — In \(\triangle ABC\): \(a = 10, b = 14, A = 35°\). Solve.

By sine law: \(\sin B = \dfrac{b \sin A}{a} = \dfrac{14 \sin 35°}{10} = \dfrac{14 \cdot 0.5736}{10} \approx 0.8030\).

Two candidates: \(B_1 = \sin^{-1}(0.8030) \approx 53.4°\) and \(B_2 = 180° - 53.4° = 126.6°\).

Check angle sum: \(A + B_1 = 35° + 53.4° = 88.4° < 180°\) ✓; \(A + B_2 = 35° + 126.6° = 161.6° < 180°\) ✓. Both triangles exist.

Triangle 1: \(B = 53.4°, C = 91.6°\). \(c = \dfrac{10 \sin 91.6°}{\sin 35°} \approx \dfrac{10 \cdot 0.9996}{0.5736} \approx 17.43\).

Triangle 2: \(B = 126.6°, C = 18.4°\). \(c = \dfrac{10 \sin 18.4°}{\sin 35°} \approx \dfrac{10 \cdot 0.3156}{0.5736} \approx 5.50\).

Example 3 (SSA, no solution) — In \(\triangle ABC\): \(a = 6, b = 12, A = 60°\). Solve.

\(\sin B = \dfrac{12 \sin 60°}{6} = 2 \sin 60° = \sqrt{3} \approx 1.732\).

Since \(\sin B > 1\), no triangle exists with these measurements.

Geometrically: \(b = 12\) is too long relative to the available "drop" from \(A\) — the second side can't reach.

Watch out for:

  • Setting up the sine law with the wrong pairing. Each side must be paired with the angle opposite to it.
  • Skipping the supplement check in SSA problems. Always test \(180° - B_1\).
  • Forgetting to verify \(A + B < 180°\) for each candidate. A "valid" supplement fails if it leaves no room for the third angle.
  • Using the sine law when the cosine law is the right tool. Sine law needs at least one opposite angle-side pair to be known.

7.5 — The Cosine Law & 2D/3D Applications
Use \( c^2 = a^2 + b^2 - 2ab\cos C \) for SAS and SSS triangles; apply both laws in surveying, navigation, and 3D problems.
In \( \triangle ABC \): \( a=8, b=11, C=42° \). Then \( c^2=64+121-2(8)(11)\cos 42°=185-130.79\approx 54.21 \), so \( c\approx 7.36 \). Cosine law is needed when sine law has insufficient information (no opposite-pair).
📖 Lesson Notes — Ontario Curriculum

The cosine law is the other major tool for solving non-right triangles. For any triangle \(ABC\) with sides \(a, b, c\) opposite the angles \(A, B, C\):

\[ c^2 = a^2 + b^2 - 2ab\cos C \]

(And similarly with the roles cycled: \(a^2 = b^2 + c^2 - 2bc \cos A\), \(b^2 = a^2 + c^2 - 2ac \cos B\).)

If \(C = 90°\), then \(\cos C = 0\) and the formula reduces to Pythagoras: \(c^2 = a^2 + b^2\). So the cosine law is a generalisation of Pythagoras that accounts for non-right angles via the correction term \(-2ab\cos C\). If \(C\) is acute (\(\cos C > 0\)), the correction subtracts; \(c\) is shorter than the Pythagorean prediction. If \(C\) is obtuse (\(\cos C < 0\)), the correction adds; \(c\) is longer.

Use the cosine law in two situations the sine law can't handle:

SAS — given two sides and the included angle, find the third side. Apply the formula directly with \(C\) as the included angle.

SSS — given all three sides, find an angle. Rearrange: \(\cos C = \dfrac{a^2 + b^2 - c^2}{2ab}\), then take the inverse cosine.

Once you have one new side or angle from the cosine law, you can usually switch to the sine law for the remaining unknowns (the sine law arithmetic is simpler).

  • Cosine law — \(c^2 = a^2 + b^2 - 2ab\cos C\), valid in any triangle.
  • SAS — side-angle-side; cosine law gives the third side directly.
  • SSS — side-side-side; rearranged cosine law gives an angle.
  • Included angle — angle between two given sides.
  • Generalisation of Pythagoras — Pythagorean theorem is the cosine law at \(C = 90°\).

Example 1 (SAS) — In \(\triangle ABC\): \(a = 8, b = 11, C = 42°\). Find \(c\).

\(c^2 = 8^2 + 11^2 - 2(8)(11)\cos 42° = 64 + 121 - 176 \cdot 0.7431 \approx 185 - 130.79 \approx 54.21\).

\(c \approx \sqrt{54.21} \approx 7.36\).

Example 2 (SSS) — In \(\triangle ABC\): \(a = 6, b = 8, c = 10\). Find angle \(C\).

\(\cos C = \dfrac{a^2 + b^2 - c^2}{2ab} = \dfrac{36 + 64 - 100}{2 \cdot 6 \cdot 8} = \dfrac{0}{96} = 0\).

\(C = \cos^{-1}(0) = 90°\). The triangle is right-angled at \(C\) — consistent with the 6-8-10 Pythagorean triple.

Example 3 (Surveying, 2D) — Two ships leave the same port. One sails 20 km at a bearing of 040°; the other sails 28 km at 110°. How far apart are they?

The angle between the two paths (as seen from the port) is \(110° - 40° = 70°\). The setup is SAS with \(a = 20\), \(b = 28\), included angle \(70°\).

\(c^2 = 400 + 784 - 2 \cdot 20 \cdot 28 \cdot \cos 70° = 1184 - 1120 \cdot 0.3420 \approx 1184 - 383.1 \approx 800.9\).

\(c \approx \sqrt{800.9} \approx 28.3\) km.

Example 4 (3D) — A flagpole of height 12 m stands on flat ground. From a point \(P\), the angle of elevation to the top is 35°. From another point \(Q\), 25 m from \(P\) along the ground, the angle of elevation is 48°. The angle \(\angle PFQ\) at the base of the pole is 110°. Find the distance \(PQ\) through the air to verify the data.

From elevations: distance from base of pole to \(P\) is \(\dfrac{12}{\tan 35°} \approx 17.14\) m; to \(Q\) is \(\dfrac{12}{\tan 48°} \approx 10.81\) m.

Cosine law in \(\triangle PFQ\): \(PQ^2 = 17.14^2 + 10.81^2 - 2 \cdot 17.14 \cdot 10.81 \cos 110° = 293.8 + 116.9 - 370.6 \cdot (-0.342) \approx 410.7 + 126.7 = 537.4\). \(PQ \approx 23.2\) m. The given 25 m is close — consistent with measurement error.

Watch out for:

  • Confusing which angle is "included." The included angle is the one between the two given sides.
  • Sign errors. \(\cos\) of an obtuse angle is negative; the \(-2ab\cos C\) term then becomes positive, making the third side longer.
  • Computing \(\sqrt{}\) of a negative result. If \(c^2 < 0\) you've made an arithmetic mistake — recompute.
  • Using the sine law when the cosine law is needed. Sine law needs at least one matched side-and-opposite-angle.

8.1 — Graphs of \( y=\sin x \) and \( y=\cos x \)
Sketch the parent sinusoidal functions in degree mode; identify period (360°), amplitude (1), domain (\(\mathbb{R}\)), range (\([-1,1]\)).
The sine function passes through (0,0), peaks at (90°,1), returns to 0 at (180°,0), reaches a minimum at (270°,-1), then back to 0 at (360°,0). Cosine is sine shifted left by 90°: \( \cos x = \sin(x+90°) \). Both repeat every 360° — they are periodic.
📖 Lesson Notes — Ontario Curriculum

The trig ratios you've been using since Chapter 7 are also functions: feed in an angle, get back a number. When you plot \(y = \sin x\) or \(y = \cos x\) across a range of \(x\) (in degrees), you get a wave shape that repeats forever. These are the sinusoidal functions — the building blocks of describing anything that cycles: vibrations, sound, light, tides, biological rhythms, alternating current.

Trace \(y = \sin x\) by stepping through the unit-circle perspective. As \(x\) rotates from 0° to 360°, the \(y\)-coordinate of the unit-circle point traces:

  • \(x = 0°\): \(y = 0\). (Start at the positive \(x\)-axis.)
  • \(x = 90°\): \(y = 1\). (Top of the circle — maximum.)
  • \(x = 180°\): \(y = 0\). (Negative \(x\)-axis.)
  • \(x = 270°\): \(y = -1\). (Bottom of the circle — minimum.)
  • \(x = 360°\): \(y = 0\). (Back to start.)

The graph of \(y = \cos x\) is identical in shape but shifted: at \(x = 0°\), cosine starts at 1 (the \(x\)-coordinate of the right side of the unit circle), peaks again at \(x = 360°\), and dips through 0 at \(x = 90°\) and \(x = 270°\). In fact \(\cos x = \sin(x + 90°)\) — cosine is just sine shifted left by 90°.

Key features of both parent functions:

  • Period: 360° (the smallest \(x\)-interval after which the graph repeats).
  • Amplitude: 1 (half the vertical distance between max and min).
  • Domain: all real numbers — every angle has a sine and a cosine.
  • Range: \([-1, 1]\).
  • Midline: \(y = 0\) (the horizontal line halfway between max and min).
  • \(y\)-intercept: \(\sin\) has \((0, 0)\); \(\cos\) has \((0, 1)\).
  • Symmetry: \(\sin\) is odd (rotational about origin); \(\cos\) is even (symmetric in \(y\)-axis).

Example 1 — Sketch \(y = \sin x\) for \(-360° \le x \le 720°\).

Plot key points at multiples of 90°: \(x = -360°, -270°, -180°, -90°, 0°, 90°, 180°, \ldots, 720°\). At each, \(y\) cycles through \(0, 1, 0, -1, 0\). Connect smoothly. The graph completes three full cycles across the given range (\(1080°\) wide divided by 360° period).

Example 2 — From a sketch of \(y = \cos x\), find all solutions to \(\cos x = -0.5\) in \([0°, 360°]\).

Sketch the graph; identify where it crosses \(y = -0.5\). Cosine equals \(-0.5\) at \(x = 120°\) and \(x = 240°\). These are the two values where the horizontal line \(y = -0.5\) intersects the cosine curve in one full period.

Example 3 — Verify graphically that \(\sin x = \cos(x - 90°)\).

The graph of \(\cos x\) shifted right by 90° lands on the same wave as \(\sin x\). At \(x = 0\): \(\sin 0 = 0\); \(\cos(-90°) = \cos 90° = 0\). ✓ At \(x = 90°\): \(\sin 90° = 1\); \(\cos 0 = 1\). ✓

Watch out for:

  • Switching between degree and radian mode without realising. In MCR3U everything is degrees.
  • Drawing pointy "V" shapes at the peaks. Sinusoidal curves are smooth — the maximum is rounded.
  • Mislabeling \(y\)-coordinates as angles or vice versa. The \(x\)-axis is the angle (in degrees); the \(y\)-axis is the sine/cosine value (between \(-1\) and \(1\)).
  • Memorising "sin starts at 0" without understanding the unit-circle origin. The unit circle is the source of every key value.

8.2 — Amplitude, Period, & Vertical Shift
Analyse \( y = a\sin(kx) + c \): amplitude \( |a| \), period \( \dfrac{360°}{|k|} \), vertical shift \( c \), midline \( y=c \).
For \( y=3\sin(2x)+1 \): amplitude 3, period \( 360°/2=180° \), midline \( y=1 \), maximum \( y=4 \), minimum \( y=-2 \). Stretch in the y-direction (amplitude) and compression in the x-direction (period) reshape the parent sine curve.
📖 Lesson Notes — Ontario Curriculum

Three parameters reshape the parent sine or cosine wave: amplitude \(a\), the horizontal-stretch coefficient \(k\) that controls period, and vertical shift \(c\). The form is:

\[ y = a\sin(kx) + c \quad \text{or} \quad y = a\cos(kx) + c \]

Amplitude \(|a|\) — half the vertical distance between the maximum and minimum. The graph stretches vertically by factor \(|a|\); if \(a < 0\), the wave is also reflected vertically. For \(y = 3\sin x\), the wave swings between \(-3\) and \(+3\). For \(y = -\sin x\), it's flipped: it starts going down from 0 instead of up.

Period \(= \dfrac{360°}{|k|}\) — the smallest horizontal distance over which the graph repeats. \(k\) horizontally compresses the wave when \(|k| > 1\) (so it cycles faster, period shrinks) and stretches it when \(0 < |k| < 1\) (period lengthens). For \(y = \sin(2x)\), the period is 180° — twice as fast as parent. For \(y = \sin(x/2)\), the period is 720° — twice as slow.

Vertical shift \(c\) — moves the entire wave up (positive \(c\)) or down. The midline moves from \(y = 0\) to \(y = c\). The max becomes \(c + |a|\), the min becomes \(c - |a|\).

Summary for \(y = a\sin(kx) + c\): amplitude \(|a|\), period \(\dfrac{360°}{|k|}\), midline \(y = c\), maximum \(c + |a|\), minimum \(c - |a|\).

  • Amplitude — \(|a|\); half the distance from max to min; vertical "swing."
  • Period — \(\dfrac{360°}{|k|}\); horizontal length of one complete cycle.
  • Midline — horizontal line \(y = c\); the centre of the oscillation.
  • Maximum — \(c + |a|\); top of the wave.
  • Minimum — \(c - |a|\); bottom of the wave.
  • Frequency — number of cycles per unit; reciprocal of period. (\(k\) controls frequency proportionally.)

Example 1 — Analyse \(y = 3\sin(2x) + 1\). State amplitude, period, midline, max, min, and sketch one period.

\(a = 3, k = 2, c = 1\). Amplitude 3. Period \(= 360°/2 = 180°\). Midline \(y = 1\). Maximum \(1 + 3 = 4\). Minimum \(1 - 3 = -2\).

Key points (over one period from 0° to 180°): at \(x = 0°\), \(y = 3\sin 0 + 1 = 1\). At \(x = 45°\) (quarter-period), \(y = 3\sin 90° + 1 = 4\) (max). At 90° (half), \(y = 1\) (midline). At 135° (three-quarters), \(y = 3\sin 270° + 1 = -2\) (min). At 180°, \(y = 1\). Connect smoothly.

Example 2 — Find the equation of a cosine function with amplitude 4, period 120°, and midline \(y = 2\).

From period: \(120° = 360°/k\), so \(k = 3\). Amplitude is the absolute value of \(a\); take \(a = 4\) (positive, so cosine starts at the maximum).

\(y = 4\cos(3x) + 2\). Maximum: \(6\); minimum: \(-2\).

Example 3 — A Ferris wheel has radius 10 m, and its centre is 12 m above the ground. The wheel makes one full revolution every 60 seconds. Write a sinusoidal model for a rider's height above the ground, starting from the bottom of the wheel at \(t = 0\).

Amplitude 10 (radius). Midline \(y = 12\) (centre height). Period 60 s, so \(k = 360°/60 = 6\) (in degrees per second).

Starting at the bottom: \(y(0) = 12 - 10 = 2\) — minimum. The cosine model with a negative sign starts at its minimum: \(y(0) = -10\cos(0) + 12 = -10 + 12 = 2\). ✓.

Model: \(h(t) = -10\cos(6t) + 12\) with \(t\) in seconds and \(6t\) interpreted in degrees.

Watch out for:

  • Confusing \(k\) with the period. Period \(= 360°/k\), not just \(k\). \(k = 2\) gives a period of 180° (faster), not a period of 2°.
  • Taking the amplitude as the maximum value. Amplitude is half the swing, not the peak height.
  • Forgetting to add (or subtract) \(c\) when reading off max/min. Max = midline + amplitude; min = midline − amplitude.
  • Treating a negative amplitude as the actual amplitude. \(|a|\) is the amplitude; the sign of \(a\) controls the wave's starting direction.

8.3 — Phase Shift & the General Sinusoidal Form
Analyse the full form \( y = a\sin(k(x-d)) + c \): the parameter \( d \) is a horizontal (phase) shift; combine with all other transformations.
For \( y=2\sin(3(x-30°))-1 \): amplitude 2, period 120°, phase shift 30° right, midline \( y=-1 \). Always factor out \( k \) inside the argument before reading off the phase shift: \( \sin(3x-90°)=\sin(3(x-30°)) \).
📖 Lesson Notes — Ontario Curriculum

The fourth and final parameter is the phase shift (also called horizontal shift). The full general form for a sinusoidal function is:

\[ y = a\sin\bigl(k(x - d)\bigr) + c \quad \text{or} \quad y = a\cos\bigl(k(x - d)\bigr) + c \]

The parameter \(d\) shifts the wave horizontally: positive \(d\) means shift right; negative \(d\) means shift left. The phase shift is exactly \(d\), provided the equation is written in the factored form \(k(x - d)\). That's the catch — equations often arrive with the inside expanded as \(kx - p\), and you must factor \(k\) before reading off \(d\).

For example, \(y = \sin(3x - 90°)\) looks at first glance like a 90° shift, but factor first: \(3x - 90° = 3(x - 30°)\). So \(d = 30°\), not 90°. The phase shift is 30° right — the wave starts at the value where the parent \(\sin\) is at "horizontal position 0," but 30° later.

With all four parameters present:

  • Amplitude \(|a|\).
  • Period \(\dfrac{360°}{|k|}\).
  • Phase shift \(d\) (right if positive).
  • Midline \(y = c\); max \(c + |a|\); min \(c - |a|\).

Sketching is fastest with a five-point method. Start at the phase shift (\(x = d\)) on the midline going up (for \(+\sin\)). The next quarter-period takes you to the max; the next to the midline going down; the next to the min; the next back to the midline. Each step horizontally is one-quarter of the period.

  • Phase shift — horizontal translation \(d\) of the wave; the value of \(d\) in \(k(x - d)\).
  • Factoring step — before reading \(d\), rewrite \(kx + p\) as \(k(x + p/k)\) so that \(d = -p/k\) appears cleanly.
  • Five-point method — for sine starting from \(x = d\): midline up, max, midline down, min, midline up — equally spaced over one period.
  • Equivalent equations — a sine and cosine can describe the same wave with different phase shifts: \(\sin(x) = \cos(x - 90°)\).

Example 1 — Analyse \(y = 2\sin(3(x - 30°)) - 1\). State amplitude, period, phase shift, midline, max, min, and sketch one period starting at \(x = 30°\).

\(a = 2, k = 3, d = 30°, c = -1\). Amplitude 2. Period \(360°/3 = 120°\). Phase shift 30° right. Midline \(y = -1\). Max \(1\), min \(-3\).

Five-point method: quarter-period = 30°. Start at \(x = 30°\), midline (\(-1\)) going up; at \(x = 60°\), max \(1\); at \(x = 90°\), midline (\(-1\)) going down; at \(x = 120°\), min \(-3\); at \(x = 150°\), midline back. One period spans \([30°, 150°]\).

Example 2 — Rewrite \(y = -4\cos(2x + 60°) + 5\) in the form \(y = a\cos(k(x - d)) + c\) and identify all parameters.

Factor inside: \(2x + 60° = 2(x + 30°) = 2(x - (-30°))\). So \(y = -4\cos(2(x - (-30°))) + 5\).

Parameters: \(a = -4\) (amplitude 4, reflection in \(x\)-axis), \(k = 2\) (period 180°), \(d = -30°\) (phase shift 30° to the left), \(c = 5\) (midline \(y = 5\)). Max: \(5 + 4 = 9\); min: \(5 - 4 = 1\). At \(x = -30°\), the reflected cosine is at its minimum (1) instead of its maximum.

Example 3 — Write a sine equation for a wave with amplitude 6, period 90°, midline \(y = 2\), and starting at the midline going up at \(x = 15°\).

\(k = 360°/90° = 4\). \(d = 15°\) (the wave behaves like the parent sine at \(x = 0\), but at \(x = 15°\)). Midline \(c = 2\).

Equation: \(y = 6\sin(4(x - 15°)) + 2\).

Watch out for:

  • Reading the phase shift directly from \(kx - p\) without factoring. Always factor \(k\) first.
  • Confusing the sign of \(d\). \((x - 30°)\) means shift right by 30°, not left. The shift direction matches what's subtracted.
  • Treating sin and cos as interchangeable without accounting for the 90° phase difference.
  • Forgetting that quarter-period spacing is the cleanest way to plot key points.

8.4 — Modeling Periodic Phenomena
Build sinusoidal models from data: tides, daylight hours, sound waves, Ferris-wheel height, pendulum motion.
Tide example: water depth varies between 1.5 m (low) and 7.5 m (high), with high tide at \( t=3 \) h and a 12-hour period. Midline \( c=4.5 \), amplitude \( a=3 \), period 12 → \( k=360°/12=30°/\text{h} \). Cosine fits cleanly with peak at \( t=3 \): \( h(t)=3\cos(30°(t-3))+4.5 \).
📖 Lesson Notes — Ontario Curriculum

Many real-world quantities oscillate between two extremes over time: ocean tides between high and low, daylight hours between summer and winter solstices, body temperature through the day, sound waves between compression and rarefaction. These are sinusoidal — modelled by \(y = a\sin(k(x - d)) + c\) or its cosine equivalent.

The modelling recipe takes data (or a verbal description) and extracts the four parameters:

Step 1 — Find midline \(c\). Average the maximum and minimum: \(c = \dfrac{\max + \min}{2}\).

Step 2 — Find amplitude \(|a|\). Half the difference: \(|a| = \dfrac{\max - \min}{2}\).

Step 3 — Find \(k\) from the period. The period is one full cycle in the appropriate time units (hours, days, years). Then \(k = \dfrac{360°}{\text{period}}\).

Step 4 — Pick sine or cosine and find \(d\). Cosine is easiest when you know when the maximum occurs: set \(d\) equal to that time, take \(a > 0\), and the cosine wave peaks at \(x = d\). Sine is easiest when you know when the function crosses the midline going up: set \(d\) to that time.

The same recipe works for any periodic phenomenon. The hardest part is usually identifying the four key values (max, min, period, time of a reference event) from a word problem.

  • Tides — typically two highs and two lows per day; semi-diurnal period ≈ 12.4 hours.
  • Daylight hours — annual cycle: period 365 days, with maximum near summer solstice (June 21).
  • Body temperature — circadian rhythm: period 24 hours, amplitude about 0.5°C around midline 37°C.
  • Ferris wheel — period equals one revolution; amplitude equals wheel radius; midline equals centre height.
  • AC voltage — period \(1/60\) s in North America; modelled as \(V(t) = V_{\max}\sin(120\pi t)\) (radians) or 21{,}600° \(t\) (degrees).

Example 1 — In a harbour, the water depth varies sinusoidally between 1.5 m (low tide) and 7.5 m (high tide). High tide occurs at \(t = 3\) hours, and the period is 12 hours. Find the depth as a function of time.

Step 1 — Midline: \(c = (7.5 + 1.5)/2 = 4.5\) m.

Step 2 — Amplitude: \(|a| = (7.5 - 1.5)/2 = 3\) m. Take \(a = 3\) (cosine peaks).

Step 3 — \(k = 360°/12 = 30°\) per hour.

Step 4 — Cosine peaks at \(x = d\), and we want a peak at \(t = 3\), so \(d = 3\).

Model: \(h(t) = 3\cos(30°(t - 3)) + 4.5\). Check: at \(t = 3\), \(h = 3 \cdot 1 + 4.5 = 7.5\) ✓. At \(t = 9\) (half-period later), \(h = 3 \cdot (-1) + 4.5 = 1.5\) ✓.

Example 2 — A Ferris wheel has radius 8 m, with its centre 10 m above the ground. It completes one revolution every 40 seconds. A passenger boards at the lowest point at \(t = 0\). Write a model for the rider's height above the ground.

Midline \(c = 10\) m. Amplitude 8 m. \(k = 360°/40 = 9°\)/s.

The rider starts at the minimum (\(t = 0\), height 2 m). A cosine with negative amplitude has its minimum at \(t = d = 0\): \(h(t) = -8\cos(9°t) + 10\).

Check: \(t = 0\): \(h = -8 + 10 = 2\) ✓ (boarding). \(t = 10\) (quarter-period): \(h = -8\cos 90° + 10 = 10\) (midline, rising). \(t = 20\) (half): \(h = -8 \cdot (-1) + 10 = 18\) (top of wheel) ✓.

Example 3 — In a Canadian city, the number of daylight hours \(D(n)\) on day \(n\) of the year varies sinusoidally between about 8 hours (winter solstice, ~day 355) and 16 hours (summer solstice, ~day 172). Find a model.

Midline: \(c = (16 + 8)/2 = 12\). Amplitude: \(|a| = (16 - 8)/2 = 4\). Period: 365 days, so \(k = 360°/365 \approx 0.9863°\) per day.

Cosine peaks at \(n = 172\): \(D(n) = 4\cos(0.9863°(n - 172)) + 12\).

Watch out for:

  • Computing amplitude as max − min (instead of half of that).
  • Confusing midline with the \(y\)-intercept. The midline is a horizontal line; the \(y\)-intercept is one specific point on the curve.
  • Forgetting that the period must match the units of \(t\) — hours, days, seconds, etc.
  • Choosing sine when cosine fits more cleanly. If the problem tells you when a maximum (or minimum) occurs, cosine usually wins.

8.5 — Solving Sinusoidal Equations Graphically
Use a graph or table to solve equations of the form \( a\sin(k(x-d))+c = K \) on a given interval; interpret in context.
From the tide model \( h(t)=3\cos(30°(t-3))+4.5 \), find when \( h=6 \) m. Solve \( 3\cos(30°(t-3))=1.5 \), so \( \cos(30°(t-3))=0.5 \), giving \( 30°(t-3)=\pm 60° \), so \( t=1 \) h or \( t=5 \) h within one period. Algebraic solution methods are studied formally in MHF4U.
📖 Lesson Notes — Ontario Curriculum

Once you have a sinusoidal model \(y = a\sin(k(x - d)) + c\) (or its cosine cousin), the next question is invariably: when does \(y\) equal some target value? When is the tide depth at least 6 m? When is daylight more than 14 hours? When is the rider 15 m above the ground? These are inequalities and equations involving sinusoidal expressions.

In MCR3U we solve these graphically: graph the function and a horizontal line \(y = K\), and read off the intersection points. Use graphing technology, or sketch the wave by hand using the five-point method and estimate. You can also use the inverse-trig functions to get one solution, then use the wave's structure (symmetry within a period, periodicity to extend across multiple periods) to find all solutions in the desired interval.

A typical workflow for \(a\sin(k(x - d)) + c = K\):

  1. Isolate the trig expression: \(\sin(k(x - d)) = \dfrac{K - c}{a}\). If the right side is outside \([-1, 1]\), no solutions exist.
  2. Let \(\theta = k(x - d)\). Solve \(\sin\theta = (K - c)/a\) in one period; use the reference angle and CAST.
  3. Convert back: \(x = \dfrac{\theta}{k} + d\) for each solution \(\theta\).
  4. Use periodicity (the function repeats every \(360°/k\)) to find all solutions in the required interval.

Inequalities are read off the graph: the regions where the sine curve is above (or below) the horizontal line.

  • Graphical solution — solve \(f(x) = K\) by finding the \(x\)-coordinates where \(y = f(x)\) meets the line \(y = K\).
  • Periodicity — if \(x_0\) is a solution, so are \(x_0 + n \cdot (\text{period})\) for every integer \(n\).
  • Symmetry within a period — sine and cosine values typically appear twice per period; use the symmetry of the wave to find the second \(x\).
  • Restricted interval — most problems ask for solutions on a specific interval (e.g. one day, one period). Identify which solutions fall inside.

Example 1 — Using the tide model \(h(t) = 3\cos(30°(t - 3)) + 4.5\), find all times in the first 12 hours when the depth is exactly 6 m.

Set up: \(3\cos(30°(t - 3)) + 4.5 = 6\), so \(\cos(30°(t - 3)) = 0.5\).

Let \(\theta = 30°(t - 3)\). Solve \(\cos\theta = 0.5\) in one full revolution: \(\theta = 60°\) or \(\theta = 300°\) (equivalently \(-60°\)).

Convert back: \(30°(t - 3) = 60° \Rightarrow t - 3 = 2 \Rightarrow t = 5\) h. \(30°(t - 3) = -60° \Rightarrow t = 1\) h.

The first 12 hours contain one full period. The next two solutions within the period: \(\theta = 60° + 360° = 420°\) gives \(t - 3 = 14\), \(t = 17\) h (outside our interval); \(\theta = -60° + 360° = 300°\) is the same as \(t = 5\). So within \([0, 12]\) the depth is 6 m at \(t = 1\) h and \(t = 5\) h. Between these times the depth is above 6 m (tide is in).

Example 2 — For the Ferris wheel model \(h(t) = -8\cos(9°t) + 10\), when is the rider exactly 14 m above the ground during the first revolution?

Set: \(-8\cos(9°t) + 10 = 14\), so \(\cos(9°t) = -0.5\).

\(\cos\theta = -0.5\) at \(\theta = 120°\) and \(\theta = 240°\) in \([0°, 360°]\).

\(9°t = 120° \Rightarrow t = 13.33\) s. \(9°t = 240° \Rightarrow t = 26.67\) s.

During the first 40-second revolution, the rider is at 14 m at \(t \approx 13.33\) s (rising) and \(t \approx 26.67\) s (descending).

Example 3 — Graphical interpretation: when is the daylight model \(D(n) = 4\cos(0.9863°(n - 172)) + 12\) above 14 hours?

Set up: \(4\cos(0.9863°(n - 172)) + 12 > 14\), so \(\cos(0.9863°(n - 172)) > 0.5\).

Cosine \(> 0.5\) where the angle is in \((-60°, 60°)\). So \(-60° < 0.9863°(n - 172) < 60°\), giving \(-60.84 < n - 172 < 60.84\), so roughly \(111 < n < 233\).

That's approximately April 21 (day 111) to August 21 (day 233) — about four months of "long days."

Watch out for:

  • Forgetting the second solution per period. Sine and cosine values typically appear twice per cycle; the calculator only gives one.
  • Treating "the graph crosses" as one event when it might cross multiple times within the interval. Sketch first to count.
  • Mixing up \(\theta\) (the angle inside the sine/cosine) with \(x\) (the time/horizontal variable). After solving for \(\theta\), always convert back to \(x\) using \(x = \theta/k + d\).
  • Solutions outside the asked-for interval. Restrict by inspecting whether each solution falls within the problem's bounds.

B.1 — Average vs. Instantaneous Rate of Change (MHF4U Preview)
For any function studied in MCR3U (quadratic, exponential, sinusoidal), the average rate of change over \([a,b]\) is \( \frac{f(b)-f(a)}{b-a} \) — the slope of the secant line. As \( b\to a \), the secant approaches the tangent, giving the instantaneous rate of change. This limit concept is the gateway to calculus and is developed formally in MHF4U and MCV4U.
For \( f(x)=x^2 \), the average rate of change on \([2,2+h]\) is \( \frac{(2+h)^2 - 4}{h} = 4+h \). As \( h\to 0 \), the instantaneous rate at \( x=2 \) is \( 4 \). The same secant-to-tangent reasoning applies to exponential growth (population doubling rate) and sinusoidal motion (instantaneous velocity of a Ferris wheel rider). Mastering this difference quotient now makes MHF4U's introduction to derivatives feel natural.
📖 Lesson Notes — Ontario Curriculum

This topic is the bridge from Grade 11 to calculus. Every function studied in MCR3U has a rate of change: how fast does \(y\) change as \(x\) changes? For a linear function, the rate of change is constant — it's just the slope. For non-linear functions, the rate of change varies from point to point, and we need two ideas to handle it precisely.

Average rate of change on an interval \([a, b]\) is the slope of the line connecting the endpoints of the graph — the secant line:

\[ \text{ARC} = \frac{f(b) - f(a)}{b - a} \]

This is what you've used since Grade 9 for "average speed" between two times. It tells you the typical rate over the interval, but it doesn't tell you what's happening at any single instant inside.

To get the rate at a single point — the instantaneous rate of change at \(x = a\) — let the interval shrink. Replace \(b\) with \(a + h\), and let \(h\) approach 0. The secant line slope becomes the slope of the line just barely touching the curve at \(x = a\): the tangent line. Symbolically:

\[ \text{IRC at } x = a \;=\; \lim_{h \to 0} \frac{f(a + h) - f(a)}{h} \]

That fraction \(\dfrac{f(a + h) - f(a)}{h}\) is the difference quotient. As \(h \to 0\), it approaches the instantaneous rate of change — which, in calculus, is called the derivative \(f'(a)\). MHF4U formalises limits; MCV4U defines and computes derivatives systematically. For MCR3U, you should be able to compute an average rate of change for any function and estimate an instantaneous rate by trying smaller and smaller \(h\).

  • Secant line — straight line through two points on a curve.
  • Tangent line — line touching the curve at a single point, in the limiting direction of secants.
  • Average rate of change — \(\dfrac{f(b) - f(a)}{b - a}\); slope of the secant.
  • Instantaneous rate of change — \(\displaystyle\lim_{h \to 0} \dfrac{f(a + h) - f(a)}{h}\); slope of the tangent at \(x = a\).
  • Difference quotient — \(\dfrac{f(a + h) - f(a)}{h}\); the central expression of differential calculus.

Example 1 — For \(f(x) = x^2\), compute the average rate of change on \([2, 2 + h]\), then estimate the instantaneous rate at \(x = 2\) by letting \(h\) shrink.

\(f(2 + h) = (2 + h)^2 = 4 + 4h + h^2\); \(f(2) = 4\).

ARC: \(\dfrac{(4 + 4h + h^2) - 4}{h} = \dfrac{4h + h^2}{h} = 4 + h\) (for \(h \ne 0\)).

At \(h = 0.1\): ARC = 4.1. At \(h = 0.01\): 4.01. At \(h = 0.001\): 4.001. The pattern is clear: as \(h \to 0\), ARC \(\to 4\). The instantaneous rate of change of \(x^2\) at \(x = 2\) is 4. (In MCV4U this comes out instantly: \(f'(x) = 2x\), so \(f'(2) = 4\).)

Example 2 — A population grows according to \(P(t) = 1000(1.05)^t\) (years). Find the average growth rate over \([0, 10]\) and estimate the instantaneous growth rate at \(t = 10\).

\(P(0) = 1000\); \(P(10) = 1000(1.05)^{10} \approx 1628.89\). ARC: \((1628.89 - 1000)/10 \approx 62.89\) per year — the average annual growth in raw population.

For instantaneous at \(t = 10\): try \(h = 0.01\). \(P(10.01) \approx 1000(1.05)^{10.01} \approx 1629.68\). ARC on \([10, 10.01]\): \((1629.68 - 1628.89)/0.01 \approx 79.4\) per year. The population is growing about 79 people per year at \(t = 10\), more than the average rate over the whole decade — consistent with exponential acceleration.

Example 3 — A Ferris wheel rider's height is \(h(t) = -10\cos(6°t) + 12\) (metres, seconds). Find the average vertical speed over \([5, 10]\) seconds.

\(h(5) = -10\cos 30° + 12 \approx -8.66 + 12 = 3.34\) m.

\(h(10) = -10\cos 60° + 12 = -5 + 12 = 7\) m.

ARC: \((7 - 3.34)/5 = 0.732\) m/s. The rider is rising at an average of 0.73 m/s over that 5-second window. At the very top or bottom of the wheel, instantaneous speed would be zero; in between, it varies sinusoidally.

Watch out for:

  • Confusing instantaneous and average. They're equal only for linear functions.
  • Forgetting units. Rate of change has units of (output units) per (input units).
  • Dividing by 0 directly. Compute \(\dfrac{f(a + h) - f(a)}{h}\) for small but nonzero \(h\), then look at the trend.
  • Mixing up the difference quotient sign. The formula has \(f(a + h) - f(a)\) on top, not the other way around.

The final exam covers all four strands. Use the question count as a rough mark allocation: the more questions in a strand, the more weight it carries. Before working through the 61-question review, refresh your strategy and your most-missed concepts.

Strand A — Characteristics of Functions. Function notation, domain and range, inverses, transformations, quadratic forms, factoring, solving and optimising quadratics. The two transformations most often confused: inside the function bracket, operations act on \(x\) in the "opposite" direction (e.g. \(x - 3\) shifts right), and the factor \(k\) compresses when \(|k| > 1\); outside, operations act on \(y\) directly. Always factor before reading off horizontal parameters.

Strand B — Exponential Functions. Exponent laws, rational exponents, graphing \(y = b^x\), transformations of exponentials, same-base solving, exponential growth and decay. Most-missed: applying half-life formulas correctly (exponent is \(t/h\), not \(h \cdot t\)); recognising when the same-base method applies and when it doesn't.

Strand C — Discrete Functions. Sequences and series (arithmetic and geometric), sigma notation, compound interest, annuities (future and present value), mortgages. The connection points are critical: arithmetic sequence ↔ linear growth ↔ simple interest; geometric sequence ↔ exponential growth ↔ compound interest; annuity ↔ geometric series.

Strand D — Trigonometric Functions. Right-triangle trig, ratios of any angle (CAST and reference angles), exact values for special angles, sine and cosine laws (including the ambiguous case), graphs of sinusoidal functions with all four parameters, modelling periodic phenomena, solving sinusoidal equations graphically. Watch the ambiguous case: SSA can give 0, 1, or 2 triangles — check the supplementary angle whenever you compute an inverse sine.

  • Categorical assessment — Ontario exams traditionally break marks into four categories: Knowledge/Understanding (recall and procedure), Application (problem-solving in context), Thinking (extended reasoning), Communication (explanation, justification, terminology).
  • Time budget — for a 100-mark, 3-hour exam, plan ~1.5 minutes per mark. Spend the first 5 minutes scanning the entire paper and noting easy K/U questions to tackle first.
  • Show all steps — partial marks are awarded for correct work even if the final answer is wrong. A bare answer with no work risks losing everything.
  • Units and rounding — financial answers to the nearest cent, lengths to one or two decimal places (or as specified), angles to a sensible precision (often 0.1° or 1°).

Top-five "guaranteed mark" checklist before the exam.

1. Function basics. Be fluent at \(f(a+h)\) substitution and inverse-finding (swap then solve).

2. Vertex form. Complete the square cleanly on any \(ax^2 + bx + c\), including factoring \(a\) first.

3. Special-angle exact values. Drill the 30-60-90 and 45-45-90 triangles. Refresh CAST.

4. Compound-interest formula. Have \(A = P(1 + i)^n\), \(FV = R \cdot \dfrac{(1+i)^n - 1}{i}\), and \(PV = R \cdot \dfrac{1 - (1+i)^{-n}}{i}\) at instant recall. Know when to use each.

5. Sinusoidal modelling. From any "max/min/period/time of max" description, set \(c = (\max+\min)/2\), \(|a| = (\max-\min)/2\), \(k = 360°/\text{period}\), \(d\) = time of cosine peak (or sine midline-going-up).

Common-error catalogue — these account for a large share of lost marks.

(a) \((a + b)^2 = a^2 + b^2\). False. \((a + b)^2 = a^2 + 2ab + b^2\).

(b) \(f^{-1}(x) = \dfrac{1}{f(x)}\). False. Inverse undoes the function; reciprocal is something else.

(c) \(\sqrt{a^2 + b^2} = a + b\). False. Never split a square root over addition.

(d) \(\dfrac{a + b}{a} = 1 + b\). False. \(\dfrac{a + b}{a} = 1 + \dfrac{b}{a}\). Cancellation requires factors, not terms.

(e) Sine law with SSA gives one answer. False. Always test the supplementary angle.

(f) \(\log\) on the MCR3U exam? No. Logarithms are MHF4U content; if a problem looks like it needs a log, the same-base method is intended.

  • Read every question once before starting any. Mark the easy ones to do first.
  • Carry one extra decimal in intermediate work; round only at the end.
  • Write what each variable represents in a word problem. Define before equating.
  • Sketch graphs for trig/sinusoidal problems and for any inequality.
  • For "show all steps" Thinking questions, write a one-sentence sign-off explaining what your answer means in context.
  • Leave time at the end to check unit-bearing answers (money, m, m/s, degrees) and sanity-check magnitudes.
