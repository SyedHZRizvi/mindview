Unit 6: Introduction to Vectors

Vector representation, operations, R2 and R3, spanning sets

📚 Strand C: Vectors 📹 9 lessons
📖 Lesson Notes — Ontario Curriculum

A scalar is a quantity that has only magnitude (temperature, mass, time). A vector has both magnitude and direction (displacement, velocity, force). Vectors are represented geometrically as directed arrows whose length encodes magnitude and whose orientation encodes direction. Algebraically, a vector in two dimensions can be written in component form \(\vec{u} = (u_1, u_2)\) and in three dimensions as \(\vec{v} = (v_1, v_2, v_3)\). Two vectors are equal if and only if they have the same magnitude and direction — position in space is irrelevant.

Vectors are added using the triangle (or parallelogram) rule: place the tail of \(\vec{v}\) at the head of \(\vec{u}\); the resultant \(\vec{u}+\vec{v}\) runs from the tail of \(\vec{u}\) to the head of \(\vec{v}\). In component form, addition is componentwise. Subtraction reverses one vector: \(\vec{u} - \vec{v} = \vec{u} + (-\vec{v})\). Scalar multiplication by \(k\) scales magnitude by \(|k|\) and reverses direction if \(k<0\); collinear vectors are exactly those that are scalar multiples of each other.

A linear combination of vectors \(\vec{v}_1, \dots, \vec{v}_n\) is any expression \(c_1\vec{v}_1 + \dots + c_n\vec{v}_n\). The set of all such combinations is the spanning set. In \(\mathbb{R}^2\), any two non-collinear vectors span the whole plane; in \(\mathbb{R}^3\), any three non-coplanar vectors span the whole space. The Cartesian unit vectors \(\hat{i} = (1,0,0)\), \(\hat{j} = (0,1,0)\), \(\hat{k} = (0,0,1)\) are the standard basis: every vector in \(\mathbb{R}^3\) has a unique expression as \(a\hat{i} + b\hat{j} + c\hat{k}\).

  • Scalar — a real number; magnitude only.
  • Vector — a directed quantity with magnitude and direction; written \(\vec{u}\) or in bold.
  • Magnitude — \(|\vec{u}| = \sqrt{u_1^2 + u_2^2 + u_3^2}\) in \(\mathbb{R}^3\); the length of the arrow.
  • Zero vector — \(\vec{0}\); has magnitude 0 and undefined direction; identity for addition.
  • Unit vector — vector with magnitude 1; \(\hat{u} = \frac{\vec{u}}{|\vec{u}|}\).
  • Collinear vectors — vectors parallel to the same line; one is a scalar multiple of the other.
  • Linear combination — \(c_1\vec{v}_1 + c_2\vec{v}_2 + \dots + c_n\vec{v}_n\) for scalars \(c_i\).
  • Spanning set — a set of vectors whose linear combinations produce every vector in the given space.
  • Cartesian (standard) basis — \(\hat{i}, \hat{j}, \hat{k}\) in \(\mathbb{R}^3\); every vector decomposes uniquely.

Worked Example: Vector arithmetic in \(\mathbb{R}^3\)

Given \(\vec{u} = (2,-1,3)\) and \(\vec{v} = (1,4,-2)\), compute \(3\vec{u} - 2\vec{v}\) and its magnitude.

Scalar multiples: \(3\vec{u} = (6,-3,9)\); \(2\vec{v} = (2,8,-4)\).

Subtract componentwise: \(3\vec{u} - 2\vec{v} = (6-2, -3-8, 9-(-4)) = (4, -11, 13)\).

Magnitude: \(|3\vec{u} - 2\vec{v}| = \sqrt{4^2 + (-11)^2 + 13^2} = \sqrt{16 + 121 + 169} = \sqrt{306} \approx 17.49\).

Worked Example: Expressing a vector as a linear combination

Write \(\vec{w} = (7,4)\) as a linear combination of \(\vec{a} = (1,2)\) and \(\vec{b} = (3,-1)\).

Set \(\vec{w} = s\vec{a} + t\vec{b}\): \((7,4) = (s + 3t,\; 2s - t)\). System: \(s + 3t = 7\) and \(2s - t = 4\).

From the second equation \(t = 2s - 4\). Substitute: \(s + 3(2s-4) = 7 \implies 7s - 12 = 7 \implies s = \frac{19}{7}\). Then \(t = 2(\tfrac{19}{7}) - 4 = \tfrac{38}{7} - \tfrac{28}{7} = \tfrac{10}{7}\).

So \(\vec{w} = \tfrac{19}{7}\vec{a} + \tfrac{10}{7}\vec{b}\). Verify: \(\tfrac{19}{7}(1,2) + \tfrac{10}{7}(3,-1) = (\tfrac{19+30}{7}, \tfrac{38-10}{7}) = (7, 4)\). ✓

  • Treating a vector as a point — vectors do not have a fixed location; only magnitude and direction matter.
  • Adding magnitudes of vectors instead of components: \(|\vec{u}+\vec{v}| \ne |\vec{u}| + |\vec{v}|\) in general.
  • Forgetting that the zero vector is parallel to every vector by convention (some texts disagree, but in Ontario MCV4U \(\vec{0}\) is taken to be parallel to all).
  • Confusing notation: \((2,-1,3)\) and \(2\hat{i} - \hat{j} + 3\hat{k}\) are the same vector written two ways.
  • Saying two vectors with the same magnitude are equal — direction must match too.

Curriculum expectation: Strand C, Expectation C1.1 — "recognize a vector as a quantity with both magnitude and direction, and identify, gather, and interpret information about real-world applications of vectors." Also C1.3 — "represent a vector in two-space geometrically as a directed line segment, with directions expressed in different ways … and algebraically (e.g., using Cartesian coordinates)." And C2.2 — "perform the operations of addition, subtraction, and scalar multiplication on vectors represented as directed line segments in two-space, and on vectors represented in Cartesian form in two-space."
